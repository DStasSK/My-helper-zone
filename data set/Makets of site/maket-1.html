Слой по центру веб-страницы.

Важное сообщение!
запустить окно

Выравнивание слоя строго по центру веб-страницы применяется обычно для размещения картинки, служащей точкой входа на сайт или информационного сообщения. Удобство подобного расположения слоя - его независимость от разрешения экрана монитора и окна браузера. В любом случае слой будет занимать положение по центру веб-страницы.

Слой по центру веб-страницы на JS

<div class="window__box" id="window__box">
  <div class="window__item">Важное сообщение!
    <div class="windows_off" title="Закрыть окно"
         onclick="window__box.classList.toggle('hidden')">
    </div>
  </div>
</div>

.window__box {
  position:fixed; top:0; left:0; right:0; bottom:0;
  background:rgba(0,0,0,.6);
  z-index:2;
}
.window__item {
  position:absolute; top:calc(50% - 3em); left:calc(50% - 6em);
  height:2em; width:10em; padding:2.2em 1em 1.8em;
  overflow:hidden; z-index:4;
}
.windows_off {
  position:absolute; right:-20px; top:-20px;
  height:40px; width:40px; transform:rotateZ(45deg);
  z-index:3;
}
.hidden {display: none;}

Слой по центру веб-страницы на
input type="checkbox"

В этой конструкции перед "окном с сообщением" необходимо необходимо добавить элемент input который и будет отключать это окно через селектор соседнего элемента - а именно добавлять его свойству display значение none. При этом сам input будет управляться через элемент label.

<div>
  <input type="checkbox" id="window__checkbox" class="window__checkbox">
  <div class="window__box">
    <div class="window__item">Важное сообщение!
      <label for="window__checkbox" class="windows_off"></label>
    </div>
  </div>
</div>

.window__checkbox {display: none;}
.window__box {
  position:fixed; top:0; left:0; right:0; bottom:0;
  background:rgba(0,0,0,.6);
  z-index:2;
}
.window__item {
  position:absolute; top:calc(50% - 3em); left:calc(50% - 6em);
  height:2em; width:10em; padding:2.2em 1em 1.8em;
  overflow:hidden; z-index:4;
}
.windows_off {
  position:absolute; right:-20px; top:-20px;
  height:40px; width:40px; transform:rotateZ(45deg);
  z-index:3;
}
/* отключаем окно при чеке на input */
.window__checkbox:checked ~ .window__box {display: none;}
