Слой по центру веб-страницы

Важное сообщение!
запустить
окно

Выравнивание слоя строго по центру веб-страницы применяется обычно для размещения картинки, служащей точкой входа на сайт или информационного сообщения. Удобство подобного расположения слоя - его независимость от разрешения экрана монитора и окна браузера. В любом случае слой будет занимать положение по центру веб-страницы.

HTML
<!-- окно на весь экран -->
<div class="window">
  <input type="checkbox" id="w1">
    <div class="box1">
      <div class="mail">Важное сообщение!           <!-- сообщение -->
      <label for="w1" title="Закрыть окно"></label> <!-- label - выключатель -->
    </div>
  </div>
</div>
<!-- /окно на весь экран -->

CSS
.box1{
  position:fixed; top:0; left:0; right:0; bottom:0;
  background:rgba(0,0,0,.6);
  z-index:2;}
.mail{
  position:fixed; top:calc(50% - 3em); left:calc(50% - 6em);
  height:2em; width:10em; padding:2em 1em;
  display:block; background:#fff;
  text-align:center; color:red; font-size:2em;
  overflow:hidden; z-index:4;}
.window label{
  position:absolute; right:calc(50% - 6.6em); top:calc(50% - 3.6em);
  cursor:pointer; z-index:3;
  height:40px; width:40px; background:red;
  transform:rotateZ(45deg);}
.window input{display:none;}
.window :checked ~ .box1{display:none;}  /* откл.сообщение */
.window :checked ~ label{display:none;}  /* откл.сообщение */
