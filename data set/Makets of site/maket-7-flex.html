Заголовок (header)

Статья

HTML 
<div class="container">
  <header class="header">Заголовок (header)</header>
  <section class="main">
    <article class="article">Статья</article>
    <nav class="nav">Навигация</nav>
    <aside class="aside">Боковая панель</aside>
  </section>
  <footer class="footer">Подвал (footer)</footer>
</div>

CSS 
* {box-sizing: border-box}
.main {
  display: flex;
  min-height: 58vh;

.article {
  flex: 1;    /* flex-grow:1; */
  overflow-x: auto;
}
.nav, .aside {
  flex: 0 0 auto;
  min-width: 120px;
}
.nav {order: -1;}

@media screen and (max-width: 575px) {
  .main {flex-direction: column;}
}

В данном примере мы делаем элемент section class="main" флекс-контейнером, а шапку и подвал оставляем блочными элементами. Другими словами, флексбоксом становится только средняя часть.

Используем display:flex, чтобы сделать флекс-контейнер. Также задаётся значение min-height. Это гарантирует, что макет будет занимать всю высоту экрана, даже если в нём мало содержимого. В итоге подвал никогда не поднимется и не оставит пустого пространства под ним, если содержимое занимает меньше высоты экрана.
