Создание двух колонок происходит с помощью стилевого свойства float со значением left, которое применяется для левой колонки. При этом одновременно следует установить отступ слева для правой колонки через margin-left, значение которого равно ширине левой колонки, иначе текст заедет под ссылки. Для футера задают отмену обтекания через clear:both. Иначе он может "наехать" на верхние колонки.

При использовании такого макета необходимо учитывать что высота блока с контентом должна быть больше чем высота блока с сылками.

Чтобы макет располагался по центру веб-страницы, добавляют слой с именем container и колонки распологают внутри него. Для самого container устанавливают ширину и выравнивание по центру (margin: 0 auto).

Также для браузера Internet Explorer, чтобы он понимал стиль для новых элементов, придется включить скрипт через условные комментарии.

HTML 
<head>
  <!--[if IE]>
    <script src="http://html5shiv.googlecode.com/svn/trunk/html5.js"></script>
  <![endif]-->
</head>

<div class="conteiner">
  <div class="header">Заголовок</div>
  <div class="sidebar">ссылки</div>
  <div class="content">содержание<br>контента</div>
  <div class="footer">© фу-у-тер</div>
</div>

CSS 
* {box-sizing: border-box;}
.conteiner {
  width: 700px;
  margin: 0 auto;
  background: #00f;    /* цвет фона левой колонки */
}
.sidebar {             /* левая колонка */
  float: left;
  width: 150px;
}
.content {             /* правая колонка */
  width: calc(100% - 150px)
  margin-left: 150px;
}
.footer {clear: both;} /* отмена обтекания */
