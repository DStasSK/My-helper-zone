Создание двух колонок происходит с помощью стилевого свойства float со значением left, которое применяется для левой колонки. При этом одновременно следует установить отступ слева для правой колонки через margin-left, значение которого равно ширине левой колонки плюс расстояние между ними, иначе текст заедет под ссылки.

Чтобы макет располагался по центру веб-страницы, добавляют слой с именем container и колонки распологают внутри него. Для самого container устанавливают ширину (width) и выравнивание по центру (margin: 0 auto).

Для футера задают отмену обтекания через clear:both. Иначе он может "наехить" на верхние колонки.

Также для браузера Internet Explorer, чтобы он понимал стиль для новых элементов, придется включить скрипт через условные комментарии.

HTML
<head>
  <!--[if IE]>
    <script src="http://html5shiv.googlecode.com/svn/trunk/html5.js"></script>
  <![endif]-->
</head>

<div class="conteiner">
  <div class="header">Заголовок</div>
  <div class="sidebar">ссылки</div>
  <div class="content">содержание<br>контента<br>тут</div>
  <div class="footer">© фу-у-тер</div>
</div>

CSS
.conteiner{
  width:700px; margin:auto;
  font:Arial, Helvetica, sans-serif;
  font-size:1em;
  background:#00f;}    /* цвет фона левой колонки */
.header{
  padding:10px;
  background:#f00;
  text-align:center;}
.sidebar{
  float:left;
  width:120px;         /* ширина левой колонки с учетоем padding 15px */
  padding:10px 15px;}
.content{
  width:520px;         /* правой левой колонки с учетоем padding 15px */
  padding:10px 15px;
  margin-left:150px;
  background:#ff0;}    /* Цвет фона правой колонки */
.footer{
  background:#32CD32;
  padding:10px 20px 15px;
  clear:both;}         /* отмена обтекания */
