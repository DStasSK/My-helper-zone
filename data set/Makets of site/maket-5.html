При создании подобного макета необходимо понимать, что размеры абсолютно позиционированных элементов подгоняются под их содержание, поэтому ширину надо указывать явно. Также слои могут накладываться поверх других элементов макета: при уменьшении текста в левой колонке, правая колонка наедет на футер. Поэтому использование позиционирования подходит далеко не во всех случаях.

<div class="conteiner">
  <div class="header">Заголовок</div>
  <div class="content">содержание<br>контента</div>
  <div class="sidebar">ссылки</div>
  <div class="footer">© фу-у-тер</div>
</div>

* {box-sizing: border-box;}
.conteiner {position: relative;}
.header {
  padding: 10px;
  line-height: 20px;
}
.content {
  width: calc(100% - 150px);
  margin-right: 150px;
}
.sidebar {
  position: absolute;
  right: 0;
  top: 40px;
  width: 150px;
}
