Используя float, распологают колонки рядом, но добавляют его только для правой колонки. В стиле левой колонки установим margin-right, значение которого равно ширине правой колонки. Общую ширину макета устанавливают с помощью еще одного слоя - container, а остальные слои располагаются внутри него.

В коде вначале приводится правая колонка, а затем левая, это важное условие.
При использовании такого макета необходимо учитывать что высота блока с контентом и высота блока с сылками имеют разное значение, это необходимо учитывать при задании цвета фона.

Также для браузера Internet Explorer, чтобы он понимал стиль для новых элементов, придется включить скрипт через условные комментарии.

HTML 
<head>
  <!--[if IE]>
    <script src="http://html5shiv.googlecode.com/svn/trunk/html5.js"></script>
  <![endif]-->
</head>

<div class="conteiner">
  <div class="header">Заголовок</div>
  <div class="sidebar">ссылки</div>
  <div class="content">содержание<br>контента</div>
  <div class="footer">© фу-у-тер</div>
</div>

CSS 
* {box-sizing: border-box;}
.conteiner {width: 700px;}
.sidebar {
  float: right;
  width: 150px;
}
.content {
  width: calc(100% - 150px);
  margin-right: 150px;
}
.footer {clear: both;}    /* отмена обтекания */
