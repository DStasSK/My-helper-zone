flex-grow

Cвойство отвечает за то, в какой пропорции будет разделено свободное место между элементами одного родителя. В качестве значения принимаются числа, они задают пропорции в которых свободное пространство будет добавляться к размерам каждого флекса. Например если есть один flex-элемент и у него значение flex-grow:1, то он займет все свободное пространство.

Применяется к флекс-элементам, не наследуется, анимируется.

Значение по умолчанию: 0.

.fl {flex-grow: число; } /* flex:1; = flex-grow:1; */

Принимаются целые (1, 2, 3,…) или дробные числа (например: 0.6). Отрицательные значения игнорируются.

Safari до версии 9 поддерживает свойство -webkit-flex-grow

Пример

<style>
  p {display:flex;}
  label {margin-right:10px;}
  .fl1 {flex-grow:1;}
  .mr {margin-right:10px;}
  .fl3 {flex-grow:3;}
</style>

<p>
  <label>Имя:</label>
  <input name="name" id="name" class="fl1" placeholder="flex-grow:1;"></p>
<p>
  <label>Имя:</label>
  <input name="name1" id="name1" class="fl1 mr" placeholder="flex-grow:1;">
  <input name="name13" id="name13" class="fl3" placeholder="flex-grow:3;">
</p>
<p>
  <label>Имя:</label>
  <input name="name2" id="name2" placeholder="flex-grow:0;">
</p>
