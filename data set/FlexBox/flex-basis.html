flex-basis

Свойство flex-basis определяет основу флекса, которая является начальным размером элемента. Похоже на свойства width и height, к которым добавляется содержимое элемента. Применяется к флекс-элементу, не наследуется, анимируется. Значение по умолчанию: auto.

.fl {flex-basis: auto | <размер> }
Safari до версии 9 поддерживает свойство -webkit-flex-basis
Значение
autoУказывает автоматический размер, основанный на содержимом элемента.
<размер>Задаёт размер элемента в px, mm, pt или в процентах вдоль главной оси. При этом размер вычисляется относительно родителя. Отрицательное значение недопустимо.

.container1 {color:#fff; font-size:30px; line-height:30px;
  display:flex; height:80px;}

.items {padding:25px 0; display:flex;
  justify-content:center;}   /* выравнивание по центру */

.i1 {background:#508694; flex-basis:50%;}
.i2 {background:#BB844C; flex-basis:25%;}
.i3 {background:#929D79; flex-basis:25%;}

Пример 2

Первый
Второй
Третий
HTML 
<div class="flex-container">
  <div class="flex-items one">Первый</div>
  <div class="flex-items two">Второй</div>
  <div class="flex-items three">Третий</div>
</div>

CSS 
.flex-container {color:#fff; font-size:2.6em;
  display:flex;
  height:300px;
  flex-direction:column;      /* Главная ось - вертикально */
  flex-wrap:wrap;}            /* перенос разрешен */

.flex-items {display:flex;
  align-items:center;         /* выравнивание текста по вертикали */
  justify-content:center;}    /* выравнивание текста по горизонтали */

.one {background:#508694; margin-right:10px; order:1;   /* первый блок */
  flex-basis:100%;}
.two {background:#BB844C; margin-bottom:10px; order:2;  /* второй блок */
  flex: 1 1 0;}        /* flex-grow:1, flex-shrink:1, flex-basis: auto */

.three {background:#929D79; order:3;                    /* третий блок */
  flex: 1 1 0;}
