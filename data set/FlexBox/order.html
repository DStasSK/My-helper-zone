order

Определяет порядок вывода флексов внутри флекс-контейнера. Элементы располагаются согласно значениям свойства order от меньшего к большему. При равных значениях order элементы выводятся в том порядке, в каком они появляются в исходном коде. Применяется к флекс-элементам, не наследуется, анимируется. Значение по умолчанию: 0.

.fl {
  -webkit-box-ordinal-group: 2;  /* iOS 6-, Safari 3.1-6 */
  -moz-box-ordinal-group: 2;     /* Firefox 19- */
  -ms-flex-order: 2;             /* IE 10 */
  -webkit-order: 2;              /* Chrome */
  order: 2;}
Safari до версии 9 поддерживает свойство -webkit-order

В качестве значения принимается любое целое число, включая отрицательные числа и ноль.

  • 2
  • 3
  • 4
<style>
  .prim1 ul {margin:0; padding:0; list-style:none; display:flex;}
  .prim1 li {margin:5px; padding:20px 30px; font-size:20px;
             color:#fff; background-color:#aaa;}
  .prim1 li:nth-child(1) {order:3}
  .prim1 li:nth-child(2) {order:2}
  .prim1 li:nth-child(3) {order:2}
  .prim1 li:nth-child(4) {order:1}
</style>

<div class="prim1">
  <ul>
    <li>1</li>   <!-- order:3 -->
    <li>2</li>   <!-- order:2 -->
    <li>3</li>   <!-- order:2 -->
    <li>4</li>   <!-- order:1 -->
  </ul>
</div>
