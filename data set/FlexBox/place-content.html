place-content

Сокращённое свойство place-content одновременно устанавливает значения свойств align-content и justify-content. Применяется к блочному контейнеру, флекс-контейнеру или грид-контейнеру; не наследуется, не анимируется. Значение по умолчанию: normal.

.fl {place-content: align-content justify-content; }

Значения - cмотрите отдельные свойства для их значений.

Если второго значения нет, то первое значение используется сразу для двух свойств, но при условии, что значение является допустимым для того и другого свойства.
