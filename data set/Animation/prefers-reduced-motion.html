prefers-reduced-motion

CSS медиа функция prefers-reduced-motion может использоваться для определения того, запросил ли пользователь, чтобы ОС минимизировала количество анимации или движения, которые она использует.

Принимаемые значения:
  • no-preference - Указывает, что пользователь не сделал никаких предпочтений, известных системе.
  • reduce - Указывает, что пользователь уведомил ОС, что он предпочитает интерфейс, который минимизирует количество движения или анимации, предпочтительно до точки, где удаляются все несущественные движения.
.elem {
  -webkit-animation: my_animation 0.3s linear infinite both;
  animation: my_animation 0.3s linear infinite both;
}

@media (prefers-reduced-motion: reduce) {
  .elem {
    -webkit-animation: none;
    animation: none;
  }
}
