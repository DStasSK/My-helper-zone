CSS cвойство animation-name

CSS cвойство animation-name задаёт список анимаций, чтобы применить к элементу. Каждое имя является правилом @keyframes, которое задаёт значения свойств для последовательности анимации. Значение по умолчанию none (oтключает анимацию).

div {
  animation-name: none | custom-ident;
}
  • none - oтключает анимацию. Может быть использовано для отключения анимации без изменения порядка других идентификаторов, или для отключения анимации, поступающей из каскада.
  • custom-ident - идентификатор анимации - строка. Состоит из комбинации букв без учёта регистра от a до z, цифр от 0 до 9, подчёркивания ( _ ), и/или черты ( - ). Первый символ без черты должен быть буквой (то есть, без цифры в начале, даже если перед ним стоит черта.) Кроме того, две черты запрещены в начале идентификатора. Идентификатор анимации не может быть none, unset, initial, или inherit в любой комбинации случаев.

Использования одного свойства animation-name при этом недостаточно. Кроме того, необходимо объявить свойство animation-duration, чтобы браузер знал, как долго должна длиться анимация до завершения.

В старом коде могут всртечаться различные префиксы:
.elem {
  -webkit-animation-name: slide;
     -moz-animation-name: slide;
       -o-animation-name: slide;
          animation-name: slide;
}
