CSS cвойство animation-iteration-count

CSS cвойство animation-iteration-count определяет сколько раз будет проигрываться анимационный цикл, перед тем как остановиться. Задается либо просто числом, либо можно указать infinite и анимация будет выполняться бесконечно. Значение по умолчанию 1.

div {
  animation-iteration-count: 1;
}

Отрицательные значения не используются. Можно использовать не целые значения, для проигрывания части анимационного цикла - например, значение 0.5 воспроизведёт половину анимационного цикла.

В старом коде могут всртечаться различные префиксы:
div {
  -webkit-animation-iteration-count: 1;
     -moz-animation-iteration-count: 1;
       -o-animation-iteration-count: 1;
          animation-iteration-count: 1;
}
