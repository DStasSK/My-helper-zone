CSS cвойство animation-fill-mode

CSS cвойство animation-fill-mode oпределяет, какие стили должны применяться к элементу, когда анимация не проигрывается. Например, после её завершения или при остановке. По умолчанию, в момент окончания анимации стиль элемента возвращается к исходному, свойство animation-fill-mode позволяет изменить это поведение и сделать так, чтобы стиль элемента оставался как у последнего ключевого кадра. Значение по умолчанию none, не наследуется, не анимируется.

div {
  animation-fill-mode: none | forwards | backwards | both;
}
  • none - не будет применять любые стили из анимации к элементу до или после того, как анимация запущена.
  • forwards - сохранит стили, объявленные в последнем указанном ключевом кадре. На эти стили, однако, могут повлиять значения свойств animation-direction и animation-iteration-count, определяющие окончание анимации.
  • backwards - будет применять стили первого заданного ключевого кадра, до того, как анимация будет работать. Включает применение этих стилей в течение всего времени, что может быть задано в animation-delay. Значение backwards также может зависеть от значения свойства animation-direction.
  • both - применит поведение сразу от обоих значений forwards и backwards.
animation: my_anim 2s linear 2;
animation-fill-mode: none;
animation-fill-mode: forwards;
animation-fill-mode: backwards;
animation-fill-mode: both;
В старом коде могут всртечаться различные префиксы:
.box {
  -webkit-animation-fill-mode: forwards;
     -moz-animation-fill-mode: forwards;
       -o-animation-fill-mode: forwards;
          animation-fill-mode: forwards;
}
