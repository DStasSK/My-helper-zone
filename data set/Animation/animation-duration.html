CSS cвойство animation-duration

CSS cвойство animation-duration задаёт время в секундах или миллисекундах, сколько должен длиться один цикл анимации. Значение по умолчанию - 0s (анимации нет). Можно указать несколько значений, перечисляя их через запятую. Отрицательные значения и значения без указания единиц времени (s или ms) будут игнорироваться.

div {
  -webkit-animation-duration: 1s;
     -moz-animation-duration: 1s;
       -o-animation-duration: 1s;
          animation-duration: 1s;
}
