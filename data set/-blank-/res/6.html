Пример 1.

В данном примере на фоновое изображение накладывается текст с белым фоном, который закрывает собой изображение. Режим наложения lighten оставляет лишь светлые цвета и поскольку цвет текста по умолчанию чёрный, то получаются цветные буквы на белом фоне

Аквариум

.header {
  background: url(img/aquaria.jpg) center no-repeat;
  text-align: center;
}
.header__text {
  background: #fff;          /* Белый цвет фона */
  mix-blend-mode: lighten;   /* Режим наложения */
  color: #000;
}
