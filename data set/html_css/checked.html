псевдокласс :checked

:checked - выбранный (чекнутый) элемент

Применяется для:

Флажка
<input type="checkbox" checked>
ДаНетПереключателя
<input type="radio" name="yesno" checked>Да
<input type="radio" name="yesno">Нет
Списка
<select>
  <option>один</option>
  <option>два</option>
  <option>три</option>
</select>

Сами поля форм плохо стилизуются, и для того чтобы как-то активно использовать :checked идут на хитрость и стилизуют не сам <input>, а элемент <label>. Для этого сразу после <input> вставляем <label> и связываем эти два элемента с помощью одинакового значения у атрибутов id и for. Теперь можно щёлкать уже по тексту и флажок будет включаться или выключаться. В стилях их тоже можно связать между собой.

<input type="checkbox" id="switch" checked>
<label for="switch">Флажок</label>

input[type="checkbox"]:checked + label {color:red}
 Чекни флажок:

Стилизация "флажка"

Для изменения стиля флажков и переключателей - прячут исходный элемент <input> с помощью значения none у свойства display, а нужный стиль применяют к элементу <label>, связанный с этим <input>. В примере флажок сделан как цветной прямоугольник, внутри которого перемещается белый квадратик. Включенный флажок выглядит как прямоугольник синего цвета с квадратиком слева; выключенный флажок выглядит как прямоугольник серого цвета с квадратиком справа.

Чтобы квадратик плавно перемещался из одного положения в другое, добавлено свойство transition, значением которого выступает время движения, в данном случае полсекунды.

Сам квадратик выводится с помощью псевдоэлемента :before, который добавляется к селектору <label>. Это позволяет легко стилизовать элемент, к примеру, сделать его круглым, добавив лишь свойство border-radius.

<div class="t3">
  <input type="checkbox" id="switch">
  <label for="switch"></label>
</div>

CSS
.t3 input{display:none;}
.t3 label{
  margin-top:10px;
  display: inline-block;
  position: relative;
  width: 60px; height: 30px;
  background: #9E9E9E;
  transition: 0.5s; /* Время смены цвета фона */
  }
.t3 label:before {
  content: '';
  position: absolute;
  background: #fff;
  width: 20px; height: 20px;
  left: 35px; top: 5px;
  transition: 0.5s; /* Время движения квадратика */
  }
.t3 input:checked + label{background: #1976D2;}
.t3 input:checked + label:before {left: 5px;} /*Квадратик слева*/

Cтилизация "переключателей"

Стилизация переключателей происходит аналогично стилизации флажков — скрывают исходный элемент <input>, а вместо него стилизуют элемент, идущий в коде HTML после <input>. При этом обязательно используют <label>, без которого поведение переключателя работать не будет.

Поскольку переключатели обычно вставляются группой, то связывать <input> с <label> через идентификатор не очень удобно из-за того, что придётся вводить множество идентификаторов. Так что поступают иначе — вставляют <input> внутрь <label>, а после <input> добавляют необходимый элемент, в данном примере - <span>.

<label><input type="radio"><span></span></label>

Вид самого переключателя задаётся через селектор <span>, а вид выбранного переключателя через селектор :checked+<span>.

<div class="t4">
  <label><input type="radio" checked name="thre"><span></span>Один</label>
  <label><input type="radio" name="thre"><span></span>Два</label>
  <label><input type="radio" name="thre"><span></span>Три</label>
</div>

CSS
.t4 {display:block;}
.t4 input {display:none;}
.t4 label {
  display:inline-block;
  margin:0px 20px;
  line-height:20px;
  padding:0;
  cursor:pointer;
  position:relative;}
.t4 span {
  content:"";
  display:inline-block;
  position:absolute;  top:0px;  left:-25px;
  width:20px;  height:20px;  border-radius:50%;
  background:#999;
  padding:0;
  box-shadow: inset 0 0 7px 4px rgba(0,0,0,0.25);
  transition: background .5s, box-shadow .4s;
  }
.t4 input:checked + span {background:red; box-shadow:none;}
.t4 input:checked + span:before {
  content:"";
  position:absolute;
  width:10px;
  height:10px;
  border-radius:50%;
  top:5px;
  left:5px;
  background:#fff;}
