CSS cвойство box-decoration-break.

Указывает, как оформлять фрагмент текста при его переносе на другую строку. Значение по умолчанию slice, не наследуется, не анимируется. Работает только для следующих свойств: background, border, border-image, box-shadow, clip-path, margin, padding.

div {
  box-decoration-break: slice | clone:
}
sliceЗначение по умолчанию. Каждый фрагмент разрезается в месте переноса и никакие эффекты оформления на месте «разреза» не отображаются. В частности, граница не показывается, тень не выводится, радиус скругления работает только для всего блока целиком.
cloneКаждый фрагмент оформляется независимо. Ко всем фрагментам отдельно применяется рамка, тень, фон и др.
Старые версии браузеров Edge, Chrome, Opera, Safari поддерживают свойство -webkit-box-decoration-break.
IE не поддерживает данное свойство.
