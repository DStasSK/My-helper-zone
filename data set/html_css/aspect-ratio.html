aspect-ratio

Свойство aspect-ratio задает соотношение сторон для элемента. Свойство вычисляет незаданную сторону, исходя из размера уже известной.

Свойство нельзя применить к строчным элементам.
.elem { aspect-ratio: auto || size }
  • auto - соотношение сторон считается автоматически.
  • width/height - соотношение сторон всегда считается относительно ширины или высоты блока. 16/9 - здесь первое число это ширина, а второе число — высота. Если какое-то значение не указано, то оно считается равным 1. Также некоторые соотношения допустимо записывать одним числом. Например, 2/1 можно записать как 0.5, это даст одинаковый результат.
  • auto width/height - когда у элемента (например изоброжения, видео, фреймы) есть собственное соотношение сторон, то оно сохраняется.

Свойство aspect-ratio не оказывает влияния в следующих ситуациях:

  • У элемента заданы свойства width и height. В этом случае соотношение сторон браузер определяет как width / height.
  • У изображения не задана ширина и высота. В этом случае изображения отображаются в их исходном размере.

.container1 {
  width: 270px;
  aspect-ratio: 2 / 1;  /* height:100px */
}

.container2 {
  width: 80px;
  aspect-ratio: 1 / 1;
  border: 3px solid red;
}

.container3 {
  padding-top: 20px;
  background: #444 content-box;
}

.container4 {
  box-sizing: border-box;
}

.container5 {
   width:270px; aspect-ratio:16 / 9;
}
  • Свойству padding лучше задавать нулевое значение.
  • Свойство line-height может увеличить высоту элемента (если его значение больше расчетного).
  • Свойство border без box-sizing:border-box увеличивает размеры элемента.
