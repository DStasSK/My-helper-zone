CSS свойство scroll-behavior

CSS-свойство scroll-behavior определяет поведение прокрутки для любого элемента на странице. Отрабатывается в момент автоматической прокрутки навигационных элементов, либо CSSOM прокручивается API. Любые другие прокручивания, например, те, которые выполняются пользователем, не затрагиваются этим свойством. Если это свойство указано на корневой элемент, оно относится ко всему viewport пространству. Значение по умолчанию auto, не наследуется, не анимируется, применяется к блоку с прокруткой.

div {
  scroll-behavior: auto | smooth;
}
  • auto - cтандартное поведение прокрутки без эффекта плавности.
  • smooth - прокрутка происходит плавно. Время и функция перехода устанавливаются непосредственно браузером.
