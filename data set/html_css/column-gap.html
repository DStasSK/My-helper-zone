CSS свойство column-gap

Данное свойство определяет промежуток между столбцами в grid, flexbox или multi-column макете.

div {
  display: flex|grid;  ||  columns: 2;
  column-gap: length |normal | initial | inherit
}
  • length - указанная величина зазора между столбцами.
  • normal - эквивалентно значению по умолчанию - 1em.
  • initial - устанавливает значение по умолчанию.
  • inherit - наследует значение от родительского элемента.
