content

Свойство content позволяет вставлять генерируемое содержание в текст веб-страницы, которое первоначально в тексте отсутствует / заменяет элемент сгенерированным значением. Обычно применяется совместно с псевдоэлементами ::after и ::before, они соответственно указывают отображать новое содержимое после или до содержимого элемента, к которому добавляются. Значение по умолчанию - normal, не наследуется, не анимируется.

.elem {
  content: строка | attr(атрибут) | url | counter | normal | none;
  content: open-quote | close-quote | no-open-quote | no-close-quote;
}
<строка>

Текст, который добавляется на веб-страницу, строка при этом должна браться в двойные или одинарные кавычки. Любое количество текстовых символов. Нелатинские символы должны быть закодированы с использованием их escape-последовательности Unicode: например, \000A9 представляет собой символ авторского права.

Спецсимволы HTML которые начинаются с амперсанда (&sect; например), будут отображаться как есть, т. е. простым текстом (&sect;, а не §).
attr(<атрибут>) Возвращает строку, которая является значением атрибута тега указанного в скобках. 
a::after {content:attr(href)}
добавит после ссылки её адрес, т. е. значение атрибута href. Если указанного атрибута нет, то вернётся пустая строка. Чувствительность к регистру в названии атрибута зависит от языка документа.
open-quote Вставляет открывающую кавычку, тип которой устанавливается с помощью стилевого свойства quotes.
close-quote Вставляет закрывающую кавычку, тип которой устанавливается с помощью стилевого свойства quotes.
no-open-quote Отменяет добавление открывающей кавычки. Не вводит никакого содержимого, но увеличивает (уменьшает) уровень вложенности для кавычек.
no-close-quote Отменяет добавление закрывающей кавычки. Не вводит никакого содержимого, но увеличивает (уменьшает) уровень вложенности для кавычек.
url Абсолютный или относительный адрес вставляемого объекта. Если указанный файл браузер не может отобразить, то значение игнорируется или отображается значение по умолчанию.
counter Выводит значение счётчика, заданного свойством counter-reset.
none Не добавляет никакого содержимого.
normal Задаётся как none для псевдоэлементов ::after и ::before.
