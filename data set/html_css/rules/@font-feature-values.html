@font-feature-values

Правило @font-feature-values позволяет использовать общее имя в свойстве font-variant-alternates для функций, которые по разному активируются в OpenType. Это может помочь упростить ваш CSS при использовании нескольких шрифтов. Это правило ​​можно использовать как на верхнем уровне CSS, так и внутри любого правила условной группы CSS.

@font-feature-values <my-font-name> {
  <declaration-list>
}
