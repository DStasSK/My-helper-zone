@layer

Правило @layer используется для объявления каскадного слоя, а также позволяет задать приоритеты при наличии нескольких каскадных слоёв.

@layer myName {
  rules
}
  • myName - название каскадного слоя.
  • rules - набор CSS-правил каскадного слоя.
Любые стили, заданные вне слоя @layer, будут переопределять стили, объявлённые внутри любого слоя, независимо от их специфичности.

Допускается создавать каскадный слой без указанных css свойств и их параметров. Более того допускается после определения правила @layer указывать несколько слоёв через запятую, при этом будет определяться приоритет правил от последнего к первому - последний имеет высший приоритет, первый - самый низкий, при этом специфичность селекторов в разных слоях не учитывается.

Объявив имена слоёв и установив их порядок, можно добавить CSS-правила в слой повторно объявив его имя. Таким образом стили добавятся к слою, и порядок слоёв и приоритет свойств между слоями не изменится. Если слой был объявлен без имени, то он также будет работать, но в него нельзя будет добавить css свойства в другом месте; и приоритет его css свойств будет устанавливаться порядком объявления слоя.

Test

@layer myGroup1;
@layer myGroup2, myGroup3, myGroup4;

p {color: rebeccapurple;}

@layer myGroup1 {
  p {font-weight:900; color:green; font-size:10px;}
}
@layer myGroup4 {
  p {font-weight:400; color:red; font-size:30px;}
}
@layer myGroup2 {
  p {font-weight:700; color:yellow; font-size:5px;}
}
@layer myGroup3 {
  p {font-weight:700; color:brown; font-size:20px;}
}
@layer{
  p {font-size: 50px;}
}

Каскадные слои также могут быть вложены друг в друга. При этом для добавления css свойств в внутренний каскадный слой к нему следует обращаться через внешний слой, отделяя названия слоёв точкой - как свойства объекта в JS.

@layer parent {
  @layer chald {
  }
}

@layer parent.chald {
  p {
    margin: 1rem;
  }
}

Каскадный слой также можно импортировать из другого css файла используя @import. В этом случае правила будут находиться в импортированном файле стилей. Нужно помнитт, что @-правило @import должно предшествовать всем другим типам правил, за исключением правил @charset и @layer.

@import "theme.css" layer(myLayerRules);
