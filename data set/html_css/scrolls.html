Cтилизовать полосу прокрутки через -webkit- префикс возможно только в браузерах, использующих механизм рендеринга Webkit и Blink (Safari, Google Chrome, Opera). То есть в Firefox и IE этот способ не сработает.

Как убрать полосу прокрутки CSS

Скрыть полосу прокрутки можно как у отдельного элемента на странице, так и у всей страницы целиком. Сделать это не сложно, достаточно написать следующее свойство:

/* Убрать полосу прокрутки у класса или элемента */
.classname::-webkit-scrollbar {width: 0;}

/* Убрать полосу прокрутки для всей страницы */
::-webkit-scrollbar {width: 0;}

Полосу прокрутки так же можно скрывать или отображать при помощи параметров свойства overflow.

Как изменить скроллбар CSS

-webkit-scrollbar состоит различных псевдо-элементов:

::-webkit-scrollbar - это фон самого скроллбара.
::-webkit-scrollbar-button - кнопки направления на полосе прокрутки.
::-webkit-scrollbar-track - пустое пространство под индикатором прокрутки.
::-webkit-scrollbar-thumb - индикатор прокрутки (перетаскиваемый элемент).
1
базовый скролл (по умолчанию)
<style>
  .element {overflow-y:scroll}
  .box {min-height:500px}
</style>

<div class="element">
   <div class="box">1</div>
</div>
2
ширина 10рх, изменены цвета
<style>
   .el2::-webkit-scrollbar {
      width:10px;
      background-color:#f9f9fd;}
   .el2::-webkit-scrollbar-thumb {
      background-color:#223c50;}
</style>

<div class="element el2">
   <div class="box">2</div>
</div>
3
добавлена тень на полосе
<style>
.el3::-webkit-scrollbar-track {
   -webkit-box-shadow:
      5px 5px 5px -5px rgba(34, 60, 80, 0.2) inset;
   background-color: #f9f9fd;}
</style>

<div class="element el2 el3">
   <div class="box">3</div>
</div>
