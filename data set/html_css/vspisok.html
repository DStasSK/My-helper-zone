В большинстве случаев выпадающие списки реализованы через использование Java Script, а при помощи CSS элементы списка только стилизуются. Java Script позволяет использовать разворачивание элементов не только при клике, но и при других событиях, при этом сам скрипт имеет небольшие размеры по сравнению с реализацией выпадающих списков на CSS, да и он более понятен.

Ваиант на тегах details и summary.

Архитектура:

<details>
  <summary>Видимый текст 1</summary>
  <p>Скрытый текст 1</p>
  <p>Скрытый текст 1</p>
</details>
<details>
  <summary>Видимый текст 2</summary>
  <p>Скрытый текст 2</p>
  <p>Скрытый текст 2</p>
</details>

Дефолтное отображение:

Видимый текст 1
Скрытый текст 1
Скрытый текст 1
Видимый текст 2
Скрытый текст 2
Скрытый текст 2

Минусы данного метода:
- старые браузеры не знают таких тегов и не будут скрывать информацию.
- дефолтный маркер надо спрятать, используя префиксы, и создать свой.

Чтобы тег <summary> не растягивался на всю ширину и были кликабельными только слова, а не вся строка - устанавливают параметр display:inline-block.

summary::-webkit-details-marker {display: none;}
summary::-moz-list-bullet {list-style-type: none;}
summary::marker {display: none;}
summary {display: inline-block;}
IE не поддерживает данные тэги.

Вариант на onclick и toggle (JS).

Нажми на меня
  • 1-й пункт
  • 2-й пункт
  • 3-й пункт
HTML 
<!-- id повторяться не должны -->
<div class="menu" onclick='submenu.classList.toggle("hiden");'>
  Нажми на меня
</div>
<ul id='submenu' class="hiden" onclick="event.stopPropagation()">
  <li>1-й пункт</li>
  <li>2-й пункт</li>
  <li>3-й пункт</li>
</ul>
<!-- onclick="event.stopPropagation()"
предотвращает всплытие клика
это не позволят при клике по пунктам сворачивать меню -->

CSS 
.menu {cursor:pointer; display:inline-block;}
.hiden {display:none;}

Вариант на input, label и :checked.

Есть два типа аккордеон-меню.

  1. При раскрытии любого пункта списка открытое содержимое закрывается; по своему действию это похоже на вкладки.
  2. Можно раскрывать любой пункт списка, остальные пункты при этом не закрываются.

Для первого типа аккордеона вставляем <input type="radio">, который позволяет выбрать лишь один пункт, а для второго типа <input type="checkbox"> - с его помощью можно раскрывать и закрывать произвольное число пунктов.

Для каждого input необходимо добавить своей уникальный id, а его значение затем вставить в атрибут for у расположенного ниже элемента label.

Для того чтобы отдельные пункты были заранее открытими, необходимо добавить к тегу input параметр checked.

<style>
  .spoiler label {cursor:pointer;}
  .spoiler input {display:none;}
  .spoiler div {display:none;}
  .spoiler :checked ~ div {display:block;}
</style>
<div class="spoiler">
  <input type="checkbox" id="spoiler1">
  <label for="spoiler1">Видимый текст</label>
  <div>Скрытый текст</div>
</div>

HTML 
<ul class="list" id="js_example">
  <li>
    <span class="toggle__hidden">Раздел №1</span>
    <ul class="hidden">
      <li>Подраздел 1</li>
      <li>Подраздел 2</li>
      <li>Подраздел 3</li>
    </ul>
  </li>
  <li>
    <span class="toggle__hidden">Раздел №2</span>
    <span>какой-то текст</span>
    <ul class="hidden">
      <li>Подраздел 1</li>
      <li>Подраздел 2</li>
      <li>Подраздел 3</li>
    </ul>
  </li>
  <li>
    <span class="toggle__hidden">Раздел №3</span>
    <ul class="hidden">
      <li>Подраздел 1</li>
      <li>Подраздел 2</li>
      <li>Подраздел 3</li>
    </ul>
  </li>
</ul>

CSS 
.toggle__hidden {cursor:pointer}
.hidden {display:none}

JS 
// вешаем событие onclick на нужный нам контейнер с id="js_example"
document.querySelector('#js_example').onclick = showUL;

// функция выполняется при клике внутри элемента с id="js_example"
function showUL(e) {
  // если элемент на который кликнули содержит класс toggle__hidden
  if (e.target.classList.contains('toggle__hidden')){
    // и следующий за ним элемент - это тег <ul>
    if (e.target.nextElementSibling.tagName == "UL"){
      // чередуем класс hidden (либо добавляем, либо убираем)
      e.target.nextElementSibling.classList.toggle('hidden');
    }
  }
}

В этом примере необходимо соблюдать порядок тегов внутри нашего выпадающего меню. Так как если после тега с селектором toggle__hiden будет идти не тег ul, то чередование класса hidden у него не будет происходить и выпадающий список не будет раскрываться.
