CSS cвойство outline универсальное свойство, одновременно устанавливающее цвет, стиль и толщину внешней границы на всех четырёх сторонах элемента. Свойство outline может быть с одним, двум или тремя значениями. Порядок следования значений не имеет значения. Значения свойства не наследуются, анимируются.

div {
  outline: outline-color || outline-style || outline-width;
}
  • outline-color - задаёт цвет линии допустимом для CSS формате.
  • outline-style - задаёт стиль линии.
  • outline-width - задаёт толщиу границы.
Если для свойства outline установлено значение 0 или none, то это удалит стандартную браузерную обводку, появляющуюся при фокусе на элемент.
В отличие от линии, задаваемой через border, у свойства outline есть следующие особенности:
  • не влияет на размеры и положение самого элемента;
  • не занимает место, не влияет на окружающие элементы и отображается поверх них;
  • свойство border-radius для старых браузеров не действует;
  • применяется сразу ко всем четырём сторонам - нельзя задать параметры линии на отдельных сторонах элемента;

CSS cвойство outline-color

CSS cвойство outline-color задаёт цвет линии-обводки элемента. Обводка — это линия вокруг элемента за пределами рамки (border). В отличие от рамки элемента, обводка находится вне границ элемента и может перекрывать другое содержимое. В качестве значения принимает код цвета в любом css формате. Значение по умолчанию invert, не наследуется, анимируется.

div {
  outline-color: invert | цвет;
}
  • invert - значение по умолчанию; цвет будет вычислен как противоположный цвету фона элемента. Если браузер не поддерживает это ключевое слово, то задается значение currentcolor.
  • цвет - цвет в любом допустимом для CSS формате.

CSS cвойство outline-style

CSS cвойство outline-style устанавливает стиль внешней границы элемента. Значения свойства аналогичны значениям свойста border-style. Значение по умолчанию none (т.е. её нет) для всех элементов кроме input.

div {
  outline-style: auto | none | dotted | dashed | solid |
                 double | groove | ridge | inset | outset;
}

CSS cвойство outline-width

CSS cвойство outline-width задаёт толщину обводки элемента. Обводка — линия вокруг элемента за пределами рамки (border). В отличие от свойства border-width, для outline-width нельзя задать границу для каждой стороны элемента индивидуально. Чтобы outline-width работал, необходимо установить у свойства outline-style любое значение кроме none.

div {
  outline-width: thin | medium | thick | размер;
}
  • thin - тонкая линия, обычно составляет 1 пиксель.
  • medium - линия средней толщины (обычно 3 пикселя).
  • thick - линия большой толщины (обычно 5 пикселей).
  • размер - любые единицы размера принятые в CSS.

CSS cвойство outline и border-radius

В старых браузеров рамка, задаваемая через свойство outline была прямоугольной формы, даже не смотря на то, что сам элемент имел значение свойства border-radius отличное от нуля.

CSS cвойство outline-offset

CSS cвойство outline-offset устанавливает расстояние между рамкой, созданной с помощью свойства outline, и краем элемента или его границей добавленной через border. Отрицательное значение отображает рамку внутри элемента, положительное - вокруг элемента.

div{
  outline: 2px dashed #000;
  outline-offset: -10px;
}
