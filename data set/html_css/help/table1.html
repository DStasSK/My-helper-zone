Каждая таблица состоит из строк и ячеек, минимально может быть одна строка и одна ячейка. Кроме того, таблице можно задать заголовок - <caption>, который кратко описывает её содержимое, указать «шапку» <thead>, «тело» - <tbody> и «подвал» таблицы - <tfoot>. Они нужны для стилевого или смыслового выделения части данных.

<tbody> может быть один или несколько. Например, в некоторых строках имеются объединения ячеек по вертикали и чтобы строки при наведении на них курсора мыши выделялись как требуется, их объединяют в <tbody> согласно требуемой логике.

Структура таблицы

HTML 4HTML 5

В HTML4 элемент <tfoot> должен располагаться перед элементом <tbody>, но после <thead>, если они присутствуют, при этом содержимое <tfoot> отображается внизу таблицы.

В HTML5 <tfoot> идёт после всех элементов <caption>, <colgroup>, <thead>, <tbody>.

<table>
   <caption>…</caption>
   <thead>…</thead>
   <tfoot>…</tfoot>
   <tbody>…</tbody>
</table>
 
<table>
   <caption>…</caption>
   <thead>…</thead>
   <tbody>…</tbody>
   <tfoot>…</tfoot>
</table>

Если элементов <thead> и <tbody> в таблице нет, тогда <tfoot> идёт после открывающего тега <table> или после необязательного элемента <caption>.

Если указанных элементов в таблице нет, тогда <tfoot> идёт после всех <tr>.

<table>
   <caption>…</caption>
   <tfoot>…</tfoot>
   <tr><td>…</td></tr>
</table>
 
<table>
   <tr><td>…</td></tr>
   <tr><td>…</td></tr>
   <tfoot>…</tfoot>
</table>

Тэги таблицы

<table>

Элемент <table> (от англ. table — таблица) служит контейнером для элементов, определяющих содержимое таблицы. Любая таблица состоит из строк и ячеек, которые задаются с помощью элементов <tr> и <td>. Внутри <table> допустимо использовать следующие элементы: <caption>, <col>, <colgroup>, <tbody>, <td>, <tfoot>, <th>, <thead> и <tr>.

<caption>

Элемент <caption> (от англ. caption — заголовок) предназначен для создания заголовка к таблице и может размещаться только внутри контейнера <table>, причём сразу после открывающего тега. Такой заголовок представляет собой текст, по умолчанию отображаемый перед таблицей и описывающий её содержание.

CSS свойство caption-side - выводит заголовок до или после таблицы.

<thead>

Элемент <thead> (от англ. table head — голова или шапка таблицы) предназначен для хранения одной или нескольких строк, которые представлены вверху таблицы. Допустимо использовать не более одного элемента <thead> в пределах одной таблицы, и он должен идти в исходном коде сразу после открывающего тега <table> или <caption> (если он есть).

<tbody>

Элемент <tbody> (от англ. table body — тело таблицы) предназначен для хранения одной или нескольких строк таблицы. Это позволяет создавать структурные блоки, к которым можно применять единое оформление через стили, а также управлять их видом через скрипты.

<tfoot>

Элемент <tfoot> (от англ. table foot — подвал таблицы) представляет собой «подвал» таблицы и отображается внизу таблицы. Предназначен для информации о колонках таблицы. В таблице разрешается использовать только один элемент <tfoot>.

<colgroup>

Элемент <colgroup> (от англ. column group — группа колонок) предназначен для задания ширины и стиля одной или нескольких колонок таблицы. Этот элемент позволяет уменьшить код таблицы за счёт сокращения повторяющихся атрибутов, и при его наличии браузер начинает показывать содержимое таблицы, не дожидаясь её полной загрузки. Элемент <colgroup> можно использовать в комбинации с <col>, который определяет характеристики одной или нескольких колонок.

Обычно закрывающий тег не требуется, но если <colgroup> выступает как контейнер для элементов <col>, тогда следует добавить тег </colgroup> в конце группы.

Разница между свойствами <colgroup> и <col> не очень велика и состоит в следующем. <colgroup> позволяет объединять колонки в определённые группы, также при добавлении атрибута rules со значением groups к элементу <table> браузер будет рисовать линию только между колонками, созданными с помощью <colgroup>. В остальных случаях поведение колонок назначенных через элементы <colgroup> и <col> идентично.

Для <colgroup> допустимо использовать следующие стилевые свойства: border, background, width, visibility. Остальные свойства не оказывают никакого эффекта на элемент.

<col>

Элемент <col> (от англ. column — колонка) задаёт ширину и другие характеристики одной или нескольких колонок таблицы. При наличии этого элемента браузер начинает показывать содержимое таблицы, не дожидаясь её полной загрузки. <col> можно использовать совместно с элементом <colgroup>, который задаёт группу колонок, обладающих общими характеристиками.

Для <col> допустимо использовать следующие стилевые свойства: border, background, width, visibility. Остальные свойства не оказывают никакого эффекта на элемент.

<th>

Элемент <th> (от англ. table header cell — ячейка заголовка таблицы) предназначен для создания одной ячейки таблицы, которая обозначается как заголовочная. Текст в такой ячейке отображается браузером обычно жирным шрифтом и выравнивается по центру. Элемент <th> должен размещаться внутри контейнера <tr>.

<table>
   <tr>
      <th>...</th>
   </tr>
</table>

<tr>

Элемент <tr> (от англ. table row - строка таблицы) служит контейнером для создания строки таблицы. Каждая ячейка в пределах такой строки устанавливается с помощью элемента <th> или <td>.

<table>
   <tr>
      <td>...</td>
   </tr>
</table>

<td>

Элемент <td> (от англ. table row - строка таблицы) предназначен для создания одной ячейки таблицы. Должен размещаться внутри контейнера <tr>, который в свою очередь располагается внутри <table>.

К <td> можно применять стили. Но margin не даст результата, потому что ячейка не может сместиться со своего положения в таблице.

Атрибуты тега <table>

Все данные атрибуты устарели, вместо них необходимо использовать стили.
<align> <background> <bgcolor> <border> <bordercolor> <cellpadding> <cellspacing> <cols> <frame> <rules> <summary> <width>
align Задаёт выравнивание таблицы по краю окна браузера. 
<table align="left | center | right">...</table>
Для выравнивания таблицы по центру через СSS, ей надо указать ширину через стилевое свойство width и задать margin со значением auto.
background Определяет изображение, которое будет использоваться в качестве фонового рисунка таблицы. В отличие от обычных изображений, для фона не устанавливаются ширина и высота, и он всегда отображается в натуральную величину с масштабом 100%. Если рисунок по размеру меньше ширины или высоты таблицы, то картинка повторяется по горизонтали вправо и вниз, выстраиваясь, как мозаика. 
<table background="адрес">...</table>
Чтобы задать фоновую картинку для таблицы, используют стилевое свойство background. 
table {background: url()}
bgcolor Цвет фона таблицы. 
<table bgcolor="цвет">...</table>
Для изменения цвета фона таблицы используйте стилевое свойство background-color. 
table {background: цвет}
border Устанавливает толщину рамки таблицы в пикселях. По умолчанию рамка изображается трёхмерной, но если используется атрибут bordercolor элемента <table>, то вид рамки меняется в зависимости от браузера. Когда в <table> используется атрибут border без значений (<table border>), то браузер отображает рамку толщиной один пиксель. Значение по умолчанию - 0. 
<table border="число">...</table>
Для изменения толщины и цвета рамки таблицы используйте стилевое свойство border.
bordercolor Устанавливает цвет рамки таблицы. Рамка обычно рисуется как трёхмерная, добавление атрибутов bordercolor и border к элементу <table> создаёт однотонную линию. 
<table bordercolor="цвет">...</table>
Для изменения цвета рамки таблицы используйте стилевое свойство border
cellpadding Отступ от рамки до содержимого ячейки. Этот атрибут добавляет пустое пространство к ячейке, увеличивая тем самым её размеры. При отсутствии границ особого значения этот атрибут не имеет, но может помочь, когда требуется установить пустой промежуток между ячеек. Значения - любое целое значение в пикселях или процентах. 
<table cellpadding="число">...</table>
В CSS вместо атрибута cellpadding используйте стилевое свойство padding, добавляя его к селектору td или th. 
td {padding:10px;}
cellspacing Задаёт расстояние в пикселях между внешними границами ячеек. Если установлен атрибут border, толщина границы принимается в расчёт. 
<table cellspacing="число">...</table>

Значение по умолчанию: 0 (когда нет атрибута border); 2 (когда атрибут border установлен).

Вместо атрибута cellspacing используйте стилевое свойство border-spacing, добавляя его к селектору table. 
table {border:1px solid #333; border-spacing: 5px}
cols Задаёт количество столбцов в таблице, помогая браузеру в подготовке к её отображению. Без этого атрибута таблица будет показана только после того, как всё содержимое таблицы будет загружено в браузер и проанализировано. Использование атрибута cols позволяет несколько ускорить отображение содержимого таблицы. 
<table cols="число">...</table>
frame Сообщает браузеру, как отображать границы вокруг таблицы. Толщина границы указывается с помощью атрибута border элемента <table>. 
<table frame="void | border | above | below |
              hsides | vsides | rhs | lhs">...</table>
Для отображения границ на разных сторонах таблицы воспользуйтесь стилевыми свойствами border-left, border-right, border-top и border-bottom, которые задают границы, соответственно, слева, справа, сверху и снизу таблицы.
rules Сообщает браузеру, где отображать границы между ячеек. Цвет границы указывается с помощью атрибута bordercolor, толщина внешней рамки таблицы — через атрибут border. По умолчанию рамка рисуется вокруг каждой ячейки, образуя тем самым сетку; толщина таких линий составляет 1px. 
<table rules="all | groups | cols | none | rows">...</table>
Для отображения границ на разных сторонах ячеек воспользуйтесь стилевыми свойствами border-left, border-right, border-top и border-bottom, которые задают границы, соответственно, слева, справа, сверху и снизу ячеек таблицы.
summary Краткое описание таблицы. В отличие от элемента <caption> содержимое summary никак не отображается и в браузере не выводится, однако может использоваться поисковыми системами или речевыми браузерами. 
<table summary="текст">...</table>
width Ширина таблицы в пикселях или процентах. Если общая ширина содержимого превышает указанную ширину таблицы, то браузер будет пытаться «втиснуться» в заданные размеры за счёт форматирования текста. В случае, когда это невозможно, например, в таблице находятся изображения, атрибут width будет проигнорирован, и новая ширина таблицы будет вычислена на основе её содержимого. 
<table width="значение">...</table>
Для изменения ширины таблицы через стили воспользуйтесь свойством width, применяя его к селектору table.

Атрибуты тега <caption>

Данные атрибуты устарели, вместо них необходимо использовать стили.
align Определяет выравнивание заголовка относительно таблицы. Результат его действия зависит от используемого браузера и установленного значения. 
<caption align="left | right | top | bottom">justify">...</caption>
Для выравнивания заголовка по горизонтали используйте стилевое свойство text-align. Чтобы задать положение заголовка до или после таблицы, воспользуйтесь свойством caption-side. 
caption {
   text-align:center;
   caption-side:bottom;
}

Атрибуты тега <thead>

Данные атрибуты устарели, вместо них необходимо использовать стили.
<align> <char> <charoff> <bgcolor> <valign>
align Задаёт выравнивание по горизонтали для всех ячеек в пределах <thead>. Если требуется применить разное выравнивание для каждой ячейки, воспользуйтесь стилями или используйте атрибут align для элемента <td> или <th>. 
<thead align="left | center | right | justify">...</thead>
Для выравнивания содержимого ячеек внутри <thead> используйте стилевое свойство text-align, применяя его к селектору thead.
char Выравнивание содержимого ячеек относительно заданного символа. Работает только в том случае, если значение атрибута align установлено как char. 
<table>
   <thead align="char" char="символ">...</thead>
</table>
Данный атрибут не поддерживается.
charoff Сдвигает содержимое ячейки относительно заданного символа. Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячейки влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" - сместить на три символа влево. 
<table>
   <thead align="char" char="символ"  charoff="число">...</thead>
</table>
Данный атрибут не поддерживается.
bgcolor Цвет фона ячеек, которые расположены внутри контейнера <thead>. 
<thead bgcolor="цвет">...</thead>
Чтобы задать цвет фона внутри <thead> используйте стилевое свойство background-color, применяя его к селектору thead.
valign Задает вертикальное выравнивание содержимого ячеек в контейнере <thead>. По умолчанию контент ячейки располагается по её вертикали в центре. 
<thead valign="baseline | bottom | middle | top">...</thead>
Для изменения выравнивания по высоте внутри <thead> воспользуйтесь свойством vertical-align, добавляя его к селектору thead.

Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" - сместить на три символа влево.

valign Задает вертикальное выравнивание содержимого ячеек в контейнере <tfoot>. По умолчанию контент ячейки располагается по вертикали в центре. 
<tfoot valign="baseline | bottom | middle | top">...</tfoot>
Для изменения выравнивания по высоте внутри <tfoot> воспользуйтесь свойством vertical-align, добавляя его к селектору tfoot.

Атрибуты тега <colgroup>

<align> <char> <charoff> <span> <valign> <width>

Атрибуты тега <th>

Некоторые атрибуты устарели или не поддерживаются, вместо них необходимо использовать стили.
<abbr> <align> <axis> <background> <bgcolor> <bordercolor> <char> <charoff> <colspan> <headers> <height> <nowrap> <rowspan> <scope> <valign> <width>
abbr Краткое описание содержимого ячейки. Текст внутри abbr браузерами не отображается и предназначен в первую очередь для доступа людей к содержанию с помощью вспомогательных технологий вроде речевых браузеров. Значение представляет сокращённую версию текста внутри элемента <th>. 
<th abbr="текст">...</th>
align Определяет выравнивание содержимого ячейки по горизонтали. 
<th align="left | center | right | justify | char">...</th>
Для выравнивания содержимого ячеек внутри <th> используйте стилевое свойство text-align, применяя его к селектору th.
axis Группирует ячейки связанные между собой похожей информацией. Значения: любой текст, представляющий собой произвольное имя ячейки. Для связывания ячеек между собой значения атрибута axis должны совпадать. 
<th axis="текст">...</th>
Данный атрибут не поддерживается браузерами.
background Задает фоновый рисунок в ячейке. В отличие от обычных изображений, для фона не устанавливаются ширина и высота, и он всегда отображается в натуральную величину с масштабом 100%. Если рисунок по размеру меньше ширины или высоты ячейки, то картинка повторяется по горизонтали вправо и вниз, выстраиваясь, как мозаика. 
<th background="адрес">...</th>
Чтобы задать фоновое изображение для ячейки используйте стилевое свойство background, добавляя его к селектору th.
bgcolor Цвет фона ячейки. 
<th bgcolor="Цвет">...</th>
bordercolor Цвет рамки. 
<th bordercolor="Цвет">...</th>
char Выравнивает содержимое ячейки по заданному символу. Эта возможность может пригодиться для выравнивания колонки чисел по точке. Работает только в том случае, если значение атрибута align установлено как char. 
<th align="char" char="символ">...</th>
Данный атрибут не поддерживается браузерами.
charoff Смещает содержимое ячейки относительно заданного символа. Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячейки влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" — сместить на три символа влево. 
<th align="char" char="," charoff="-2">...</th>
Данный атрибут не поддерживается браузерами.
colspan Устанавливает число ячеек, которые должны быть объединены по горизонтали. 
<th colspan="число">...</th>
headers

Позволяет связать ячейки таблицы с заголовками. Этот атрибут предназначен для повышения доступности таблицы пользователям речевых браузеров, в обычных браузерах результат применения атрибута headers не заметен.

Для связывания ячеек между собой, ячейке в <th> пишется атрибут id, а в другой ячейке — атрибут headers со значением, совпадающим со значением id. 
<th id="идентификатор">...</th>
<th headers="идентификатор">...</th>
height Браузер сам устанавливает высоту таблицы и её ячеек исходя из их содержимого. Однако при использовании атрибута height высота ячеек будет изменена. Здесь возможны два варианта. Если значение height меньше, чем содержимое ячейки, то этот атрибут будет проигнорирован. В случае, когда установлена высота ячейки, превышающая её содержимое, добавляется пустое пространство по вертикали. 
<th height="значение">...</th>
nowrap Добавление атрибута nowrap к элементу <th> заставляет текст внутри ячейки отображаться без переносов, одной сплошной строкой. Неправильное использование этого атрибута может привести к тому, что таблица будет слишком широкой и не поместится целиком на веб-страницу. 
<th nowrap>...</th>
Чтобы запретить перенос текста в ячейке таблицы используйте стилевое свойство white-space со значением nowrap, добавляя его к селектору th.
rowspan Устанавливает число ячеек, которые должны быть объединены по вертикали. 
<th rowspan="число">...</th>
scope Атрибут scope связывает между собой ячейки с заголовком и обычные ячейки. По своему действию напоминает атрибут headers, но используется для простых таблиц. Атрибут предназначен для экранных ридеров вроде речевых браузеров, в обычных браузерах результат добавления scope никак не заметен. 
<th scope="col | colgroup | row | rowgroup">...</th>
Значение colgroup применяется в таблице, в которой используется элемент <colgroup> или при объединении ячеек с помощью атрибута colspan. Значение rowgroup требуется при объединении ячеек через атрибут rowspan.
valign Устанавливает вертикальное выравнивание содержимого ячейки. По умолчанию содержимое ячейки располагается по центру её вертикали. 
<th valign="baseline | bottom | middle | top">...</th>
Для изменения выравнивания в ячейках воспользуйтесь свойством vertical-align, добавляя его к селектору th
width Задаёт ширину ячейки. Если общая ширина содержимого превышает указанную ширину ячейки, то браузер будет пытаться «втиснуться» в заданные размеры за счёт форматирования текста. В случае, когда это невозможно, например, в ячейке находятся изображения, атрибут width будет проигнорирован, и новая ширина ячейки будет вычислена на основе её содержимого. 
<th width="значение">...</th>

<td nowrap>...</td>
Чтобы запретить перенос текста в ячейке таблицы используйте стилевое свойство white-space со значением nowrap, добавляя его к селектору td.
rowspan Устанавливает число ячеек, которые должны быть объединены по вертикали. Максимально допустимое значение равно 65534. Если значение установлено как 0, то ячейки объединяются до конца раздела таблицы (<thead>, <tbody> или <tfoot>) или самой таблицы. 
<td rowspan="число">...</td>
scope Атрибут scope связывает между собой ячейки с заголовком и обычные ячейки. По своему действию напоминает атрибут headers, но используется для простых таблиц. Атрибут предназначен для экранных ридеров вроде речевых браузеров, в обычных браузерах результат добавления scope никак не заметен. 
<td scope="col | colgroup | row | rowgroup">...</td>
Значение colgroup применяется в таблице, в которой используется элемент <colgroup> или при объединении ячеек с помощью атрибута colspan. Значение rowgroup требуется при объединении ячеек через атрибут rowspan.
valign Устанавливает вертикальное выравнивание содержимого ячейки. По умолчанию содержимое ячейки располагается по центру её вертикали. 
<td valign="baseline | bottom | middle | top">...</td>
Для изменения выравнивания в ячейках воспользуйтесь свойством vertical-align, добавляя его к селектору td
width Задаёт ширину ячейки. Если общая ширина содержимого превышает указанную ширину ячейки, то браузер будет пытаться «втиснуться» в заданные размеры за счёт форматирования текста. В случае, когда это невозможно, например, в ячейке находятся изображения, атрибут width будет проигнорирован, и новая ширина ячейки будет вычислена на основе её содержимого. 
<td width="значение">...</td>

border-collapse (CSS)

Применяется к элементу <table> или к элементам, у которых свойство display установлено как table или inline-table. Свойство наследуется.

Устанавливает, как отображать границы вокруг ячеек таблицы. Это свойство играет роль, когда для ячеек установлена рамка, тогда в месте стыка ячеек получится линия двойной толщины. Значение collapse заставляет браузер анализировать подобные места в таблице и убирать в ней двойные линии. При этом между ячейками остается только одна граница, одновременно принадлежащая обеим ячейкам. То же правило соблюдается и для внешних границ, когда вокруг самой таблицы добавляется рамка.

Значения:
collapse Линия между ячейками отображается только одна, также игнорируется значение атрибута cellspacing.
separate Вокруг каждой ячейки отображается своя собственная рамка, в местах соприкосновения ячеек показываются сразу две линии. Значение по умолчанию.
inherit Наследует значение родителя.
table {border-collapse: collapse;}
Internet Explorer до версии 8.0 не отменяет действие атрибута cellspacing.
Eсли у таблицы стоит свойство border-collapse, а у ячеек свойство position установлено в relative и они раскрашены, то IE, Edge и Firefox могут не прорисовывать границы. В таких случаях к ячейкам надо добавлять "background-clip: padding-box;".

border-spacing (CSS)

Применяется к элементу <table> или к элементам, у которых свойство display установлено как table или inline-table. Свойство наследуется.

Задает расстояние между границами ячеек в таблице. border-spacing не работает в случае, когда для таблицы установлено свойство border-collapse со значением collapse.

Одно значение устанавливает одновременно расстояние по вертикали и горизонтали между границами ячеек. Если значений два, то первое определяет горизонтальное расстояние, а второе — вертикальное.

table {
  border-collapse: separate;
  border-spacing: 10px 5px;
}
Если к элементу <table> добавлен атрибут cellspacing, то при использовании стилевого свойства border-spacing атрибут cellspacing не принимается во внимание и его значение игнорируется.

caption-side (CSS)

Применяется к <caption> или ко всем элементам, у которых значение display установлено как table-caption.

Определяет положение заголовка таблицы, который задается с помощью тега <caption>, относительно самой таблицы. Свойство caption-side выводит заголовок до или после таблицы, а выравнивание текста по правому или левому краю устанавливается через свойство text-align.

Браузер Firefox также поддерживает расположение заголовка слева или справа от таблицы, но эти значения не входят в спецификацию CSS.

Значения:
top Располагает заголовок по верхнему краю таблицы.
bottom Заголовок располагается под таблицей.
left Заголовок размещается слева от таблицы.
Это значение работает только в браузерах Firefox.
right Заголовок размещается справа от таблицы.
Это значение работает только в браузерах Firefox.
caption {
  caption-side: bottom; /* Заголовок под таблицей */
}

empty-cells (CSS)

Это свойство применяется к <td>, <th> или к элементам, у которых display: table-cell.

Задает отображение границ и фона в ячейке, если она пустая. При одновременном добавлении к таблице свойства border-collapse со значением collapse, свойство empty-cells игнорируется.

Ячейка считается пустой в следующих случаях:
- нет вообще никаких символов;
- в ячейке содержится только перевод строки, символ табуляции или пробел;
- значение visibility установлено как hidden.

Добавление неразрывного пробела &nbsp; воспринимается как видимое содержание, т.е. ячейка уже будет непустой.

Значения:
show Отображает границу вокруг ячейки и фон в ней.
hide Граница и фон в пустых ячейках не отображается. Если все ячейки в строке пустые, то строка прячется целиком.
td {
  empty-cells: hide; /* Прячем пустые ячейки */
}
Internet Explorer до версии 7 включительно не поддерживает свойство empty-cells и отображает пустые ячейки всегда так, словно для них установлено empty-cells: hide.

table-layout (CSS)

Определяет, как браузер должен вычислять ширину ячеек таблицы, основываясь на ее содержимом. Применяется к тегу <table> или к элементу, у которого значение display установлено как table или inline-table.

table {
  table-layout: fixed; /* Фиксированная ширина ячеек */
  width: 100%;
  }
Значения:
auto Браузер загружает всю таблицу, анализирует ее для определения размеров ячеек и только после этого отображает.
fixed Ширина колонок в этом случае определяется либо с помощью тега <col>, либо вычисляется на основе первой строки. Если данные о форматировании первой строки таблицы по каким-либо причинам получить невозможно, в этом случае таблица делится на колонки равной ширины. При использовании этого значения, содержимое, которое не помещается в ячейку указанной ширины, будет «обрезано» либо наложено поверх ячейки. Это зависит от используемого браузера, но в любом случае ширина ячейки меняться не будет. Для корректной работы этого значения обязательно должна быть задана ширина таблицы.
inherit Наследует значение родителя.
