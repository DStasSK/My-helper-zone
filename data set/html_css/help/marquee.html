<marquee>

Элемент <marquee> создаёт бегущую строку на странице. На самом деле содержимое контейнера <marquee> не ограничивается строками и позволяет перемещать (скролировать) любые элементы веб-страницы — изображения, текст, таблицы, элементы форм и т. д. Перемещение можно задать не только по горизонтали, но и по вертикали, в этом случае указываются размеры области, в которой будет происходить движение.

В качестве альтернативы <marquee> применяется стилевое свойство animation.

Это нестандартный элемент (но поддерживается всеми браузерами), взамен него используйте стили.
<marquee behavior="alternate" direction="left" bgcolor=""> ... </marquee>
Атомистика контролирует трансцендентальный катарсис.

Атрибуты

behaviorЗадаёт тип движения содержимого
bgcolorЦвет фона
directionУказывает направление движения содержимого
heightВысота области прокрутки
hspaceГоризонтальные поля вокруг контента
loopЗадаёт, сколько раз будет прокручиваться содержимое
scrollamountСкорость движения контента
scrolldelayВеличина задержки между движениями
truespeedОтменяет встроенный ограничитель скорости
vspaceВертикальные поля вокруг содержимого
widthШирина области прокрутки

behavior

Устанавливает способ движения содержимого внутри элемента <marquee>. Значние по умолчанию - scroll

<marquee behavior="alternate | scroll | slide">...</marquee>
Значения:
alternateКонтент перемещается между правым и левым краями элемента.
scrollКонтент перемещается в направлении, заданным атрибутом direction, затем скрывается за пределами области, после чего начинает движение с начала.
slideКонтент перемещается в направлении, заданным атрибутом direction, доходит до края области и останавливается.
Lorem ipsum dolor sit amet... behavior="alternate" Lorem ipsum dolor sit amet... behavior="scroll" Lorem ipsum dolor sit amet... behavior="slide"

bgcolor

Устанавливает цвет фона под содержимым элемента <marquee>. Значение по умолчанию - cовпадает с цветом фона веб-страницы, обычно это белый цвет.

<marquee bgcolor="<цвет>">...</marquee>

direction

Содержимое контейнера <marquee> может перемещаться в четырёх направлениях, не только влево и вправо, но также вверх и вниз. Напрпавление определяестя значением свойства direction. Значение по умолчанию - left

<marquee direction="down | left | right | up">...</marquee>
downДвижение сверху вниз.
leftДвижение справа налево.
rightДвижение в правую сторону.
upДвижение вверх.
Оно приближается... direction="down"

height

Элемент <marquee> отображается как прямоугольник, ширина и высота которого устанавливаются, соответственно, с помощью атрибутов width и height. Прокрутка содержимого происходит в пределах заданных размеров.

Допускается использовать значения в пикселях или процентах. Если установлена процентная запись, то размеры <marquee> вычисляются относительно родительского элемента.

<marquee height="<значение>">...</marquee>
Компульсивность вероятна... height="50"

hspace

Атрибуты hspace и vspace предназначены для добавления пустого пространства вокруг содержимого <marquee>. hspace устанавливает поля слева и справа от элемента (по горизонтали), а vspace — сверху и снизу (по вертикали). В качестве значения используется любое целое положительное число в пикселях.

Поля не увеличивают фоновую область под содержимым и служат для изменения расстояния между <marquee> и другими элементами веб-страницы - что-то вроде margin.

<marquee hspace="<число>">...</marquee>
Компульсивность вероятна... hspace="100" vspace="10"

loop

Устанавливает, сколько раз скролировать содержимое элемента <marquee>. После того, как заданное число раз отсчитано, содержимое остаётся в конечной точке.

Значения - любое целое положительное число или -1 для бесконечного воспроизведения движения. Значение по умолчанию -1.

<marquee loop="<число>">...</marquee>

scrollamount

Анимация содержимого элемента <marquee> происходит за счёт периодического стирания информации и отображения её на новом месте. Атрибут scrollamount устанавливает расстояние в пикселях между старым и новым положениями, которые влияют на скорость движения и плавность хода. Чем выше значение, тем быстрее перемещается содержимое контейнера. Если значение равно нулю, то движения не происходит.

Значения - любое целое положительное число в пикселях. Значение по умолчанию -6.

<marquee scrollamount="<число>">...</marquee>
Компульсивность вероятна... scrollamount="1" Компульсивность вероятна... scrollamount="10" Компульсивность вероятна... scrollamount="20"

scrolldelay

На скорость скролирования содержимого <marquee>, как и изменение позиции, влияет частота обновления содержания. Атрибут scrolldelay задаёт время задержки между движениями в миллисекундах, чем ниже значение, тем выше скорость движения, и наоборот.

Значения - любое целое положительное число миллисекунд. Значение по умолчанию: 85.

<marquee scrolldelay="<число>">...</marquee>
Компульсивность вероятна... scrolldelay="50" Компульсивность вероятна... scrolldelay="100"

truespeed

Если содержимое <marquee> перемещается слишком быстро, то срабатывает встроенный ограничитель скорости, который насильно замедляет скролирование. Это сделано для того, чтобы сохранить читабельность текста. Если вы намерены обойти это ограничение, добавьте атрибут truespeed, он заставляет прокручиваться содержимое с заданной скоростью. По умолчанию этот атрибут выключен.

Для атрибута scrolldelay ограничение скорости начинается со значения 60 миллисекунд и ниже.

Поддерживается только в IE и Ffox
<marquee truespeed>...</marquee>
