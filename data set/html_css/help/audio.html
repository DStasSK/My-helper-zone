Элемент <audio> (от англ. audio — звук) добавляет, воспроизводит и управляет настройками аудиозаписи на веб-странице. Путь к файлу задаётся через атрибут src или вложенный элемент <source>. Внутри контейнера <audio> можно написать текст, который будет выводиться в браузерах, не работающих с этим элементом.

Для универсального воспроизведения в указанных браузерах аудио кодируют с помощью разных кодеков и добавляют файлы одновременно через элемент <source>.

<audio src="адрес" controls></audio>

<audio controls>
  <source src="адрес1" type="тип">
  <source src="адрес2" type="тип">
</audio>

Атрибуты

autoplay При наличии этого атрибута аудио начинает воспроизводиться автоматически после загрузки страницы. По умолчанию этот атрибут выключен. Атрибут autoplay отменяет действие атрибута preload. 
<audio autoplay="autoplay" controls muted
       src="media/music.mp3"></audio>
controls Добавляет панель управления к аудиофайлу. Вид панели и её содержимое зависит от браузера и может в себя включать: кнопку воспроизведения, паузы, перемотки, ползунок для изменения уровня громкости и др. В качестве значения указывается controls, также допустимо писать атрибут без значения. 
<audio controls="controls"></audio>
loop Повторяет воспроизведение звука с начала после его завершения. По умолчанию этот атрибут выключен. 
<audio loop="loop"></audio>
muted Отключает звук при воспроизведении музыки. В качестве значения указывается muted, также допустимо вообще не указывать никакое значение. 
<audio muted="muted"></audio>
preload Используется для предварительной загрузки аудиофайла или его данных вместе с загрузкой веб-страницы. Этот атрибут игнорируется, если установлен атрибут autoplay. Пустое значение атрибута воспринимается как auto. Значение по умолчанию - none. 
<audio preload="none | metadata | auto"></audio>
src Указывает путь к воспроизводимому файлу. Для этой же цели также можно использовать элемент <source>. 
<audio src="media/music.mp3"></audio>

Пример 1.

При добавлении песни на сайт в Safari отображается плеер, но нет громкости, только плэй, тогда как в FFox все нормально. Для Safari что то еще нужно прописывать?

В Safari нужно подрубить style чтобы отобразилась громкость. Если просто указать controls, то он как будто обрубает плеер, а если добавить такой код например --> style="width:100%; max-width:600px;" то плеер отобразится полностью.
