HTML элемент <meter>

Элемент <meter> (от англ. meter — измеритель, счётчик), отображает значение в виде рисунка, на котором цветом помечается текущее значение в указанном диапазоне. Браузеры, которые не поддерживают данный тэг, выводят текст, написанный внутри <meter>.

Используется для вывода значения в некотором известном диапазоне. Применяется преимущественно для отображения числовых значений, например, количества результатов поиска, объёма жидкости, давления и др.

В качестве значения указывается целое или дробное число. Допускается использование отрицательных значений.

<meter value="число">текст</meter>
13 6 11 6 16 16 11 10.3

Атрибуты

value Устанавливает значение. обязательный
Значение должно быть в диапазоне, задаваемым атрибутами min и max.

36,6 градусов

<meter value="36.6" min="35" max="42">36,6 градусов</meter>
min Задаёт минимально возможное значение. Значение по умолчанию - 0.
Обязательно должно соблюдаться условие
min ≤ value ≤ max
max Задаёт максимально возможное значение. Значение по умолчанию - 1.
low Определяет предел, при достижении которого значение считается низким.
Обязательно должно соблюдаться условия
( min ≤ value ≤ max ) и ( low ≤ high ) 
<meter value="число" low="число"></meter>
high Определяет предел, при достижении которого значение считается высоким.
optimum Определяет наилучшее или оптимальное значение.
Обязательно должно соблюдаться условие
min ≤ optimum ≤ max

Пример

Температура воды

Низкая Нормальная Горячая Горячая Кипяток

<meter max="100" high="60" low="50" value="80">Горячая</meter>
<meter max="100" high="60" low="50.1" value="80">Горячая</meter>
<meter max="100" value="100">Кипяток</meter>
