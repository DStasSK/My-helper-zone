Элемент <param>

  • Атрибуты
    name type value valuetype

Элемент <param> (от англ. parameter — параметр) предназначен для передачи значений параметров Java-апплетам или объектам веб-страницы, созданным с помощью элементов <applet> или <object>. Такой подход позволяет прямо в коде HTML-документа изменять характеристики объекта без его дополнительной компиляции. Количество одновременно используемых элементов <param> может быть больше одного и для каждого из них задаётся пара «имя=значение» через атрибуты name и value.

<param name="имя" value="значение">

Атрибуты

name Имя параметра. Это имя должно быть программно заложено в код объекта. В качестве имени используется набор символов, включая числа и буквы.
type MIME-тип объекта. устарел
Используется только в случае, когда valuetype установлен как ref. 
<param type="MIME-тип">
value Значение параметра. Этот атрибут не является обязательным, поскольку для объекта вполне может быть достаточно одного имени, задаваемым атрибутом name. В качестве значения может использоваться строка или число.
valuetype

Тип значения параметра. устарел

Атрибут обеспечивает механизм определения типа данных передаваемых родительскому объекту через атрибут value.

<param valuetype="data | object | ref">
<object width="400" height="320">
  <param name="src"
         value="video/snowman.mp4"
         valuetype="ref"
         type="video/mp4" />
</object>
