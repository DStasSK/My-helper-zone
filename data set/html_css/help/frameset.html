<frameset>

Элемент <frameset> (от англ. frame set — набор рамок) заменяет собой на веб-странице. Допустимо использовать вложенную структуру элементов, это позволяет разбить один фрейм на две и более области.

Определяет структуру фреймов на веб-странице. Фреймы разделяют окно браузера на отдельные области, расположенные вплотную друг к другу. В каждую из таких областей загружается самостоятельная веб-страница, определяемая с помощью элемента <frame>. С помощью фреймов веб-страница делится на два или более документа, которые обычно содержат навигацию по сайту и его контент. Механизм фреймов позволяет открывать документ в одном фрейме, по ссылке, нажатой в совершенно другом фрейме

<frameset>
   <frame>
</frameset>
Данный элемент устарел, не используйте его.
При использовании фреймов примите во внимание их следующие особенности:
  • Поисковые системы плохо работают с фреймовой структурой, поскольку на страницах, которые содержат контент, обычно нет ссылок на другие документы.
  • Фреймы скрывают адрес страницы на которой находится посетитель и устанавливаемый через элемент <title>, и всегда показывают только адрес сайта. По этой причине понравившуюся страницу невозможно поместить в раздел «Избранное» браузера.
  • Пользователь зачастую оказывается на сайте, совершенно не представляя, куда он попал, потому что всего лишь нажал на ссылку, полученную в поисковой системе. Чтобы посетителю сайта было проще разобраться, где он находится, на каждую страницу помещают название сайта, заголовок страницы и навигацию. Фреймы, как правило, нарушают данный принцип, отделяя заголовок сайта от содержания, а навигацию от контента. Представьте, что вы нашли подходящую ссылку в поисковой системе, нажимаете на неё, а в итоге открывается документ без названия и навигации. Чтобы понять, где мы находимся или посмотреть другие материалы, придётся редактировать путь в адресной строке, что в любом случае доставляет неудобство.
  • Большое число фреймов требует для браузера выделения больше памяти, чем обычно.

Парaметры.

border Толщина границы между фреймами. По умолчанию линия отображается трёхмерной, но используя атрибуты bordercolor и border можно настроить её вид по своему усмотрению.
Браузеры по-разному интерпретируют атрибуты <frameset> и отображают линию.
<frameset border="число">...</frameset>
bordercolor Определяет цвет видимых границ между фреймов. Цвет родительского элемента <frameset> может быть изменён с помощью дочернего элемента <frameset> или индивидуального элемента <frame>. 
<frameset bordercolor="цвет">...</frameset>
cols Устанавливает ширину или пропорции фреймов в виде колонок.
Pазмеры задаются: в пикселях (при этом px не указывается - только число); в процентах от доступной ширины фрейма; символом *, который обозначает всю свободную ширину, оставшуюся после указания размеров в пикселях или процентах. 
<frameset cols="размер, размер, ...">...</frameset>
rows Задаёт размер или пропорции фреймов в виде строк.
Pазмеры задаются: в пикселях (при этом px не указывается - только число); в процентах от доступной ширины фрейма; символом *, который обозначает всю свободную ширину, оставшуюся после указания размеров в пикселях или процентах. 
<frameset rows="размер, размер, ...">...</frameset>
frameborder Определяет, отображать рамку вокруг фрейма или нет. По умолчанию этот атрибут включен. 
<frameset frameborder="yes | no">...</frameset>
<frameset frameborder="1 | 0">...</frameset>

Пример.

<iframe src="frame/start.html" width="100%" height="200px" class="if1"></iframe>
		
<!-- start.html -->
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" 
  "http://www.w3.org/TR/html4/frameset.dtd">
<html>
  <head>
    <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
    <title>Frame Start</title>
  </head>
  <frameset rows="80,*" cols="*">
    <frame src="top.html" name="tf" scrolling="no" noresize>
    <frameset cols="80,*">
      <frame src="left.html" name="lf" scrolling="no" noresize>
      <frame src="main.html" name="mf">
    </frameset>
  </frameset>
</html>
