Элемент <ruby> (от англ. ruby — рубин) предназначен для добавления небольшой аннотации сверху или снизу от заданного текста. Такая форма записи преимущественно используется для идеографической письменности вроде китайского языка, но может применяться и для других языков, если требуется написать один текст над другим. Название происходит от слова «рубин», которым в типографии обозначается небольшой шрифт.

Сам элемент <ruby> выступает контейнером для <rt> (он и формирует аннотацию к тексту, после которого идёт) и <rp> (этот элемент предназначен для браузеров, которые не поддерживают <ruby>).

Браузеры, которые не поддерживают этот элемент, отобразят текст рядом по горизонтали и без изменения его размеров.

<ruby>
  текст
  <rt>аннотация</rt>
</ruby>

<rt>

Элемент <rt> (от англ. ruby text — текст рубина) добавляет аннотацию сверху или снизу от текста, заключенного в контейнер <ruby>. Сама аннотация выводится уменьшенным шрифтом.

<rp>

Элемент <rp> (от англ. ruby parenthesis — скобки рубина) Используется для вывода текста в браузерах, которые не поддерживают элемент <ruby>. В остальных браузерах текст, заключенный в контейнер <rp>, не отображается. Элемент <rp> должен идти до или после <rt>.

<ruby>
  текст
  <rp>текст</rp><rt>аннотация</rt><rp>текст</rp>
</ruby>

<rtc>

Элемент <rtc> (от англ. ruby text container — текстовый контейнер рубина) обозначает текстовый контейнер внутри <ruby>. В основном применяется в качестве описательной части или аннотации для иероглифов.

<ruby>
  текст
  <rtc>аннотация</rtc>
</ruby>
Поддерживается только в FireFox.

Как сделать чтобы аннотации были снизу?

Если вам не для азиатских языков использовать, а просто для оформления, то лучше вообще от <ruby> отказаться.
