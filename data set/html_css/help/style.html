<style>

<style> (от англ. style — стиль) применяется для определения стилей элементов веб-страницы. Элемент <style> необходимо использовать внутри контейнера <head>. Можно задавать несколько <style>.

Также тег <style> можно использовать непосредственно в теле документа изменяя параметры отдельных компонентов (тегов).

<!DOCTYPE HTML>
<html>
  <head>
    <meta charset="utf-8">
    <title>Тег STYLE</title>
    <style>
      html, body {margin:0; padding:0;}
    </style>
  </head>
  <body>
    <style>
      p {font-size:20px; color:red;}
    </style>
    <p>Текст</p>
    <div style="text-align:center;">Текст</div>
  </body>
</html>

Атрибуты

media

Устанавливает устройство вывода, для которого предназначена таблица стилей. Для каждого устройства можно определить свой собственный стиль, который бы учитывал специфику устройства и подгонял под него вид веб-страницы.

Можно устанавливать сразу несколько значений, перечисляя их через запятую. 
<style media="all | print | screen | speech">...</style>
type

Сообщает браузеру, какой синтаксис использовать, чтобы правильно интерпретировать стили. Как правило, браузеры корректно работают со стилями и при отсутствии этого атрибута, он необходим для некоторых старых браузеров, которые могут не распознать содержимое контейнера <style>.

В качестве значения используется MIME-тип. 
<style type="text/css">...</style>
В HTML4 этот атрибут является обязательным, в HTML5 его можно опустить.
<head>
  <style media="screen">
    .window {padding:10px;
     border:1px solid red; font-size:0.9em; color:#fc0;
     background: #333;}
  </style>
  <style media="print">
    .window {padding:10px;
     border:1px solid black; font-size:0.9em; color:black;
     font-family:Arial;}
  </style>
  <style type="text/css">
    a:link    {color:#003366}
    a:visited {color:#660066}
    a:hover   {color:#800000}
    a:active  {color:#FF0000}
  </style>
</head>
