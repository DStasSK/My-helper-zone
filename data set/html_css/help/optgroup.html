Элемент <optgroup>

Контейнер optgroup (от англ. option group — группа вариантов) представляет собой контейнер, внутри которого располагаются элементы option, объединённые в одну группу внутри элемента select.

Особенностью optgroup является то, что его нельзя выбрать как обычный элемент списка и он акцентируется с помощью жирного начертания, а все элементы, входящие в этот контейнер, смещаются вправо от своего исходного положения.

<select>
  <optgroup || disabled || label="текст">
    <option>...</option>
  </optgroup>
</select>
  • disabled - блокирует доступ к группе списка, при этом группа отображается другим цветом (обычно серым) и не доступна для выбора.
  • label - содержит текст, который будет входить в список в виде заголовка группы. Значения - любая текстовая строка.

<form action="handler.php">
   <select>
      <optgroup label="Цвет">
         <option value="c1">Апельсиновый</option>
         <option value="c2">Лимонный</option>
         <option value="c3">Персиковый</option>
      </optgroup>
      <optgroup label="Тон" disabled>
         <option value="t1">Светлый</option>
         <option value="t2">Нормальный</option>
         <option value="t3">Темный</option>
      </optgroup>
   </select>
   <p><input type="submit" value="Отправить"></p>
</form>
