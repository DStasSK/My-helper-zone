Элемент <textarea> представляет собой элемент формы для создания области, в которую можно вводить несколько строк текста. В отличие от элемента <input> в текстовом поле допустимо делать переносы строк, они сохраняются при отправке данных на сервер.

При оформлении многострочного поля применяются те же стилевые свойства, что и для однострочного текста. А именно можно изменять ширину (свойство width), высоту (height), границу (border), цвет текста и фона (color и background соответственно) и др.

Между тегами <textarea> и </textarea> можно поместить любой текст, который будет отображаться внутри поля.

<textarea cols="40" rows="5"></textarea>
<textarea cols="40" rows="5" style="resize:none;">текст</textarea>

CSS свойство resize

Указывает, можно ли пользователю изменять размеры текстового поля. Применяется к <textarea> или к любому элементу, у которого значение свойства overflow отличается от visible.

Все браузеры кроме Internet Explorer поддерживают изменение размера для <textarea>. Достаточно потянуть за правый нижний уголок, который по своему виду отличается от остальных.

Каждый браузер имеет свои предварительные настрой параметра resize, для кроссбраузерности необходимо предопределять его значение.
textarea {
   resize: none | both | horizontal | vertical;
}
Значения:
none размеры элемента не изменяются.
both можно изменять размеры элемента по горизонтали и вертикали.
horizontal можно изменять размеры элемента только по горизонтали.
vertical можно изменять размеры элемента только по вертикали.
IE, Edge, IOS Safari, Android WebView - не поддерживают данное свойство.

Атрибуты

cols disabled name readonly rows autocomplete autofocus dirname form maxlength minlength placeholder required wrap
cols

Ширина текстового поля, которое определяется числом символов моноширинного шрифта. Иными словами, ширина задаётся количеством близстоящих букв одинаковой ширины по горизонтали. Если размер шрифта изменяется с помощью стилей, ширина также соответственно меняется.

Значение по умолчанию: в HTML4 зависит от настроек браузера и операционной системы, в HTML5 — 20. 
<textarea cols="число"></textarea>
disabled Блокирует доступ и изменение текстового поля. Оно в таком случае отображается серым и недоступным для активации пользователем. Кроме того, такое поле не может получить фокус путем нажатия на клавишу Tab, мышью или другим способом. Тем не менее, такое состояние поля можно менять с помощью скриптов. Заблокированный в поле текст не передаётся на сервер. 
<textarea disabled></textarea>
name

Имя поля, предназначено для того, чтобы обработчик формы мог его идентифицировать. Как правило, это имя используется при отправке данных на сервер или для доступа к полю через скрипты.

В качестве имени используется набор символов, включая числа и буквы. JavaScript чувствителен к регистру, поэтому при обращении к элементу <textarea> по имени соблюдайте ту же форму написания, что и в атрибуте name. 
<textarea name="имя"></textarea>
readonly Устанавливает, что поле не может изменяться пользователем, в том числе вводиться новый текст или модифицироваться существующий. Тем не менее, состояние и содержимое поля можно менять с помощью скриптов. Также содержимое поле можно выделить и скопировать в буфер обмена. 
<textarea readonly></textarea>
rows

Высота текстового поля, которое определяется количеством отображаемых строк без прокрутки содержимого. Если размер шрифта изменяется с помощью стилей, высота поля также соответственно меняется.

Значение по умолчанию: в HTML4 зависит от настроек браузера и операционной системы, в HTML5 — 2. 
<textarea rows="число"></textarea>
autocomplete

Включает или отключает автозаполнение. Этот атрибут помогает заполнить текстовое поле текстом, который был введён в нём ранее. Значения сохраняет и подставляет браузер, при этом автозаполнение по соображениям безопасности может отключаться пользователем в настройках браузера и не может в таком случае управляться атрибутом autocomplete.

Автозаполнение конкретного поля формы привязывается к его атрибуту name и при смене значения теряется. Значение по умолчанию - зависит от настроек браузера. 
<textarea autocomplete="on | off"></textarea>
Данный атрибут не браузерами поддерживается.
autofocus Автоматически устанавливает фокус в текстовое поле. В таком поле можно сразу набирать текст без переключения курсора в текстовое поле. 
<textarea autofocus></textarea>
dirname Параметр, который передаёт на сервер направление текста. При наличии атрибута dirname на сервер отправляется пара имя/значение, где имя задаётся атрибутом dirname, а значение подставляется браузером. Для текста, идущего слева направо, значение будет ltr, а для текста справа налево (арабского, к примеру) значение будет rtl. В качестве значения - произвольная текстовая строка, которая служит именем параметра. Чтобы показать, что этот параметр указывает направление текста, обычно в конце строки добавляют .dir. 
<textarea dirname="строка"></textarea>

<form action="handler.php">
  <p>Комментарий:</p>
  <p><textarea name="comment" dirname="comment.dir"></textarea></p>
  <p><button type="submit">Отправить</button></p>
</form>

В данном примере на сервер будет отправлена строка comment.dir со значением ltr, поскольку для русского языка текст идёт слева направо.

IE, Edge, Ffox не поддерживают данный атрибут.
form Связывает текстовое поле с формой по её идентификатору (значение атрибута id элемента <form>). Такая связь необходима в случае, когда поле по каким-либо причинам располагается за пределами <form>. 
<textarea form="идентификатор"></textarea>
maxlength Устанавливает максимальное число символов, которое может быть введено пользователем в текстовом поле. Когда это количество достигается при наборе, дальнейший ввод становится невозможным. 
<textarea maxlength="число"></textarea>
minlength Задаёт минимальное число символов, которое может быть введено пользователем в текстовом поле. Если число символов меньше указанного значения, то браузер выдаёт сообщение об ошибке и не отправляет форму. 
<textarea minlength="число"></textarea>
IE, Edge, Safari и мобильные браузеры не поддерживают данный атрибут.
placeholder Выводит подсказывающий текст внутри текстового поля, который исчезает при получении фокуса или наборе текста. В качестве подсказки обычно применяется короткий текст, состоящий из одного слова или фразы. 
<textarea placeholder="текст"></textarea>
required Устанавливает текстовое поле обязательным для заполнения перед отправкой формы на сервер. Если в обязательном поле нет текста, браузер выведет сообщение, а форма отправлена не будет. Вид и содержание сообщения зависит от браузера и меняться пользователем не может. 
<textarea required></textarea>
wrap Параметры переноса строк. 
<textarea wrap="soft | hard"></textarea>
softСлова в текстовом поле переносятся механически, если они не поместились в размер области, но при отправке на сервер переносы не сохраняются. Значение по умолчанию.
hardДлинный текст, который самостоятельно не помещается в поле по ширине, автоматически переносится на новую строку и вместе с переносами отправляется на сервер. Для работы требуется наличие атрибута cols (ширина поля в символах).

Пример 1.

Введите ваш отзыв:

<form action="textarea1.php" method="post">
   <p><b>Введите ваш отзыв:</b></p>
   <p><textarea rows="10" cols="45" name="text"></textarea></p>
   <p><input type="submit" value="Отправить"></p>
</form>

Пример 2.

Введите комментарий

<form action="handler.php">
  <p><b>Введите комментарий</b></p>
  <p><textarea rows="10" cols="20" name="comment"></textarea></p>
  <p><input type="button" value="ОК" onClick="dataField(this.form)"></p>
</form>
