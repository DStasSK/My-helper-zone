Элемент <option>

  • Атрибуты
    disabled label selected value

Элемент <option> (от англ. option — вариант, выбор) определяет отдельные пункты списка, создаваемого с помощью контейнера <select>. Ширина списка определяется самым широким текстом, указанным в <option>, а также может изменяться с помощью стилей. Если планируется отправлять данные списка на сервер, то требуется поместить элемент <select> внутрь формы. Это также необходимо, когда к данным списка идёт обращение через скрипты.

Eсли требуется выделить сразу несколько пунктов списка через атрибут selected, то для <select> следует добавить атрибут multiple, иначе это всё работать не будет.

<select>
   <option>Пункт 1</option>
   <option>Пункт 2</option>
</select>

Атрибуты

disabled Блокирует элемент списка для его выбора. 
<option disabled>...</option>
label Атрибут предназначен для указания метки пункта списка, сокращённой по сравнению с текстом внутри <option>. Если атрибут label присутствует, то текст внутри <option> игнорируется и в списке выводится значение label.
Не поддерживается FireFox
<select name="question">
  <option label="Бах" value="1">Иоганн Себастьян Бах</option>
  <option label="Бетховен" value="2">Людвиг ван Бетховен</option>
  <option label="Шопен" value="3">Фредерик Шопен</option>
  <option label="Огинский" value="4">Михаил Клеофас Огинский</option>
</select>
selected Заранее устанавливает определённый пункт списка выделенным. Если к элементу <select> добавлен атрибут multiple, то можно выделять более одного пункта списка. 
<option selected>...</option>
value Значение пункта списка, которое будет отправлено на сервер или прочитано с помощью скриптов. На сервер отправляется пара «имя=значение», где имя задаётся атрибутом name элемента <select>, а значение — атрибутом value выделенных пунктов списка. Значение может как совпадать с текстом пункта, так быть и самостоятельным. Также атрибут value применяется для получения значений данных через скрипты. Значения - любая текстовая строка. 
<option value="текст">...</option>
