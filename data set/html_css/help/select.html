Элемент select (от англ. selection — выбор) позволяет создать элемент интерфейса в виде раскрывающегося списка, а также список с одним или множественным выбором.

Конечный вид зависит от использования атрибута size, который устанавливает высоту списка. Ширина списка определяется самым широким текстом, указанным в элементе option, а также может изменяться с помощью стилей. Каждый пункт создаётся с помощью элемента option, который должен быть вложен в контейнер select.

Если планируется отправлять данные списка на сервер, то требуется поместить select внутрь формы. Это также необходимо, когда к данным списка идёт обращение через скрипты.

<select>
   <option>Пункт 1</option>
   <option>Пункт 2</option>
</select>
Элемент select нельзя вкладывать внутрь ссылок a и кнопок button; внутри select должен быть хотя бы один элемент option.

Внутри контейнера select элементы option могут группироваться в броки при помощи элемента optgroup, при этом он сам не является элементом списка.

<select>
   <optgroup label="текст 1">
      <option>1-1</option>
      <option>1-2</option>
   </optgroup>
   <optgroup label="текст 2">
      <option>2-1</option>
      <option>2-2</option>
   </optgroup>
</select>

Атрибуты

autofocus disabled form multiple name required size
autofocus Устанавливает, что список получает фокус после загрузки страницы, при этом список становится доступным для выбора пунктов, например, с помощью клавиатуры. 
<select autofocus>...</select>
disabled Блокирует доступ и изменение элементов списка. Блокированный список не может получить фокус через курсор или клавиатуру, быть изменён, значение такого списка не передаётся на сервер. 
<select disabled>...</select>
form Связывает список с формой по её идентификатору. Такая связь необходима в случае, когда список располагается за пределами form. Значения - идентификатор формы (значение атрибута id элемента form). 
<select form="идентификатор">...</select>
Не поддерживается в IE и Safari
multiple Позволяет одновременно выбирать сразу несколько элементов списка. Конечный вид списка зависит от используемого атрибута size и браузера. 
<select multiple>...</select>
name Имя элемента для отправки на сервер или обращения через скрипты. В качестве имени используется набор символов, включая числа и буквы. JavaScript чувствителен к регистру, поэтому при обращении к элементу select по имени соблюдайте ту же форму написания, что и в атрибуте name. 
<select name="имя">...</select>
required Устанавливает список обязательным для выбора перед отправкой формы на сервер. Если пункт списка не выбран, браузер выведет сообщение, а форма не будет отправлена. Вид и содержание сообщения зависит от браузера и меняться пользователем не может. 
<select required>...</select>
size

Количество отображаемых строк списка.

Если значение атрибута size равно 1, то список превращается в раскрывающийся.

Когда атрибута multiple нет, то по умолчанию значение атрибута size равно 1. Если он присутствует, то размер списка по умолчанию - равен количеству элементов.

При добавлении атрибута multiple к элементу select при size="1" список отображается как «крутилка». 

<select size="число">...</select>

Пример 1 - size.

Вид как с multiple, но выбор только одного значения.
<select size="4">
  <option>First</option>
  <option>Second</option>
  <option>Third</option>
  <option>Fourth</option>
</select>

Пример 2 - disabled.

<select disabled name="hero">
   <option selected>Выберите героя</option>
   <option value="t1">Чебурашка</option>
   <option value="t2">Крокодил Гена</option>
   <option value="t3">Шапокляк</option>
   <option value="t4">Крыса Лариса</option>
</select>

Пример 3 - multiple.

Чтобы на сервер отправлялся массив данных, значение атрибута name следует писать с квадратными скобками — hero[], к примеру.

Не вводите людей в заблуждение. Если в PHP такая конвенция, то это ещё не значит, что атрибут "должен" выглядеть так. С другими серверными платформами всё может быть (и есть) по-другому.

<form action="handler.php">
  <select multiple name="hero[]">
    <option>Чебурашка</option>
    <option>Крокодил Гена</option>
    <option>Шапокляк</option>
    <option>Крыса Лариса</option>
  </select>
</form>

Пример 4 - required.

Какое растение в списке вечнозелёное?

<form action="handler.php">
   <p>Какое растение в списке вечнозелёное?</p>
   <p><select name="pregunta" required>
      <option></option>
      <option value="1">Ольха</option>
      <option value="2">Берёза</option>
      <option value="3">Бузина</option>
      <option value="4">Тис</option>
   </select></p>
   <p><input type="submit" value="Отправить"></p>
</form>

Пример 5 - name + script.

<script>
   function dataSelect(f) {
      n = f.hero.selectedIndex
      if(n) alert("Выбран герой: " + f.hero.options[n].value)
   }
</script>

<form action="handler.php">
  <p><select name="hero">
    <option>Выберите героя</option>
    <option value="Чебурашка">Чебурашка</option>
    <option value="Крокодил Гена">Крокодил Гена</option>
    <option value="Шапокляк">Шапокляк</option>
    <option value="Крыса Лариса">Крыса Лариса</option>
  </select>
  <input type="button" value="OK" onClick="dataSelect(this.form)"></p>
</form>
