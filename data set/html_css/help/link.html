<link>

    charset crossorigin href hreflang media rel rev referrerpolicy sizes type title
  • Пример: медиа-запросы для подключения каскадных таблиц стиля.

Элемент <link> (от англ. link — ссылка) обычно размещается внутри контейнера <head> и не создаёт ссылку, в отличие от элемента <a>. Устанавливает связь с внешним документом вроде файла со стилями или со шрифтами.

<head>
  <link href="адрес">
</head>
<head>
  <link rel="icon" href="favicon.ico">
  <link rel="shortcut icon" href="/img/logo.png" type="image/png"> HTML4
  <link rel="shortcut icon" href="favicon.ico" type="image/x-icon"> HTML4
  <link rel="stylesheet" type="text/css" href="main.css">
  <link rel='stylesheet' type='text/css'
        href='https://fonts.googleapis.com/css?family=Open+Sans:300italic,
              600&subset=latin,cyrillic'>
  <link rel="alternate" type="application/rss+xml" 
        title="Статьи с сайта webref.ru" href="https://webref.ru/rss.xml">
</head>

В данном примере подключается внешний файл со стилями с помощью атрибута rel="stylesheet" элемента <link>, указывается RSS-документ текущего сайта (rel="alternate" type="application/rss+xml") и устанавливается иконка сайта в адресной строке браузера (link rel="icon").

Файл, который будет подключён к HTML-странице, может располагаться в назначенной директории или в корне сайта.

Атрибуты.

charset Указывает кодировку документа на который ссылается элемент <link>.
Данный атрибут устарел, не используйте его. 
<link charset="кодировка">
<link href="okinawa.html" rel="parent" charset="euc-jp">

<meta charset="utf-8">
crossorigin Описывает, как элемент обрабатывает CORS-запросы, предназначен для использования со ссылками на внешние ресурсы.
href Путь к файлу, на который делается ссылка. 
<link href="адрес">
hreflang Задает язык связанного ресурса.
media Определяет устройство, для которого следует применять стилевое оформление. Это позволяет сделать разный стиль для отображения документа на экране монитора и при его печати. Допускается писать несколько значений через запятую. 
<link media="all | print | screen | speech">
allВсе устройства (значение по умолчанию).
printПечатающее устройство вроде принтера.
screenЭкран монитора.
speechРечевые синтезаторы, а также программы для воспроизведения текста вслух. Сюда же входят речевые браузеры.
В HTML5 в качестве значений могут быть указаны медиа-запросы. Пример.
rel Определяет отношения между текущим документом и файлом, на который делается ссылка. Это необходимо, чтобы браузер знал, как использовать подключаемый документ. 
<link rel="тип">
alternateАльтернативный тип, используется, к примеру, для указания ссылки на файл в формате XML для описания ленты новостей, анонсов статей.
authorУказывает ссылку на автора текущего документа или статьи.
helpУказывает ссылку на контекстно-зависимую справку.
iconАдрес картинки, которая символизирует текущий документ или сайт.
licenseСообщает, что основное содержание текущего документа распространяется по лицензии, описанной в указанном документе.
nextСообщает, что текущий документ является частью связанных между собой документов, а ссылка указывает на следующий документ.
prevСообщает, что текущий документ является частью связанных между собой документов, а ссылка указывает на предыдущий документ.
searchУказывает ссылку на ресурс, который применяется для поиска по документу или сайту.
stylesheetОпределяет, что подключаемый файл хранит таблицу стилей (CSS).
rev Описывает обратное отношение текущего документа к связываемому документу, как определено атрибутом href.
referrerpolicy Указывает URL источника запроса при переходе с одной страницы на другую.
sizes Указывает размер иконок для визуального отображения.
Вначале указывается ширина иконки в пикселах без указания единиц (например - 16), затем пишется латинская буква x в верхнем (X) или нижнем регистре (x), после чего идёт высота иконки. Если в файле хранится сразу несколько иконок, можно задавать их размеры через пробел. Ключевое слово any указывает, что иконка может масштабироваться в любой размер, к примеру, если она хранится в векторном формате SVG. 
<link rel="icon" sizes="ширинаXвысота">
<link rel="icon" 
      sizes="ширина1Xвысота1 ширина2Xвысота2 | any">
<link rel="icon" sizes="48x48" 
      href="android.ico" type="image/x-icon">
<link rel="icon" sizes="16x16" 
      href="android.png" type="image/png">
type MIME-тип данных подключаемого файла. Для подключаемых таблиц связанных стилей применяется тип text/css. Значение по умолчанию - text/css. 
<link type="MIME-тип">
<link rel="stylesheet" type="text/css" href="css/webref.css">
title Устанавливает заголовок ссылки, альтернативное имя таблицы стилей.

Медиа-запросы для подключения каскадных таблиц стиля.

<link rel="stylesheet" href="all.css" media="all" />
<link rel="stylesheet" href="small.css" media="(min-width: 20em)" />
<link rel="stylesheet" href="medium.css" media="(min-width: 64em)" />
<link rel="stylesheet" href="large.css" media="(min-width: 90em)" />
<link rel="stylesheet" href="extra-large.css" media="(min-width: 120em)" />
<link rel="stylesheet" href="print.css" media="print" />
