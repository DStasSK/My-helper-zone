HTML тег <head>

Тег <head> (от англ. head — голова) предназначен для хранения других элементов, цель которых — помочь браузеру в работе с данными. Также внутри контейнера <head> находятся метатеги, которые используются для хранения информации предназначенной для браузеров и поисковых систем. Например, механизмы поисковых систем обращаются к метатегам для получения описания сайта, ключевых слов и других данных.

Содержимое <head> не отображается напрямую на веб-странице, за исключением элемента <title>, он задаёт заголовок окна веб-страницы.

Внутри контейнера <head> допускается размещать следующие элементы: <base>, <basefont>, <bgsound>, <link>, <meta>, <script>, <style>, <title>.

Порядок расположения тега <head>

<html>
    <head>
        <!-- Между данными тегами размещается заголовок
         и дополнительные параметры страницы.
         -->
    </head>
    <body>
        <!-- В этом месте размещается содержание сайта,
         которое будет отображено на странице.
         -->
    </body>
</html>
