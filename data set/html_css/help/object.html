Элемент <object>

  • Атрибуты
    align archive classid code codebase codetype data height hspace type vspace width

Элемент <object> (от англ. object — объект) сообщает браузеру, как загружать и отображать объекты, которые исходно браузер не понимает. Как правило, такие объекты требуют подключения к браузеру специального модуля, который называется плагин, или запуска вспомогательной программы.

Дополнительно внутрь контейнера <object> можно поместить элемент <param>, который передаёт дополнительные параметры для отображения объекта.

<object></object>
Так же всесто <object> можно использовать <embed>

Атрибуты

align Определяет, как объект будет выравниваться на странице и способ его обтекания текстом. устарел 
<object align="значение">...</object>
absmiddleВыравнивание середины объекта по середине текущей строки.
baselineВыравнивание объекта по базовой линии текущей строки.
bottomВыравнивание нижней границы объекта по окружающему тексту.
leftВыравнивание объекта по левому краю окна, текст обтекает его справа.
middleВыравнивание середины объекта по базовой линии текущей строки.
rightВыравнивает объект по правому краю окна, текст обтекает слева.
texttopВерхняя граница объекта выравнивается по самому высокому текстовому элементу текущей строки.
topВерхняя граница объекта выравнивается по самому высокому элементу текущей строки.
archive

Устанавливает путь к файлам, необходимым для работы объекта. устарел

Позволяет задать произвольное число файлов необходимых для корректной работы объекта или для его отображения. Файлы перечисляются через пробел вместе с полным или локальным адресом.

<object archive="адрес">...</object>
classid

Адрес программы (приложения или плагина), которая работает с данным объектом, и будет запускать его. устарел

Атрибут classid указывает браузеру, какую программу, плагин или апплет нужно загрузить. Можно также использовать идентификатор зарегистрированного ActiveX приложения, предваряя его ключевым словом clsid.

<object classid="адрес">...</object>

<object codetype="application/java-vm"
        codebase="http://htmlbook.ru/applet.class"
        classid="java:program.start" height="100" width="100">
code

Имя объекта для его выполнения. устарел

Атрибут code используется для замены элемента <applet>, который загружает и выполняет Java-aпплет. В качестве значения code используется имя класса Java-апплета (файл с расширением class). Если файл располагается не в той же папке что и текущий документ, то следует указать путь к нему через атрибут codebase.

<object code="адрес">...</object>
<object code="applet/Сlock.class"></object>
codebase

Путь к папке с объектом, который указан атрибутом code или classid. устарел

Атрибут codebase должен содержать только имя папки, а не файла.

<object codebase="адрес">...</object>
codetype

Указывает на тип объекта, который задан атрибутом classid. устарел

Браузер может использовать эту информацию, чтобы подготовить необходимые ресурсы для воспроизведения файла. Если атрибут codetype отсутствует, то рассматривается значение type, хотя оно обычно и связывается с атрибутом data. В случае, когда не указаны оба атрибута codetype и type, браузер получает данные о типе загружаемого файла самостоятельно.

<object codetype="MIME-тип">...</object>

<object classid="clsid:02BF25D5-8C17-4B23-BC80-D3488ABDDC6B" 
        codetype="video/quicktime"></object>
data

Адрес файла для его отображения в окне браузера. обязательный

Определяет файл, который следует отобразить в окне браузера. Для популярных форматов данных достаточно указать путь к файлу и его тип (атрибут type) для загрузки и демонстрации результата.

Путь к файлу определяется относительно папки, заданной атрибутом codebase. Если codebase не указан, тогда путь следует задавать относительно текущего документа.

<object data="адрес">...</object>

<object data="video/duel.mp4" type="application/mpeg" 
        width="300" height="200"></object>
height

Устанавливает высоту объекта. В заданные размеры входит не только само изображение, например в случае воспроизведения видеофайла, но и панель управления им, включая кнопки проигрывания, паузы, остановки и т. д. По этой причине на размер отображаемого объекта влияет тип файла и применяемый плагин.

Если используется процентная запись, то размеры объекта вычисляются относительно родительского элемента — контейнера, где находится элемент <object>. В случае отсутствия родительского контейнера, в его качестве выступает окно браузера.

hspace Горизонтальный отступ от объекта до окружающего контента. устарел
Значения - любое целое положительное число в пикселях. 
<object hspace="число">...</object>
type MIME-тип объекта. обязательный
vspace Вертикальный отступ от объекта до окружающего контента. устарел
Значения - любое целое положительное число в пикселях. 
<object vspace="число">...</object>
width Ширина объекта. В заданные размеры входит не только само изображение, например в случае воспроизведения видеофайла, но и панель управления им, включая кнопки проигрывания, паузы, остановки и т. д. По этой причине на размер отображаемого объекта влияет тип файла и применяемый плагин.

Если используется процентная запись, то размеры объекта вычисляются относительно родительского элемента — контейнера, где находится элемент <object>. В случае отсутствия родительского контейнера, в его качестве выступает окно браузера.

Пример

<object type="application/x-shockwave-flash"
        data="swf/mouse.swf"
        width="400" height="300" style="margin:0 auto; display:block">
  <param name="quality" value="high">
  <param name="wmode" value="opaque">
</object>
