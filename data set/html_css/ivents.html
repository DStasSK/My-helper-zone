События Мыши и Клавиатуры

АтрибутЗначениеОписаниеСтатус
onclickкод скриптаКод скрипта будет выполнен после щелчка мыши.-
oncontextmenuкод скриптаКод скрипта будет выполнен при вызове контекстного меню.
ondblclickкод скриптаКод скрипта будет выполнен после двойного щелчка мыши.-
ondragкод скриптаКод скрипта будет выполнен при перетаскивании элемента.
ondragendкод скриптаКод скрипта будет выполнен, когда закончится перетаскивание элемента.
ondragenterкод скриптаКод скрипта будет выполнен, когда перетаскиваемый элемент будет заведен в область, которая принимает его.
ondragleaveкод скриптаКод скрипта будет выполнен, когда перетаскиваемый элемент покинет пределы области, которая принимает его.
ondragstartкод скриптаКод скрипта будет выполнен, когда перетаскивание элемента начнется.
ondropкод скриптаКод скрипта будет выполнен, когда элемент будет перетащен в область, которая принимает его.
onmousedownкод скриптаКод скрипта будет выполнен, когда будет нажата клавиша мыши.-
onmousemoveкод скриптаКод скрипта будет выполнятся при передвижении курсора мыши.-
onmouseoutкод скриптаКод скрипта будет выполнен, когда курсор мыши покинет пределы границ элемента.-
onmouseoverкод скриптаКод скрипта будет выполнен, когда курсор мыши будет заведен в пределы границ элемента.-
onmouseupкод скриптаКод скрипта будет выполнен, когда нажатая клавиша мыши будет отпущена.-
onmousewheelкод скриптаКод скрипта будет выполнен при прокрутке колесика мыши.
onscrollкод скриптаКод скрипта будет выполнен при прокрутке полосы прокрутки элемента.
onkeydownкод скриптаКод скрипта будет выполнен, когда клавиша на клавиатуре будет нажата.-
onkeypressкод скриптаКод скрипта будет выполнен, когда клавиша на клавиатуре будет нажата и отпущена.-
onkeyupкод скриптаКод скрипта будет выполнен, когда нажатая на клавиатуре клавиша будет отпущена.-

События Объекта Window

АтрибутЗначениеОписаниеСтатус
onafterprintкод скриптаКод скрипта будет выполнен после того, как документ будет распечатан.
onbeforeprintкод скриптаКод скрипта будет выполнен перед тем, как документ будет распечатан.
onbeforeonloadкод скриптаКод скрипта будет выполнен перед тем, как документ будет загружен.
onblurкод скриптаКод скрипта будет выполнен после того, как документ станет неактивным.-
onerrorкод скриптаКод скрипта будет выполнен если при загрузке документа произойдет ошибка.-
onfocusкод скриптаКод скрипта будет выполнен после того, как документ станет активным.-
onhaschangeкод скриптаКод скрипта будет выполнен если содержимое документа изменится.
onloadкод скриптаКод скрипта будет выполнен при загрузке документа.-
onmessageкод скриптаКод скрипта будет выполнен при получении документом сообщения.
onofflineкод скриптаКод скрипта будет выполнен, когда документ будет отключен от интернета.
ononlineкод скриптаКод скрипта будет выполнен, когда документ будет подключен к интернету.
onpagehideкод скриптаКод скрипта будет выполнен, когда окно с документом будет скрыто.
onpageshowкод скриптаКод скрипта будет выполнен, когда окно с документом будет отображено.
onpopstateкод скриптаКод скрипта будет выполнен, когда история (список посещенных в данном окне браузера страниц) документа будет изменена.
onredoкод скриптаКод скрипта будет выполнен, когда будет совершен переход на следующую страницу в истории документа.
onresizeкод скриптаКод скрипта будет выполнен, когда размеры окна документа будут изменены.-
onundoкод скриптаКод скрипта будет выполнен, когда будет совершен переход на предыдущую страницу в истории документа.
onunloadкод скриптаКод скрипта будет выполнен при закрытии окна браузера с документом.

События Форм

АтрибутЗначениеОписаниеСтатус
onblurкод скриптаКод скрипта будет выполнен, когда форма станет неактивной.-
onchangeкод скриптаКод скрипта будет выполнен при изменении содержимого формы.-
onfocusкод скриптаКод скрипта будет выполнен, когда форма станет активной.-
onformchangeкод скриптаКод скрипта будет выполнен, когда содержимое формы изменится.
onforminputкод скриптаКод скрипта будет выполнен при введении пользователем информации в форму.
oninputкод скриптаКод скрипта будет выполнен при введении пользователем информации в элемент.
onselectкод скриптаКод скрипта будет выполнен, когда элемент будет выбран.-
onsubmitкод скриптаКод скрипта будет выполнен при отправке формы.-
onresetкод скриптаСобытие onreset возникает при очистке формы, происходящей при нажатии на кнопку 
<input type="reset">

Медиа События

АтрибутЗначениеОписаниеСтатус
onabortкод скриптаКод скрипта будет выполнен если загрузка файла будет прервана.-
oncanplayкод скриптаКод скрипта будет выполнен если файл был загружен достаточно для начала воспроизведения.
oncanplaythroughкод скриптаКод скрипта будет выполнен если файл был загружен достаточно для воспроизведения до конца без остановок для подгрузки.
ondurationchangeкод скриптаКод скрипта будет выполнен при изменении длины файла.
onemptiedкод скриптаКод скрипта будет выполнен если файл станет недоступен.
onendedкод скриптаКод скрипта будет выполнен, когда файл будет воспроизведен до конца.
onerrorкод скриптаКод скрипта будет выполнен если при загрузке файла произойдет ошибка.
onloadeddataкод скриптаКод скрипта будет выполнен при загрузке данных.
onloadedmetadataкод скриптаКод скрипта будет выполнен при загрузке мета данных.
onloadstartкод скриптаКод скрипта будет выполнен, когда загрузка файла начнется.
onpauseкод скриптаКод скрипта будет выполнен если воспроизведение файла остановится либо с помощью кнопки пауза, либо программно через JavaScript.
onplayкод скриптаКод скрипта будет выполнен, когда медиа файл будет готов к воспроизведению.
onplayingкод скриптаКод скрипта будет выполнен, когда проигрывание медиа файла начнется.
onprogressкод скриптаКод скрипта будет выполнен при считывании мета данных файла.
onratechangeкод скриптаКод скрипта будет выполнен если скорость воспроизведения медиа файла изменится.
onreadystatechangeкод скриптаКод скрипта будет выполнен при изменении состояния медиа файла.
ontimeupdateкод скриптаКод скрипта будет выполнен при перемотки воспроизведения медиа файла.
onvolumechangeкод скриптаКод скрипта будет выполнен при изменении громкости медиа файла.
onwaitingкод скриптаКод скрипта будет выполнен, когда воспроизведение медиа файла будет остановлено.
