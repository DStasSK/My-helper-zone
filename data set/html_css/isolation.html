CSS свойство isolation

Свойство isolation определяет должен ли элемент создавать новый контекст наложения. Это свойство особенно полезно при использовании совместно с background-blend-mode.

div {
  isolation: auto | isolate;
}
  • auto - новый контекст наложения создаётся только в том случае, если это требуется для одного из свойств, применяемых к элементу.
  • isolate - новый контекст наложения должен быть создан, при этом эффект от значения свойства background-blend-mode будет игнорироваться, т.е. отменен для заданного блока.
