Горизонтальное меню представляет собой список разделов сайта, поэтому логичнее разметить внутри элемента ul, а затем применить CSS-стили к его элементам. Такое расположение меню является наиболее распространенным в силу очевидных преимуществ в его позиционировании на веб-странице.

По умолчанию все элементы списка li располагаются вертикально, занимая по ширине всю ширину элемента контейнера ul, который в свою очередь занимает всю ширину его блока-контейнера.

Горизонтальное меню, находящееся внутри тега ul, можно дополнительно помещать внутрь элемента nav и/или header. Благодаря этому html-разметке придается семантический смысл, а также появляется дополнительная возможность для форматирования блока меню.

<ul>
  <li><a href="">Пункт меню</a></li>
  <li><a href="">Пункт меню</a></li>
  <li><a href="">Пункт меню</a></li>
  ...
</ul>

Существует несколько способов разместить их горизонтально. Для начала нужно сбросить стили браузера по умолчанию для элементов навигации:

ul {
  list-style: none; /*убираем маркеры списка*/
  margin: 0;        /*убираем верхнее и нижнее поле, равное 1em*/
  padding-left: 0;   /*убираем левый отступ, равный 40px*/
}
a {text-decoration: none;} /*убираем подчеркивание текста ссылок*/

Способ 1.   li {display: inline;}

Делаем элементы списка строчными. В результате они располагаются по горизонтали, с правой стороны между ними добавляется промежуток, равный 0.4em (вычисляется относительно размера шрифта). Чтобы убрать его, можно добавить для li отрицательное правое поле li {margin-right: -0.4em}. Дальше стилизуем ссылки по своему желанию.

  • Home
  • Work
  • About
  • Blog
  • Contact
HTML 
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Work</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Blog</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

CSS 
nav{display:block;
    width:100%;
    margin:0 auto 30px;
    text-align:right;}
ul {padding:1em 0;
    background:#ECDAD6;
    list-style:none;
    margin:0 auto;}
 a {padding:1em;
    background:rgba(177, 152, 145, .3);
    border-right:1px solid #b19891;
    color:#695753; text-align: center;
    text-decoration:none;
    transition: .5s linear;
    display:inline-block;
    width:calc(20% - 2em - 5px)}
a:hover {background:#b19891}
li {display:inline}

Способ 2.   li {float: left;}

Делаем элементы списка плавающими. В результате они располагаются по горизонтали. Высота блока-контейнера ul становится равной нулю. Чтобы решить эту проблему, добавляем для ul {overflow: hidden}, расширяя его и позволяя ему таким образом содержать плавающие элементы. Для ссылок добавляем a {display: block} и стилизуем их по своему желанию.

  • Home
  • Work
  • About
  • Blog
  • Contact
HTML 
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Work</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Blog</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

CSS 
nav{display:block; width:100%; margin:0 auto 30px; text-align:right}
ul {background:#D4E7EE; margin:0 auto; padding:0;
    overflow:hidden;
    list-style:none;}
li {float:left;
    width:20%;
    margin:0; text-align:center;}
 a {display:block; padding:1em 0; border-right:1px solid #ADC0CE;
    background:rgba(173, 192, 206, .3); text-decoration:none;
    transition:.5s linear; color:#29838C}
a:hover {background: #ADC0CE}

Способ 3.   li {display: inline-block;}

Делаем элементы списка строчно-блочными. Они располагаются по горизонтали, с правой стороны образуется промежуток, как и в первом случае. Для ссылок добавляем a {display: block} и стилизуем их по своему желанию.

  • Home
  • Work
  • About
  • Blog
  • Contact
HTML 
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Work</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Blog</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

CSS 
nav{display:block; width:100%; margin:0 auto 30px; text-align:right}
ul {list-style:none; margin:0 auto; padding:0; background:#E4EFD1}
li {display:inline-block;
    width:calc(20% - 3.8px);
    text-align:center;}
 a {text-decoration:none; transition:.5s linear; display:block;
    color:#74924C; border-right:1px solid #BAD78B;
    background:rgba(186, 215, 139, .3);}
a:hover {background:#BAD78B}

Способ 4.   ul {display: flex;}

Делаем список ul гибким контейнером с помощью flexbox. В результате элементы списка располагаются горизонтально. Добавляем для ссылок a {display: block} и стилизуем их по своему желанию.

  • Home
  • Work
  • About
  • Blog
  • Contact
HTML 
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">Work</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Blog</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

CSS 
nav{display:block; width:100%; margin:0 auto; text-align:right;}
ul {display:-webkit-flex; display:flex;
    list-style:none; margin:0 auto; background:#FFD7A7; padding-left:0;}
li {width:15%; line-height:60px;  text-align:center;}
 a {text-decoration:none; transition:.2s linear; padding:0; display:block;
    color:#9B4222; border-right:1px solid #FFAB51;
    background:rgba(255, 171, 81, .3)}
a:hover {font-size:24px; background:#FFAB51}

При задании цвета фона, элементы :before и :after с отрицательным значением свойства z-index могут "пропадать" - уходить за фон.
HTML 
<nav>
  <ul>
    <li><a href="" class="current">Work</a></li>
    <li><a href="">About</a></li>
    <li><a href="">Blog</a></li>
    <li><a href="">Contact</a></li>
  </ul>
</nav>

CSS 
@import url('https://fonts.googleapis.com/css?family=PT+Sans+Caption');
nav {margin:0 60px; position:relative; background:#5A394E;
    box-shadow:
       inset 1px 0 0 rgba(255,255,255,.1),
       inset -1px 0 0 rgba(255,255,255,.1),
       inset 150px 0 150px -150px rgba(255,255,255,.12),
       inset -150px 0 150px -150px rgba(255,255,255,.12);
}
nav:before, nav:after {content:""; width:100%; height:3px;
    position:absolute; z-index:2; left:0;}
nav:before {top:0; border-bottom:1px dashed rgba(255,255,255,.2);}
nav:after {bottom:0; border-top: 1px dashed rgba(255,255,255,.2);}

ul {list-style:none; padding:0; margin:0; text-align:center;}
ul: before, ul:after {content:""; width:50px; height: 0;
    position:absolute; top:8px; z-index:-1; transform:rotate(360deg);
    border-top:18px solid #5A394E; border-bottom:18px solid #5A394E;}
ul:before {left:-30px; border-left:12px solid #fff;}
ul:after {right:-30px; border-right:12px solid #fff;}

li {display:inline-block; margin-right:-4px;}
a {text-decoration:none; color:white;
    display:inline-block; transition:.3s linear;}
a.current, a:hover {background: rgba(0,0,0,.2);}

@media (max-width: 680px) {
  nav{margin: 0}
  li {display:block; margin-right:0;}
  ul:before, ul:after {content: none}
  a  {display: block}
}
