Как скрыть элемент дума на странице сайта

Структура страницы сайта как правило определяется макетом, по которому ее "собирают". При этом, хоть страница и будет выыглядеть понятно, но для плоховидящих людей при использовании скринридера - будет просто перечисление текста страницы без каких либо разделений и структуры, т.е. абсолютно непонятная страница.

Чтобы сделать страницу более понятной для скринридера необходимо добавить различные симантические теги или заголовки, если их нет. Если структура страницы подразумевает заголовок, но по макету его нет, можно добавить его в разметку и скрыть с помощью класса:

.visually-hidden {
  position: absolute;
  width: 1px;
  height: 1px;
  margin: -1px;
  border: 0;
  padding: 0;

  white-space: nowrap;

  clip-path: inset(100%);
  clip: rect(0 0 0 0);
  overflow: hidden;
}

Обычные посетители не заметят разницы, а пользователи скринридеров получат возможность быстро попадать в любой раздел страницы. Данный клас можно применять не только для скрытия заголовков, но и для скрытия инпутов, при этом для инпутов сохранится доступность для скринридеров и навигации с клавиатуры.

При скрытии элемента с помощью display: none элементы убираются со страницы полностью — и визуально, и для скринридеров. Пользователь скринридера не сможет попасть на них никаким образом.
