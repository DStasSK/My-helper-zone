Псевдоэлемент ::placeholder

Псевдоэлемент ::placeholder задаёт стилевое оформление подсказывающего текста, созданного атрибутом placeholder у элемента input или textarea. К этому псевдоэлементу могут применяться только стилевые свойства, связанные со свойствами шрифта, полями, отступами, границами, цветом и фоном.

<style>
  textarea::-moz-placeholder          {color:#009}
  textarea::-webkit-input-placeholder {color:#009}
  textarea:-ms-input-placeholder      {color:#009} /* не ошибка */
  textarea::-ms-input-placeholder     {color:#009} /* их два */
  textarea::placeholder               {color:#009}
  textarea {color: red}
</style>

<textarea placeholder="введите красный текст"></textarea>
Т.к. в разных браузерах цвет шрифта placeholder может быть прозрачной, то лучше дополнительно указывать параметр opacity 
  ::-webkit-input-placeholder {color:#f00; opacity:1;}
  ::-moz-placeholder          {color:#f00; opacity:1;}
  :-ms-input-placeholder      {color:#f00; opacity:1;}
Placeholder не является заменой элемента label. Без метки, которая программно связывается с формой с помощью for и id атрибутов, такие программы, как скринридеры не смогут анализировать элементы label.
  • Internet Explorer использует псевдокласс :-ms-input-placeholder.
  • Microsoft Edge использует псевдоэлемент ::-ms-input-placeholder.
  • Старые версии браузеров используют псевдоэлемент ::-webkit-input-placeholder.
  • Firefox до версии 51 использует псевдоэлемент ::-moz-placeholder.
IE8 не поддерживают атрибут «placeholder» — он просто не будет отображаться. Для отображени подсказки в браузерах, которые не поддерживают атрибут, необходимо использовать скрипты.
