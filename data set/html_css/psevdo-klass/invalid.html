Псевдокласс :invalid

Псевдокласс :invalid находит любые input или form элементы, контент которых не проходит валидацию, в соответствии с типом поля. Он позволяет вам легко менять внешний вид полей, что позволяет пользователю видеть и исправлять ошибки.

Применяется к полям формы, содержимое которых не соответствует указанному типу. Например, для type="number" должно вводиться число, а не буквы, для type="email" корректный адрес электронной почты.

<style>
  .form_5 input:invalid {background: #fdd} /* Красный цвет фона */
  .form_5 input:valid {background: #dfd}   /* Зелёный цвет фона */
  .form_50 {background: #ccc}
  #form_51 {background: #ccc}
</style>

<form class="form_5">
  <p>Адрес сайта</p>
  <p><input type="url" required></p>
  <p>Адрес электронной почты</p>
  <p><input type="email" required></p>
  <p><input type="submit" value="Отправить" class="form_50"></p>
  <p><input type="submit" value="Отправить" id="form_51"></p>
</form>
Если любая из радиокнопок в группе (т.е., с одинаковым атрибутом name) имеет атрибут required, псевдокласс :invalid применяется ко всем из них, если ни одна из кнопок группы не выбрана.
