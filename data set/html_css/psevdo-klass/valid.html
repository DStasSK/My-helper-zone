Псевдокласс :valid

Псевдокласс :valid находит любые input или form элементы, контент которых валиден, в соответствии с типом поля. Применяется к полям формы, содержимое которых проходит проверку в браузере на соответствие указанному типу. Например, для type="number" вводится число, а не буквы, для type="email" корректный адрес электронной почты.

form:valid { background-color: green }
