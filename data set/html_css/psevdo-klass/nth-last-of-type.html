Псевдокласс :nth-last-of-type()

Псевдоклас :nth-last-of-type() используется для добавления стиля к элементам указанного типа на основе нумерации в дереве элементов. В отличие от псевдокласса :nth-of-type отсчёт ведётся не от первого элемента, а от последнего.

  1. Lorem ipsum dolor sit.
  2. Lorem ipsum dolor sit.
  3. Lorem ipsum dolor sit.
li:nth-last-of-type(1) { color:red }
