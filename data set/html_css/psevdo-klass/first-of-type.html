Псевдокласс :first-of-type

Псевдокласс :first-of-type находит первого потомка своего типа среди детей родителя. Псевдокласс :first-of-type без указания селектора выберет первые элементы каждого типа. При добавлении селектора к :first-of-type сперва выбираются указанные элементы, затем из них берётся первый элемент.

.table tr:first-of-type { color: red }   /* первая строка в таблице */
.table td:first-of-type { color: red }   /* первая ячейка (внутри каждой строки) */
Вместо :first-of-type допустимо использовать :nth-of-child(1).
