Псевдокласс :dir

Псевдокласс :dir применяет стилевые правила к элементам на основе направления текста.
.elem:dir(ltr | rtl) { ... }

Для указания направления текста применяется атрибут dir со значением ltr или rtl. Поэтому в некоторых случаях селекторы :dir(rtl) и [dir="rtl"] будут давать одинаковый результат. Однако использование псевдокласса более универсально, поскольку оно основывается на направлении текста, которое определяет браузер.

<div dir="rtl">
  <p>…</p>
  <p>…</p>
</div>

p[dir="rtl"] { color:#333 }  /* не сработает т.к. у элемента <p> не задан атрибут dir. */
p:dir(rtl)   { color:#333 }
16.4 49 115 16.4
