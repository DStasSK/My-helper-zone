Псевдокласс :autofill

Псевдокласс :autofill позволяет стилизовать элементы input или textarea, когда браузер автоматически заполняет текстовое поле сохранёнными данными. Действие псевдокласса прекращается при редактировании поля формы. Возможность автозаполнение полей форм управляется через настройки браузера - например: «Автозаполнение и пароли», «Помнить историю поиска и данных форм».
