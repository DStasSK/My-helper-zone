Псевдокласс :first-child

Псевдокласс :first-child устанавливает стилевое оформление первого элемента-потомка своего родителя.
:first-child { color: red }    /* каждый первый элемент где бы он ни был */
p:first-child { color: red }   /* каждый первый <p> где бы он ни был */
p :first-child { color: red }  /* каждый первый элемент внутри всех <p> */

При добавлении селектора к :first-child сперва берётся первый элемент, затем смотрится какого он типа. Если элемент совпадает с указанным селектором, то он будет выбран. Таким образом, выбирается элемент, если он удовлетворяет двум условиям одновременно: это первый элемент и это элемент указанного типа.

Для выбора первого элемента определённого типа используют псевдокласс :first-of-type.
Вместо :first-child допустимо использовать :nth-of-child(1).
