Псевдокласс :read-only

Псевдокласс :read-only применяется к полям формы, у которых задан атрибут readonly. Такое поле не может быть модифицировано, при этом оно получает фокус, а данные в таком поле можно выделить и скопировать.

<style>
  input:-moz-read-only {  }
  input:read-only {  }
</style>
Псевдокласс :readonly применяется не только к элементам <input> или <textarea>, он применяется к любым элементам, которые пользователь не может изменить - такими как p, div и пр.
Firefox поддерживает псевдокласс :-moz-read-only.

<style>
  :read-write {border:2px solid red}
  :read-only {background: #741}
  input:read-only {background:#85fef4; opacity:0.8; margin:3px}
</style>
