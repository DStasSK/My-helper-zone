Псевдокласс :defined

Псевдокласс :defined применяется к элементу, когда он определён.
.elem:defined {   }
К таким элементам относятся следующие:
  • стандартные HTML-элементы, такие как p, h1, header и др.;
  • зарегистрированные пользовательские элементы.
Для обратного выбора — элементов, которые не были определены, можно использовать селектор :not(:defined).
IE не поддерживает данный элемент.
