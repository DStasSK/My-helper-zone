Псевдокласс :placeholder-shown

Псевдокласс :placeholder-shown oпределяет стиль элемента <input> или <textarea>, который в данный момент отображает текст подсказки, заданной атрибутом placeholder (при этом текст в поле еще не введен).

Так же можно прятать текст подсказки при фокусе поля ввода - задавая ей цвет как transparent;

.text {color:red; resize:none; background:#fcfcfc; width:500px}
.text::placeholder {color:#009}
.text:placeholder-shown {border-radius:10px}
.text:focus:placeholder-shown {-webkit-text-fill-color:transparent}

<textarea class="text" placeholder="введите красный текст"></textarea>
<textarea class="text"></textarea>
