Псевдоэлемент ::selection

Псевдоэлемент ::selection Позваляет задать цвет выделения текста. Выделение обычно применяется для копирования в буфер фрагмента текста, чтобы после этого вставить выделенный фрагмент в другое место. Например, в текстовый редактор для дальнейшей правки текста. Сам цвет и фон выделенного фрагмента определяется настройками операционной системы, но с помощью CSS эти параметры можно задать самостоятельно.

В правилах стилей допускается использовать следующие свойства: color, background, background-color, cursor, outline и text-shadow.

Firefox до версии 62 использует нестандартный псевдоэлемент ::-moz-selection.
.elem ::selection {
   color:#007;
   background-color:#eef;
}
