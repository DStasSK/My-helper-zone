Псевдокласс :only-child

Псевдокласс :only-child выбирает дочерний элемент, только если он является единственным у своего родителя. Если у родителя два дочерних элемента, то они не выбираются. Например: p:only-child находит элемент только при соблюдении двух условий: это единственный дочерний элемент и это элемент p.

Псевдокласс :only-child без указания селектора выберет все единственные элементы и установит для них соответствующие свойства, т.е. если элеметнов на странице два или более, то они не будут выбраны.

В Safari до версии 3.0 псевдокласс :only-child воспринимается как :first-child.

1. Lorem ipsum dolor sit.

2. Lorem ipsum dolor sit.
<div>
  <p>1. Lorem ipsum dolor sit.</p>
</div>
<div>
  <p>1. Lorem ipsum dolor sit.</p>
  <div>2. Lorem ipsum dolor sit.</div>
</div>

p:only-child { color:red }
