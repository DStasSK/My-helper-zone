Псевдокласс :link

Псевдокласс :link определяет необходимое стилевое оформление ссылкам, которые ещё не посещались. Благодаря псевдоклассу можно изменить стиль ссылки и сайта в целом установив для неё необходимый формат.

<a href="#">которые ещё не посещались</a>

a{color: #009}
a:link{color: #963}
a:visited{color: #a0d}
a:active{color: #f00}

a:link:after{content:' (перехода не было)'}
a:visited:after{content:'(переход был)'}     /* этот параметр работать не будет */
a:active:after {content:' (ссылка нажата)'}

Псевдоклассы для оформления стилей могут переопределять друг друга, чтобы избежать этого необходимо их указывать в определенном порядке, а именно:

a:link, a:visited, a:hover, a:active, a:focus {}
