Псевдокласс :focus-visible

Псевдокласс :focus-visible используется для применения стилей к элементу, который получил фокус, но когда переход к элементу происходит с помощью клавиатуры.

.elem:focus-visible {...}

<p>
  <input type="text" placeholder="нажми сюда - потом tabs">
  <button class="btn1">Кнопка</button>
  <button class="btn2">Кнопка</button>
</p>

button {outline: none}
button:focus {outline: none}
.btn1:focus {outline: 2px solid orange}
.btn2:focus-visible {outline: 2px solid green}
