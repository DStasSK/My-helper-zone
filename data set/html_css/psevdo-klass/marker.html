Псевдоэлемент ::marker

Данный псевдоэлемент позволяет устанавливить вид маркера через свойство content, а также управлять стилем маркера (менять его цвет или параметры шрифта). ::marker работает для элементов li, summary, а также для элементов, которые имеют свойство display:list-item.

С помощью псевдоэлемента ::marker допустимо устанавливать любые свойства шрифта, а так же свойства white-space, color, content, анимацию и переходы.

  • 1
  • 2
  • 3
<ul class="list">
   <li>1</li>
   <li>2</li>
   <li>3</li>
</ul>

<style>
   .list ::marker {color: red}
   .list li:nth-child(2)::marker {content:'+'}
   .list li:nth-child(3)::marker {content:'+  '}
</style>
Браузер Safari, а также мобильный Firefox, не поддерживают свойства анимации и переходов.
