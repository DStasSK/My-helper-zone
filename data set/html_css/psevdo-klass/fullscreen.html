Псевдокласс :full-screen

Применяется к элементам, когда браузер находится в полноэкранном режиме. При этом скрывается заголовок окна, адресная строка, вкладки, меню и другие стандартные элементы и браузер отображается на весь экран. Для перехода из стандартного режима в полноэкранный и обратно обычно применяется клавиша F11.

Чтобы псевдокласс работал, предварительно для отдельных элементов следует разрешить полноэкранный режим с помощью скриптов.
  • Internet Explorer поддерживает псевдокласс :-ms-fullscreen
  • Chrome, Opera, Safari поддерживают псевдокласс :-webkit-full-screen
  • Firefox поддерживает псевдокласс :-moz-full-screen
Псевдоклассы для каждого браузера следует писать отдельно, а не перечислять их через запятую.
<style>
  .elem:-ms-fullscreen { background: #6088a5; }
  .elem:-webkit-full-screen { background: #6088a5; }
  .elem:-moz-full-screen { background: #6088a5; }
</style>

Пример

Для перевода выбранного элемента в полноэкранный режим необходимо использовать стандартный метод requestFullscreen() на элементе. Для выхода из полноэкранного режима необходимо использовать метод document.exitFullscreen() (данный метод всегда вызывается на объекте document).

При переходе в полноэкранный режим и при выходе из него браузер генерирует событие fullscreenchange. Если по какой-то причине переход в полноэкранный режим недоступен, то генерируется ошибка fullscreenerror - это следует учитывать, чтобы не сломать страницу.

При переходе в полноэкранный режим, элемент, на котором был вызван метод requestFullscreen() записывается как значение свойства document.fullscreenElement, само свойство по умолчанию имеет значение null.

При выходе из полноэкранного режима, прокрутка страницы документа может меняться.
<script>
  function fullScreen() {
    let el = document.querySelector('#msg');

    if (el.webkitRequestFullscreen) el.webkitRequestFullscreen();// webkit, Safari
    else if (el.mozRequestFullScreen) el.mozRequestFullScreen(); // Firefox
    else if (el.msRequestFullscreen) el.msRequestFullscreen();   // IE, Edge
    else if (el.requestFullscreen) el.requestFullscreen();       // Стандарт
  }

  function fullScreenCancel() {
    if (document.fullscreenElement) document.exitFullscreen();
  }
</script>

<style>
  #msg {padding:10px; background:#bc7062; color:#fff;}
  #msg:-ms-fullscreen { background: #6088a5; }
  #msg:-webkit-full-screen { background: #6088a5; }
  #msg:-moz-full-screen { background: #6088a5; }
</style>

<div id="msg">
   Элемент страницы отображается в полноэкранном режиме.
</div>
<button onclick="fullScreen()">Посмотреть на полном экране!</button>
<button onclick="fullScreenCancel()">Выход из полного экрана!</button>
