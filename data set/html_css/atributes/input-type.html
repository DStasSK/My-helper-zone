Атрибут type элемента input.

Атрибут type cообщает браузеру, к какому типу относится элемент формы. Если этот атрибут не указан, по умолчанию используется text.
<input type="значение">
button checkbox color date datetime datetime-local email file hidden image month number password radio range reset search submit tel text time url week
button

Кнопка без предопределённого поведения.


<input type="button" value="текст">
Атрибуты input которые дополняют тип button
  • value - надпись на кнопке.
  • name - имя кнопки, чтобы обработчик формы мог её идентифицировать.
  • form - идентификатор формы для связывания кнопки с form.
  • disabled - блокирует кнопку.
  • autofocus - кнопка получает фокус после загрузки документа.
submit Кнопка для отправки данных формы на сервер. 
 <input type="submit">

 <input type="submit" value="Искать">
image

Поле с изображением, при нажатии на рисунок данные формы отправляются на сервер.

Дополнительные атрибуты:
  • src - определяет путь к изображению
  • alt - задает альтернативный текст
  • height - высота кнопки
  • width - ширина кнопки

<input type="image" src="img/myButtonImg.jpg">
checkbox

Представляет элемент input в виде флажка. Позволяет выбрать более одного варианта из предложенных.

Атрибут value определяет значение, которое будет отдано этим элементом.
Атрибут checked делает флажок выбранным.
Атрибут indeterminate указывает, что флажок находится в неопределённом состоянии. 

<input type="checkbox" checked>
<input type="checkbox">
<input type="checkbox">
color

Элемент управления цветом, представляет собой элемент управления цветовым диапазоном для установки значения элемента в строку, представляющую сериализацию цвета CSS.

Атрибут value, если он указан, должен иметь значение, которое является цветом CSS (#aa12ef). 

<input type="color" value="#aa12ef">

date Поле для выбора календарной даты (год, месяц и день, без времени). 

<input type="date">
datetime Указание даты и времени (час, минута, секунда и доля секунды) в соответствии с часовым поясом UTC. 

<input type="datetime">
datetime-local
 Указание местной даты и времени без часового пояса. 

<input type="datetime-local">
email Поле для редактирования адреса электронной почты. Перед отправкой проверяется, что входное значение содержит либо пустую строку, либо один действительный адрес электронной почты. 

<input type="email">
file

Элемент управления, который позволяет пользователю выбрать файл. Aтрибут accept определяет типы файлов, которые могут быть выбраны.


<input type="file"
       style="letter-spacing:-.565em">


без letter-spacing
hidden Скрытое поле. Оно никак не отображается на веб-странице, но чьё значение будет отправлено на сервер. 

<input type="hidden">
month Элемент управления для ввода месяца и года без часового пояса. 

<input type="month">
number

Элемент управления ввода числа (тип float).

атрибут min - определяет минимально допустимое значение
атрибут max - определяет максимально допустимое значение
атрибут step - задает шаг приращения значения 

<input type="number" min="0" step="0.01" max="10">
password Однострочное текстовое поле, чьё значение скрыто символом "звёздочка". Аатрибуты minlength и maxlength позволяют задать минимальную и максимальную длину значения, которое может быть введено. 

<input type="password">
radio

Переключатели. Используются, когда следует выбрать один вариант из нескольких предложенных.

Переключатели объединяются в группы при помощи задания атрибута name="имя_группы" с одинаковым значением. 

<input type="radio" name="radio">
range Ползунок для выбора чисел в указанном диапазоне. 

<input type="range" min="50" max="100" value="50">
значения атрибутов по умолчанию:
  • min: 0
  • max: 100
  • value: min + (max-min)/2, or min if max < min
  • step: 1
reset Кнопка для возвращения данных формы в первоначальное значение (в состояние по умолчанию). 

<input type="reset">
search Поле для поиска. Однострочное текстовое поле для ввода строк поиска; разрывы строк автоматически удаляются из входного значения. 
 
<input type="search"> 
<input type="submit" value="Искать">
tel Элемент управления для ввода номера телефона; разрывы строк автоматически удаляются из входного значения, но никакой другой синтаксис не применяется. Можно использовать атрибуты pattern и maxlength, чтобы ограничить вводимое значение. 

<input type="tel">
text Однострочное текстовое поле. Предназначено для ввода символов с помощью клавиатуры. Переносы строк автоматически удаляются из входного значения. 

<input type="text">
time Элемент управления для ввода значения времени без часового пояса. 

<input type="time">
url Поле для редактирования URI. Введённое значение должно содержать либо пустую строку, либо допустимый абсолютный URL. В противном случае значение не будет принято. Переводы строк, лидирующие и завершающие пробельные символы будут автоматически удалены из введённого значения. Можно использовать атрибуты pattern и maxlength, чтобы ограничить вводимое значение. 

<input type="url">
week Элемент управления для ввода даты, содержащей число неделя-год и номер недели без часового пояса. 

<input type="week">
