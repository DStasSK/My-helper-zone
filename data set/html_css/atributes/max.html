Атрибут max.

Данный атрибут доступен для элементов progress, meter и элементам input у которых атрибут type имеет значение date, month, week, time, datetime-local, number, range.

Атрибут max определяет максимальное значение, которое является приемлемым и действительным для входных данных, содержащих атрибут. Если значение элемента больше этого, элемент не проходит проверку. Это значение должно быть больше или равно значению атрибута min.

Если атрибут max присутствует, но его значение не указано или является недействительным, атрибут не применяется. Если атрибут max указан и вводимое значение больше максимально допустимого атрибутом max, проверка ограничений предотвратит отправку формы.

Если введенные пользователем данные превышают максимально допустимое значение, определенное атрибутом max, то элемент будет сопоставлен псевдоклассам :out-of-range и :invalid.

Для элемента progress атрибут max описывает объем работы, требуемый для задачи, указанной элементом progress. Если присутствует, должен иметь значение больше нуля и быть допустимым числом с плавающей точкой.

Идёт загрузка…
60% 
<p>Идёт загрузка…</p>
<progress value="60" max="100">60%</progress>

Для элемента meter атрибут max определяет верхнюю числовую границу измеряемого диапазона. Оно должно быть больше минимального значения (атрибут min), если оно указано. В обоих случаях, если оно опущено, значение по умолчанию равно 1.

36,6 градусов

<meter value="36.6" min="35" max="42">36,6 градусов</meter>

Для элемента input атрибут max устанавливает верхнее значение для ввода числа или даты в поле формы. В качестве значения использует:

  • Целое положительное или отрицательное число (для type="number", type="range").
  • Дата в формате ГГГГ-ММ-ДД (например: 2012-12-22) для type="date".
 <input type="date" max="2019-12-25" step="1"> 
 <input type="month" max="2019-12" step="12"> 
 <input type="week" max="2019-W23" step=""> 
 <input type="time" max="17:00" step="900"> 
 <input type="datetime-local" max="2019-12-25T23:59"> 
 <input type="number" min="0" step="5" max="100"> 
 <input type="range" min="60" step="5" max="100">
