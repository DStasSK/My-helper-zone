Атрибут dir

Атрибут dir задаёт направление и отображение текста — слева направо или справа налево. Если свойство не задано, то его значение наследуется от родительского элемента.

<div dir="rtl | ltr">
  текст
</div>
  • rtl - направление текста идет слева направо.
  • ltr - направление текста идет справа налево.
  • auto - браузер определит направление текста на основе содержимого; рекомендуется только в том случае, если направление текста неизвестно - например, для данных, поступающих из пользовательского ввода или из внешних данных.
Этот атрибут может быть переопределен CSS свойствами direction и unicode-bidi.
Этот атрибут не наследуется элементом bdi. Если он не установлен, его значение — auto.
