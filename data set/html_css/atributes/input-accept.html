Атрибут accept элемента input

Устанавливает фильтр на типы файлов, которые вы можете отправить через поле загрузки файлов. Если файл не подходит под установленный фильтр, он не показывается в окне выбора файлов. Применяется к полю для отправки файла. В качестве значения принимает список из одного или нескольких типов файлов, разделенных запятыми, или уникальных спецификаторов типов файлов, описывающих, какие типы файлов разрешены.

<input type="file" accept="MIME-тип | значение">
Значения:
  • расширение файла с точкой впереди (например: .jpg).
  • имя MIME-типа (например: image/jpeg).
  • несколько значений перечисляются через запятую (image/jpeg, image/png).
  • audio/* для выбора всех звуковых файлов
  • video/* для видеофайлов
  • image/* для всех графических файлов

Укажите картинку в формате JPEG, PNG или GIF

<form action="handler.php">
  <strong>Укажите картинку в формате JPEG, PNG или GIF</strong>
  <input type="file" name="img" accept=".png, .jpg, .jpeg, .gif">
  <input type="submit" value="Отправить">
</form>
Существует несколько способов идентификации файлов Microsoft Word, поэтому сайт, принимающий файлы Word, может использовать input следующим образом: 
<form action="handler.php">
  <strong>Выберите файл word документа:</strong>
  <input type="file" id="docpicker"
         accept=".doc,.docx,application/msword,
         application/vnd.openxmlformats-officedocument.wordprocessingml.document" />
  <input type="submit" value="Отправить">
</form>
