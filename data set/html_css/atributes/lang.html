Атрибут lang

Атрибут lang позволяет установить язык элемента, то есть тот язык, на котором написаны нередактируемые элементы, или язык, на котором пользователем должны быть написаны редактируемые элементы. Атрибут содержит код языка.

<div lang="код языка">
  ...
</div>

Браузер использует значение атрибута для правильного отображения некоторых национальных символов. Текст документа может быть набран как на одном языке, так и содержать вставки на других языках, которые могут различаться по своим правилам оформления текста. Например, для русского, немецкого и английского языка характерны разные кавычки, в которые берется цитата. Чтобы указать язык, на котором написан текст внутри текущего элемента и применяется атрибут lang

Цитата на французском языке: Ce que femme veut....

Цитата на немецком: Der Mensch, versuche....

Цитата на английском: То be or not to be.

<p>Цитата на французском языке: <q lang="fr">Ce que femme veut...</q>.</p>
<p>Цитата на немецком: <q lang="de">Der Mensch, versuche...</q>.</p>
<p>Цитата на английском: <q lang="en">То be or not to be</q>.</p>


q:lang(de) {quotes:"\201E" "\201C"} /* Вид кавычек для немецкого языка */
q:lang(en) {quotes:"\201C" "\201D"} /* Вид кавычек для английского языка */
q:lang(fr) {quotes:"\0AB" "\0BB"}   /* Вид кавычек для французского языка*/
