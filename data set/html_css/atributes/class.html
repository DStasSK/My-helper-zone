Атрибут class

Атрибут class содержит разделённый пробелом список регистро-зависимых обозначений классов элемента. Имена классов могут содержать в себе латинские буквы (A–Z, a–z), цифры (0–9), символ дефиса ( - ) и подчёркивания ( _ ) и не должны начинаться с цифры.

<div class="my_div_class">
  <span class="my_span_class">text</span>
</div>

Рекомендуется выбирать значения для атрибута class, которые являются допустимыми идентификаторами CSS, не требующими экранирования.
