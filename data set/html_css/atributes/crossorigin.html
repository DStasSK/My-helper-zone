Атрибут crossorigin.

Данный атрибут доступен для элементов audio, img, link, script и video.

Aтрибут crossorigin позволяет настроить CORS запросы для данных получаемых элементом. Oбеспечивает поддержку CORS, определяя, как элемент обрабатывает запросы кросс-источника, тем самым позволяя настраивать запросы CORS для извлеченных данных элемента. Позволяет загружать ресурсы c другого домена с помощью CORS-запросов.

<img crossorigin="anonymous | use-credentials" />
  • anonymous - запрос использует заголовки CORS, а флаг учетных данных установлен на 'same-origin'. Обмен учетными данными пользователя (через файлы cookie, клиентские сертификаты TLS или HTTP-аутентификацию) не производится, если только пункт назначения не является тем же источником.
  • use-credentials - запрос выполняется с передачей учетных данных. Запрос использует заголовки CORS, флаг учетных данных установлен на «include», и учетные данные пользователя всегда включены.
  • "" - недопустимое ключевое слово и пустая строка будут обработаны как anonymous.
  • По умолчанию (то есть, когда атрибут не указан) CORS вообще не используется.
Атрибут crossorigin не поддерживается для rel="icon" в браузерах на базе Chromium.
