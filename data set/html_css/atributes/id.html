Атрибут id

Атрибут id устанавливает элементу уникальный идентификатор (ID), имя которого должно быть уникальным в документе (на странице). Атрибут используется для изменения стиля элеметна с этим id, навигации по странице и обращения к нему через скрипты. Идентификатор в коде документа должен быть в единственном экземпляре, иными словами, встречаться только один раз. Элементы могут иметь только один id.

<div id="идентификатор">text</div>
В качестве значения атрибут id принимает строку в виде:
  • состоящую по меньшей мере из одного символа;
  • начинается с латинской буквы;
  • идентификатор не должен начинаться с цифры;
  • не должен содержать пропусков (пробелов, табуляции и т.д);
  • допустимо использовать латинские буквы, цифры, символ подчёркивания, дефис, точку.
В HTML4 идентификатор не должен начинаться с цифры и содержать другие символы, кроме дефиса, подчёркивания и точки. Придерживайтесь этих правил, если вам требуется взаимодействие идентификаторов с JavaScript или совместимость со старыми браузерами.
