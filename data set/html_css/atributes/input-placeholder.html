Атрибут placeholder.

Данный атрибут доступен для элементов input и textarea.

Выводит текст внутри поля формы, который исчезает при получении фокуса или при наборе текста. Обычно отображается серым цветом. В качестве значения используется текстовая строка или слово. Если внутри строки предполагается пробел, её необходимо брать в двойные или одинарные кавычки.

Aтрибут placeholder может добавляться только к полю формы, у которого type имеет одно из значений: text, password, search, url, tel, email.

<input placeholder="текст">
Есть случаи, в которых атрибут placeholder никогда не отображается пользователю, поэтому форма должна быть понятна и без него.
<form action="handler.php">
  <input type=search placeholder="Введите текст для поиска">
  <input type="submit" value="Искать">
</form>
