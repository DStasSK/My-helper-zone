Атрибут itemprop

Атрибут itemprop используется для добавления свойств словаря микроданных к элементу HTML. Значения свойств представляют собой либо строку, либо URL и могут быть связаны с очень широким диапазоном элементов. Когда значением атрибута является URL, оно выражается с помощью элемента a и его атрибута href, элемента img и его атрибутa src или других элементов, которые ссылаются на внешние ресурсы или встраивают их.

<div itemprop="свойство">
  ...
</div>
"cвойство" - определяется словарём данных. К примеру, на https://schema.org, которую поддерживает Яндекс и Google, перечислены словари с их возможными свойствами.
