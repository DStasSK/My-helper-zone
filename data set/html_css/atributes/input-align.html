Атрибут align элемента input

Определяет, как выравнивать поле с изображением относительно текста или других элементов формы. Данный атрибут устарел, он хоть и поддерживается, но лучше использовать стили.

<input type="image" align="bottom | left | middle | right | top">
  • bottom - выравнивание нижней границы изображения по окружающему тексту.
  • left - выравнивание изображения по левому краю окна.
  • middle - выравнивание середины изображения по базовой линии текущей строки.
  • right - выравнивание изображения по правому краю окна.
  • top - верхняя граница изображения выравнивается по самому высокому элементу текущей строки.
