Атрибут itemid

Атрибут itemid предоставляет микроданные в форме уникального глобального идентификатора элемента. Атрибут itemid может быть указан только для элемента, у которого одновременно заданы атрибуты itemscope и itemtype. Также, itemidможет быть указан только для элементов, обладающих атрибутом itemscope, соответствующий ссылке itemtype или определяющий словарь, поддерживающий глобальные идентификаторы. Точное значение глобального идентификатора itemtype обеспечивается определением этого идентификатора в пределах указанного словаря.

<div itemid=" url | urn | text ">
  ...
</div>

В качестве значения указывается относительный или абсолютный адрес документа, который служит идентификатором. Кроме того, допустимо задавать и обычный текст.

К примеру, для словаря http://schema.org/Book, который описывает книги, в качестве идентификатора может служить ISBN — уникальный международный номер книжного издания.

Игра престолов

Джордж Мартин
Твёрдый переплёт
Эпическая, чеканная сага о мире Семи Королевств...
<div itemscope itemtype="http://schema.org/Book"
               itemid="isbn:978-5-17-075250-8">
  <h2 itemprop="name">Игра престолов</h2>
  <div itemprop="author">Джордж Мартин</div>
  <div itemprop="bookFormat">Твёрдый переплёт</div>
  <div itemprop="description">
    Эпическая, чеканная сага о мире Семи Королевств...
  </div>
</div>
Структура данных словаря:
itemscope itemtype:itemid http://schema.org/Book: urn:isbn:978-5-17-075250-8
itemprop name Игра престолов
itemprop author Джордж Мартин
itemprop bookFormat Твёрдый переплёт
itemprop description Эпическая, чеканная сага о мире Семи Королевств...
