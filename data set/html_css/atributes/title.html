Атрибут title

Атрибут title добавляет всплывающую подсказку с текстом, которая появляется при наведении курсора на элемент. Задаваемый текст представляющий собой справочную информацию, относящуюся к элементу, к которому он принадлежит. Вид такой подсказки зависит от браузера, настроек операционной системы и не может быть изменён напрямую с помощью HTML-кода или стилей.

<div title="текст всплывающей подсказки">...</div>

Атрибут title может содержать несколько строк так как переносы текста в значении атрибута остаются и отображаются как есть. Каждый символ U+000A (LINE FEED или LF) представляет собой разрыв строки.

Этот элемент span имеет атрибут title с двумя строками. 
Этот <span title="Здесь есть
перенос строки" class="tdu">элемент span</span> имеет
атрибут title с двумя строками.

<pre id="output" title="my id = 'output'"></pre>

<script>
  const span = document.querySelector("#span");
  const output = document.querySelector("#output");
  output.textContent = span.title;
</script>
