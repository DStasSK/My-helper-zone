Атрибут loading.

Данный атрибут доступен для элементов img и iframe.

Aтрибут loading позволяет браузеру откладывать загрузку содержимого элементов, находящихся за пределами видимой области страницы, до тех пор, пока пользователь, прокручивая страницу, не окажется достаточно близко к этим элементам. Это помогает оптимизировать загрузку содержимого документа, откладывая загрузку элемента до тех пор, пока оно не понадобится, а не сразу во время начальной загрузки страницы.

<img loading="lazy | eager" />
  • lazy - загружает элемент только тогда, когда он попадает в видимую область страницы.
  • eager - значение по умолчанию, загрузить элемент без задержки.
Загрузка откладывается только при включенном JavaScript. Это мера против отслеживания.
