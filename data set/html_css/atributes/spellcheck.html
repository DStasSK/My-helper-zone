Атрибут spellcheck

Атрибут spellcheck указывает браузеру проверять или нет правописание и грамматику в тексте. Хотя атрибут можно устанавливать практически для всех элементов, результат будет заметен только для полей форм (input, textarea), а также редактируемых элементов у них установлен атрибут contenteditable. При включении проверки слова с ошибками подчёркиваются красной волнистой линией. Браузер может не поддерживать проверку орфографии или она может быть отключена.

<textarea spellcheck="true | false"></textarea>
  • true - или пустая строка включает режим проверки правописания.
  • false - отключает проверку текста.
  • Значение по умолчанию - зависит от настроек браузера.
Обычно нередактируемые элементы не проверяются на наличие орфографических ошибок даже если атрибут spellcheck установлен в значение true и браузер поддерживает проверку орфографии.
Использование проверки орфографии может иметь последствия для безопасности и конфиденциальности пользователей. Спецификация не регламентирует, как выполняется проверка орфографии, и содержимое элемента может быть отправлено третьей стороне для получения результатов проверки орфографии. Cледует отключать проверку орфографии для элементов, которые могут содержать конфиденциальную информацию.

В мобильных браузерах отключение проверки орфографии spellcheck="false" не даёт эффекта, слова с ошибками подчёркиваются красной волнистой линией.
