Атрибут autofocus

Атрибут autofocus — это логический атрибут, указывающий, что элемент должен быть сфокусирован при загрузке страницы или при отображении элемента dialog, частью которого он является. Допускается не более одного элемента в документе или диалоговом окне с атрибутом autofocus, в противном случае фокус будет получен первым элеметном.

<input autofocus />
Атрибут autofocus можно применить к любому элементу, хотя обычно его применяют к элементам формы.

Когда назначается автофокусировка, экранные ридеры «телепортируют» своего пользователя к элементу управления формы, не предупреждая его заранее. Это может сбить с толку людей с нарушениями зрения, использующих технологию чтения с экрана, и людей с когнитивными нарушениями. Хотя экранный ридер озвучит метку элемента управления формы, получающего фокус, экранный ридер не озвучит ничего перед меткой, и зрячий пользователь на небольшом устройстве также пропустит контекст, созданный предыдущим содержимым.

Браузеры, которые не поддерживают атрибут autofocus, будут его игнорировать. Если есть необходимость автофокуса на элементе, то можно помимо атрибута autofocus в коде HTML использовать JS для проверки поддержки браузером атрибута autofocus, и если нет, запустить собственный скрипт.

<form name="f">
  <input id="q" autofocus>
  <script>
    if (!("autofocus" in document.createElement("input"))) {
      document.getElementById("q").focus();
    }
  </script>
  <input type="submit" value="Вперёд">
</form>
