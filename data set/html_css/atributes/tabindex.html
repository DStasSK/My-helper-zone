Атрибут tabindex

Атрибут tabindex позволяет делать HTML-элементы фокусируемыми, а также разрешать или запрещать им последовательную фокусировку (обычно с помощью клавиши Tab ) и определять их относительный порядок для последовательной навигации по фокусу. В качастве значения принимает целое число не более 32767.

  • Если установить атрибут tabindex="-1", то к нему нельзя будет обратиться через клавищу Tab для фокуса, но это делает возможным вокусировку на элементе с помощью JavaScript.
  • Значение атрибута tabindex="0" означает, что элемент должен быть доступен для фокусировки при последовательной навигации с помощью клавиатуры, но после любых положительных значений tabindex. Порядок фокусировки этих элементов определяется их порядком в исходном документе.
  • Положительное значение атрибута делает элемент доступным для фокусировки при последовательной навигации с помощью клавиатуры.
  • Если атрибут tabindex установлен в элементе без заданного значения, то возможность фокусировки на нем определяется настройками браузера.
Порядок табуляции по элементам с tabindex определяется значением числа - от меньшего к большему, потом "0" и элемнты с tabindex без заданного значения. Элементы с одинаковым значением tabindex, получаю фокус в порядке их положения в документе.

Некоторым элементам HTML браузеры по умолчанию устанавливают tabindex="0". Не следует добавлять атрибут tabindex к этим элементам, если он не изменяет поведение по умолчанию:

  • элементы a и area с атрибутом href.
  • button, frame, iframe, input, object, select, textarea.
  • элемент summary, который предоставляет сводку для элемента details.
Aтрибут tabindex нельзя использовать в элементе dialog.

Если идет запрос к странице по хешу, например: site.ru/#blabla и на странице есть элемент с id равным "blabla", то при нажатии на на клавиатуре клавиши Tab порядок получения фокуса изменится. Фокус получит ближайший элемент возле элемента с этим id.
