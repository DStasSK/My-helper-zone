Атрибут style

Атрибут style Применяется для определения стиля элемента с помощью правил CSS. В качестве значений указываются стилевые правила: вначале следует имя стилевого свойства, затем через двоеточие его значение. Стилевые свойства разделяются между собой точкой с запятой.

<div style="property: value; | property: value;">...</div>
  • property - название CSS свойства.
  • value - значение CSS свойства.

Рекомендуется определять стили в отдельном файле или файлах. Атрибут style и элемент style в основном предназначены для быстрого добавления стилей, например, для целей тестирования.

Нужно помнить, что специфичность параметров css свойств, добавленных таким образом, выше чем в CSS-файле (без использования !important) и при необходимости их будет сложно переопределить, только если через JS.
