Атрибут hidden

Атрибут hidden cкрывает содержимое элемента от просмотра. Такой элемент не отображается на странице, но доступен через скрипты. Атрибут может: не содержать значение, иметь в качестве значения пустую строку, эти и все любые значения, кроме until-found - делают элемент скрытым. По умолчанию атрибут выключен.

<div hidden>text</div>                  <!-- элемент скрыт -->
<div hidden="">text</div>               <!-- элемент скрыт -->
<div hidden="hidden">text</div>         <!-- элемент скрыт -->
<div hidden="until-found">text</div>    <!-- элемент скрыт до обнаружения -->
<div hidden="abracatabra">text</div>    <!-- элемент скрыт -->
Если через стили установлено свойство display со значением отличным от none, то элемент не будет скрыт. Если свойство display не установлено, то элемент будет скрыт.

При значении атрибута hidden="until-found" элемент скрыт, но его содержимое будет доступно для функции браузера «найти на странице» или для навигации по фрагментам. Когда эти функции вызывают прокрутку к элементу, браузер: выполнит событие beforematch для скрытого элемента, удалит атрибут hidden из элемента и прокрутит страницу к элементу, кроме случая когда свойство display имеет значение none.

Если браузер не поддерживает перечислимое значение until-found атрибута hidden, то элемент будет скрыт так как значение свойства будет считаться не валидным.
