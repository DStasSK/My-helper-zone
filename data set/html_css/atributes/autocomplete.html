Атрибут autocomplete.

Данный атрибут доступен для элементов input, которые принимают текстовое или числовое значение в качестве входных данных, элементов textarea, элементов select и элементов form.

Этот атрибут помогает заполнять поля форм текстом, который был введён в них ранее. Значения сохраняет и подставляет браузер. Aвтозаполнение по соображениям безопасности может отключаться пользователем в настройках браузера и не может в таком случае управляться атрибутом autocomplete. К качестве значения атрибут принимает ключевые слова off или on, либо упорядоченный список токенов, разделенных пробелами.

<input autocomplete="off">
<input autocomplete="on">
<input autocomplete="shipping street-address">
Значение по умолчанию - зависит от настроек браузера

Атрибут autocomplete может добавляться к полю формы, у которого атрибут type имеет одно из значений: text, password, search, url, tel, email, date, month, week, time, datetime-local, number, range, color.

При вводе первых букв текста выводится список сохранённых ранее значений, из которого можно выбрать необходимое. Автозаполнение поля формы привязывается к его атрибуту name и при его смене - теряется.

Если элемент input, select или textarea не имеет атрибута autocomplete, браузер будет использовать атрибут autocomplete формы, внутри которой вложен элемент. Либо формой у которой идентификатор совпадает с идентификатором элемента (если id присутствует).

form 12 14 16 6 4 4.4 4 14 6
input 12 14 12.1 6 4 4.4 4 12.1 6
select 79 66 53 9.1 59 66 59 47 9.3
textarea 79 66 53 9.1 59 66 59 47 9.3
