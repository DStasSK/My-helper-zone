Атрибут itemtype

Атрибут itemtype указывает адрес словаря, который будет применяться для определения свойств элемента в структуре данных. Яндекс и Google поддерживают стандарт разметки Schema.org, соответственно, в качестве значения itemtype указывается адрес словаря на этом сайте.

<div itemscope itemtype="адрес">
  ...
</div>
  • Атрибут itemtype может быть указан только для элементов, у которых указан атрибут itemscope.
  • Атрибут itemtype должен иметь значение, которое является неупорядоченным набором уникальных токенов, чувствительных к регистру, каждый из которых является допустимым и абсолютным URL-адресом, и все они определены для использования одного и того же словаря. Значение атрибута должно иметь по крайней мере один токен.
  • Все типы элементов должны быть типами, определенными в применимых спецификациях (например, schema.org), и все они должны быть определены для использования одного и того же словаря.
  • Атрибут itemid может быть указан только для элементов, у которых указаны как атрибут itemscope, так и атрибут itemtype. Они должны быть указаны только для элементов с атрибутом itemscope, атрибут itemtype которых указывает словарь, не поддерживающий глобальные идентификаторы для элементов, как определено спецификацией этого словаря.
  • Точное значение глобального идентификатора определяется спецификацией словаря. Такие спецификации определяют, разрешено ли существование нескольких элементов с одним и тем же глобальным идентификатором (на одной странице или на разных страницах), а также каковы правила обработки для этого словаря с точки зрения обработки случая наличия нескольких элементов с одним и тем же идентификатором.
