Атрибут for.

Данный атрибут доступен для элементов output и label.

При использовании в элементе label он указывает идентификатор id элемента формы, к которому он относится. Атрибут for необходимо задавать в том случае, когда элемент формы и текст разделены. Если элемент формы размещается внутри контейнера label, то применять атрибут for не требуется.

<label for="username">Your name</label>
<br><input type="text" id="username" />

При использовании в элементе output атрибут for имеет значение, представляющее собой разделенный пробелами список значений идентификаторов элементов, которые используются для создания выходных данных.

<input type="range" id="b" name="b" value="50" /> +
<input type="number" id="a" name="a" value="10" /> =
<output name="result" for="a b">60</output>
Значение атрибута for чуствительно к регистру.
