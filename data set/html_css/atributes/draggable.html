Атрибут draggable

Атрибут draggable указывает, можно ли перетаскивать элемент с помощью собственного поведения браузера, либо с помощью HTML Drag and Drop API. Атрибут draggable может быть применен к элементам, которые строго попадают под пространство имен HTML, что означает, что его нельзя применять к SVG.

<div draggable="true | false | auto">
  text
</div>
  • true - Разрешает перетаскивание элемента.
  • false - Запрещает перетаскивание элемента.
  • auto - Задаёт поведение браузера по умолчанию.

Атрибут draggable перечисляемый, а не логический. Значение true или false является обязательным, а сокращение типа <img draggable> запрещено. Если этот атрибут не задан, его значение по умолчанию — auto, что означает, что поведение перетаскивания — это поведение браузера по умолчанию: можно перетаскивать только выделенные тексты, изображения и ссылки. Для других элементов необходимо задать событие ondragstart, чтобы работало перетаскивание

element.draggable [ = value ] // JS

Возвращает true, если элемент можно перетаскивать; в противном случае возвращает false. Можно задать, чтобы переопределить значение или задать атрибут элемента.

спецификация

Пример.

Переместите картинку в прямоугольник.

<p>Переместите картинку в прямоугольник.</p>
<div class="ptrash"
     ondrop="drop(event)" ondragover="allowDrop(event)"></div>
<img src="help/img/fig.jpg" alt="" id="photo"
     draggable="true" ondragstart="drag(event)">

<script>
  function allowDrop(ev) {ev.preventDefault();}
  function drag(ev) {ev.dataTransfer.setData("text", ev.target.id);}
  function drop(ev) {
    ev.preventDefault();
    var data = ev.dataTransfer.getData("text");
    ev.target.appendChild(document.getElementById(data));
  }
</script>
