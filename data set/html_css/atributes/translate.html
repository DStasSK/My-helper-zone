Атрибут translate

Атрибут translate используется для указания того, следует ли переводить значения переводимых атрибутов элемента и его дочерние элементы текстового узла при локализации страницы или оставить их без изменений.

<div translate="yes | no">...</div>
  • yes - или пустая строка - элемент должен быть переведен при локализации страницы.
  • no - текст элемента не должен быть переведен.

Значения этого атрибута учитывается автоматическими системами перевода, такими как Google Translate. Поэтому важно, чтобы веб-авторы использовали этот атрибут для обозначения контента, который не следует переводить.
