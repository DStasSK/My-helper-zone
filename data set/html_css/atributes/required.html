Атрибут required.

Данный атрибут доступен для элементов input и textarea.

Этот атрибут указывает, что пользователь должен заполнить значение элемента перед отправкой формы. Если обязательное поле пустое, браузер выведет сообщение об ошибке, а форма отправлена не будет. Вид и содержание сообщения зависит от браузера и меняться пользователем не может.
