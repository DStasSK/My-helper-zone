Атрибут itemscope

Атрибут itemscope - это логический атрибут, который определяет область действия связанных метаданных. Задаёт область действия словаря в структуре данных. Как правило, работает совместно с атрибутом itemtype и задаёт пределы, где itemtype будет активен. Элемент itemscope, не имеющий связанного itemtype, должен иметь связанный itemref.
