Атрибут accesskey

Атрибут accesskey генерирует сочетания клавиш для доступа к текущему элементу. Состоит из разделенного пробелами списка символов. В качестве значений принимает цифру (0-9) или латинскую букву (a-z). Браузер в первую очередь выбирает те клавиши, которые существуют на раскладке клавиатуры. Применяется к следующим элементам: a, area, button, input, label, legend, textarea.

При использовании атрибута accesskey следует учитывать:
  • Значение атрибута accesskey может конфликтовать с комбинациями клавиш системы, браузера или вспомогательными инструментами доступности.
  • Определённые значения атрибута accesskey могут отсутствовать на некоторых клавиатурах. Это особенно сильно проявляется, если вы используете язык отличный от английского.
  • Не существует стандартизированного способа информировать пользователя о наличии атрибута accesskey на элементе. Неинформированный пользователь может случайно активировать элемент с атрибутом accesskey.
Атрибут accesskey позволяет получить доступ к элементу с помощью сочетания клавиш с заданной в атрибуте буквой или цифрой. Браузеры в windows при этом используют различные комбинации клавиш:
  • IE:  Alt  +  key 
  • Chrome:  Alt  +  key 
  • Safari:  Alt  +  key 
  • Opera:  Shift  +  Esc ,  key 
  • Firefox:  Shift  +  Alt  +  key
