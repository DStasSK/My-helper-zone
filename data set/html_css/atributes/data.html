Атрибут data-*

Атрибут data-* создает пользовательский атрибут с данными, которые позволяют хранить и обмениваться информацией между HTML и его представлением DOM посредством JS. Данные могут быть получены с помощью скриптов или через стилевую функцию attr(). Символ * может быть заменён любым именем, соответствующим производственному правилу имён xml, со следующими ограничениями:

  • имя не должно содержать точку с запятой (U + 003A).
  • имя не должно содержать заглавных букв.
  • имя не должно начинаться с xml, в любом случае для этих букв используется регистр.
  • можно использовать латинские буквы в нижнем регистре, цифры и символы: дефис ( - ), двоеточие ( : ), подчёркивание ( _ ).
<div id="user"
     data-id="1234567890"
     data-user="Вася Пупкин" 
     data-date-of-birth="01.04.1990">Пользователь</div>
При использовании в JS имена атрибутов трансформируются в переменные, к которым в дальнейшем можно обращаться и получать значения, по следующим правилам:
  • часть названия атрибута data- идещее первым, удаляется;
  • любой дефис в названии атрибута идущий перед буквой удаляется, а буква за ним становится заглавной;
  • любые другие буквы остаются неизменными.
Например, атрибут data-date-of-birth преобразуется в переменную dateOfBirth.

Для обращения к атрибутам и получения их значений через JS для выбранного элемнта применяется метод dataset. Он же используется и для установки нового значения атрибута.

значение = элемент.dataset.атрибут
элемент.dataset.атрибут = значение
Здесь атрибут — это переменная, полученная путём изменения атрибута по вышеприведённым правилам: dateOfBirth, а не data-date-of-birth или date-of-birth.
Пользователь
<div id="user"
     data-id="1234567890"
     data-user="Вася Пупкин" 
     data-date-of-birth="01.04.1990">Пользователь</div>

<script>
  let el = document.getElementById('user'); // элемент с id="user"
  let id = el.dataset.id;              // значение атрибута data-id
  let user = el.dataset.user;          // значение атрибута data-user
  let dob = el.dataset.dateOfBirth;    // значение атрибута data-date-of-birth

  el.dataset.ban = 'забанен';          // новый атрибут data-ban и его значение
  console.log(user);                   // Вася Пупкин
  console.log(dob);                    // 01.04.1990
</script>

<style>
  #user::after { content: ' ' attr(data-user) ' - ' attr(data-ban); }
</style>
