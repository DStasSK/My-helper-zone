Атрибут formnovalidate элемента input.

Данный атрибут отменяет встроенную проверку данных, введённых пользователем в форме, перед отправкой формы на корректность. Такая проверка делается браузером автоматически для полей <input type="email">, <input type="url"> даже при наличии атрибута pattern или required у input.

Атрибут formnovalidate может добавляться только к кнопке для отправки формы (<input type="submit">) или к кнопке с изображением (<input type="image">).

<form action="handler.php">
  <p><input required placeholder="Имя"></p>
  <p><input type="email" required placeholder="Ваш email"></p>
  <p><input type="url" required placeholder="Сайт"></p>
  <p><input type="submit" formnovalidate value="Отправить без проверки"></p>
</form>
