Атрибут min.

Данный атрибут доступен для элементов meter и элементам input у которых атрибут type имеет значение date, month, week, time, datetime-local, number, range.

Атрибут min определяет минимальное значение, которое является приемлемым и действительным для входных данных, содержащих атрибут. Если значение элемента меньше этого, элемент не проходит проверку. Это значение должно быть меньше или равно значению атрибута max.

Если атрибут min присутствует, но его значение не указано или является недействительным, атрибут не применяется. Если атрибут min указан и вводимое значение меньше допустимого атрибутом min, проверка ограничений предотвратит отправку формы.

Если введенные пользователем данные не соответствуют установленному минимальному значению, значение считается недействительным при проверке ограничений и будет соответствовать псевдоклассам :out-of-range и :invalid.

Для элемента meter атрибут min определяет нижнюю числовую границу измеряемого диапазона. Оно должно быть меньше максимального значения (атрибут max), если оно указано. В обоих случаях, если оно опущено, значение по умолчанию равно 0.

36,6 градусов

<meter value="36.6" min="35" max="42">36,6 градусов</meter>

Для элемента input атрибут min устанавливает нижнее значение для ввода числа или даты в поле формы. В качестве значения использует:

  • Целое положительное или отрицательное число (для type="number", type="range").
  • Дата в формате ГГГГ-ММ-ДД (например: 2012-12-22) для type="date".
 <input type="date" min="2019-12-25" step="1"> 
 <input type="month" min="2019-12" step="12"> 
 <input type="week" min="2019-W23" step=""> 
 <input type="time" min="10:00" step="900"> 
 <input type="datetime-local" min="2019-12-25T13:59"> 
 <input type="number" min="0" step="5" max="100"> 
 <input type="range" min="60" step="5" max="100">
