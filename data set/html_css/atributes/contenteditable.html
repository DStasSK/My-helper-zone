Атрибут contenteditable

Атрибут contenteditable указывает на возможность редактирования элемента пользователем. Если это так, браузер изменит свой виджет, чтобы разрешить редактирование. Значение наследуется, применяется ко всем элементам.

<div contenteditable="true | false">
  <span>text</span>
</div>
  • false - элемент является нередактируемым;
  • true - элемент является редактируемым. Вместозначения true допустимо указывать пустое значение (contenteditable="")
Внесенные изменения не сохраняются в исходном html файле. После обновления страницы они пропадут.
