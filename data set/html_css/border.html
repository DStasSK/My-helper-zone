CSS cвойство border позволяет задавать толщину, цвет и тип линии по периметру вокруг элемента. Параметры данного свойства могут записываться в одной строке, разделяясь через пробел и в любом порядке.

div {
  border: border-width || border-style || border-color;
}
  • border-width - определяет толщину границы
  • border-color - устанавливает цвет границы в любом CSS формате
  • border-style - определяет вид границы
    • dotted
    • dashed
    • solid
    • double
    • groove
    • ridge
    • inset
    • outset

Для установки границы только на определённых сторонах элемента, воспользуйтесь свойствами border-top, border-bottom, border-left, border-right - позволяет одновременно установить толщину, стиль и цвет границы элемента. Значения могут идти в любом порядке, разделяясь пробелом.

border-top: [border-top-width || border-top-style || border-top-color]

CSS cвойство border-style.

Устанавливает стиль границы вокруг элемента. Допустимо задавать индивидуальные стили для разных сторон элемента. Значение по умолчанию none, не наследуется, применяется ко всем элементам, не анимируется.

Для управления видом границы предоставляется несколько значений свойства border-style. Вид зависит от используемого браузера и заданной толщины границы.

div {
  border-style: none | hidden | dotted | dashed | solid |
                double | groove | ridge | inset | outset;
}
none не отображает границу и её толщина (border-width) задаётся нулевой.
hidden имеет тот же эффект, что и none за исключением применения border-style к ячейкам таблицы, у которой значение свойства border-collapse установлено как collapse. В этом случае вокруг ячейки граница не будет отображаться вообще.

Разрешается использовать одно, два, три или четыре значения, разделяя их между собой пробелом. Если указано одно значение - стиль границы будет задан для всех сторон элемента. Если указано два значения - первое значение устанавливает стиль верхней и нижней границ, второе - левой и правой. Если указано три значения - первое значение задаёт стиль верхней границы, второе - одновременно левой и правой границ, а третье - нижней границы. Если указано четыре значения - поочередно устанавливается стиль верхней, правой, нижней и левой границ.

Свойство отдельных участков границы.

Разноцветная рамка
.elem {
  border-top:8px solid red;
  border-bottom:8px solid blue;
  border-left:8px solid grey;
  border-right:8px solid chocolate;
}

Штриховая рамка (градиент).

Штриховая рамка
{
  border: 10px solid transparent;
  border-image: 10 repeating-linear-gradient
  (135deg, #C42061, #C42061 10px, transparent 10px, 
  transparent 20px, #FBCFDF 20px, #FBCFDF 30px,
  transparent 30px, transparent 40px);
}
Штриховая рамка
{
  border: 10px solid transparent;
  border-image: 10 repeating-linear-gradient
  (135deg, black, black 10px, gold 10px, gold 20px);
}
