CSS свойство gap

Свойство gap — это короткая запись значений свойств row-gap и column-gap, определяющее размер промежутка между строками и столбцами в макете flexbox, grid или в макете с несколькими столбцами.

div {
   display: flex|grid;
   gap: row-gap column-gap | initial | inherit;
}
  • row-gap - размер промежутка между строками в макете.
  • column-gap - размер промежутка между столбцами в макете.
  • initial - устанавливает значение по умолчанию.
  • inherit - наследует значение от родительского элемента.
div {
   display: flex;
   gap: 20px  40px;
}

div {
   display: flex;
   gap: 40px;  /* row-gap == column-gap == 40px  */
}
