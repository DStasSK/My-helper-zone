CSS-функция image-set() позволяет перечислить несколько фоновых изображений с условиями, по которым браузер решит, какое из них наиболее подходящее для загрузки в данный момент.

Похож по принципу работы на тег picture.

На сайт могут заходить с разных устройств. У многих дисплеи с повышенной плотностью пикселей и им нужно показывать картинку хорошего качества. А на телефонах лучше показывать небольшие файлы ради ускорения загрузки страницы. Нужно как-то давать разным клиентам разные версии изображений. Раньше для этого использовали медиавыражения, а теперь есть простые и удобные решения, которые позволяют через запятую указать все версии изображений.

Внутри круглых скобок указываем через запятую две и более ссылки на изображения при помощи функции url() и пишем условие для выбора этой картинки. В качестве условия можно использовать коэффициент плотности пикселей: 1x, 2x и так далее. Или указываем тип изображения в формате MIME. Картинка будет загружена, если формат поддерживается браузером. Можно комбинировать условия.

Порядок перечисления форматов картинок важен: так сначала необходимо указывать более предпочтительные (AVIF, WebP), потом — менее (PNG, JPEG). Порядок перечисления картинок разной плотности не важен, браузер сам выберет подходящую.

div {
   background-image: url("pick.png");

   /* Для Chrome и Safari (в Safari не отображается AVIF) */
   background-image: -webkit-image-set(
      url("pick@2x.png") 2x,
      url("pick@1x.png") 1x
   );

   /* Для Firefox */
   background-image: image-set(
      url("pick@2x.avif") type("image/avif") 2x,
      url("pick@1x.avif") type("image/avif") 1x,
      url("pick@2x.webp") type("image/webp") 2x,
      url("pick@1x.webp") type("image/webp") 1x,
      url("pick@2x.png") 2x,
      url("pick@1x.png") 1x
   );
}
