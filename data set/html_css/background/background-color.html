background-color

Свойство background-color определяет цвет фона элемента. Хотя это свойство не наследует свойства своего родителя, из-за того, что начальное значение устанавливается прозрачным, цвет фона дочерних элементов совпадает с цветом фона родительского элемента. Параметрами свойства являются цвета, в формате шестнадцатеричного кода, форматов RGB, RGBA, HSL, HSLA и других стандартных значений. Значение по умолчанию transparent.

.elem {
  background-color: yellow;
  background-color: transparent;
  background-color: inherit;
  background-color: #467892;
  background-color: rgb(249, 201, 16);
  background-color: rgba(163, 110, 29, 0.8);
  background-color: hsl(212, 31%, 46%);
  background-color: hsla(129, 46%, 47%, 0.8);
}
