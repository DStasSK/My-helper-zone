background-origin

Свойство background-origin определяет область позиционирования фонового рисунка. Это свойство не применяется, когда значение свойства background-attachment задано как fixed.

.elem {
  background-origin: padding-box | border-box | content-box;
}
  • padding-box - значение по умолчанию, фон позиционируется относительно края элемента - до его границ.
  • border-box - фон позиционируется относительно границы, при этом линия границы перекрывает фон.
  • content-box - фон позиционируется относительно содержимого элемента - до падинга.
Если фон задан один, а значений background-origin несколько, то браузеры покажут разное поведение. Firefox и Opera используют первое значение, Chrome и Safari создадут несколько фоновых рисунков.
background-origin:
padding-box;
background-origin:
border-box;
background-origin:
content-box;
