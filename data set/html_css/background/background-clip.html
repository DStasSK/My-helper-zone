background-clip

Cвойство background-clip oпределяет, как цвет фона или фоновая картинка должна выводиться под границами. Эффект заметен при прозрачных или пунктирных границах. Применяется ко всем элементам, не наследуется, не анимируется. Значение по умолчанию border-box.

.elem {
  background-clip: padding-box | border-box | content-box | text;
}
  • padding-box - параметр устанавливает вывод фона внутри границ.
  • border-box - отображение фона под границами (по умолчанию).
  • content-box - отображение фона только внутри контента.
  • text - отображение фона только внутри текста.
В Chrome, Safari, Opera и Android значение text работает только со свойством -webkit-background-clip.

padding-box;

border-box;

content-box;

text;
