Функция linear-gradient()

Стилевая функция linear-gradient() добавляет линейный градиент к фону элемента. Она выступает в качестве значения свойств background-image или background.

.elem {
  background: linear-gradient(направление, цвет1 ||, цвет2...);
}
Перед определением любого градиентного фона необходимо задать свойству background цвет по умолчанию. Такой цвет должен быть использован в качестве запасного, если браузер не поддерживает градиент.

Направление градиента задаётся или указанием необходимого угла наклона градиентной линии, или при помощи ключевых слов. По умолчанию направление градиента задаётся снизу - вверх.

  • Угол наклона градиентной линии, который показывает направление градиента, отсчитывается от вертикальной линии. В качестве значений обычно используются градусы - deg, при этом допускается отрицательное значение угла. Нулю градусов (или 360°) соответствует градиент снизу вверх, далее отсчёт ведётся по часовой стрелке.
  • При использовании ключевых слов - вначале пишется слово to, а затем добавляются ключевые слова top, bottom, left, right, а также их сочетание из двух слов. Порядок слов не важен, можно написать to left top или to top left. Ключевые слова указывают - в каком направлении будет производится градиентная заливка используемыми цветами.

Например, когда мы применяем диагональный градиент под 45° в не совсем квадратном элементе, градиент не переходит напрямую из одного угла в другой. Вместо этого градиент определит абсолютный центр элемента, расчитает опорные точки, откуда он должен продвигаться (так чтобы цвета начинались и оканчивались в границах элемента) и только затем двинется в направлении угла, заданном в значении.

При использовании ключевых слов, если мы хотим, чтобы градиент в элементе двигался слева направо, мы можем использовать значение to right для указания направления, в котором линейный градиент должен продвигаться. Если мы хотим, чтобы градиент переходил от левой верхней до правой нижней части элемента (по диагонали), мы можем использовать значение ключевого слова to right bottom.

.el1 {background: linear-gradient(0deg, #a0f, #fff);}
.el2 {background: linear-gradient(90deg, #a0f, #fff);}
.el3 {background: linear-gradient(to right, #a0f, #fff)}
.el3 {background: linear-gradient(to left, #a0f, #fff)}
.el3 {background: linear-gradient(to top, #a0f, #fff)}
.el3 {background: linear-gradient(to bottom, #a0f, #fff)}
.el3 {background: linear-gradient(to top left, #a0f, #fff)}
.el3 {background: linear-gradient(to bottom left, #a0f, #fff)}
.el3 {background: linear-gradient(-135deg, #a0f, #fff)}

Для корректного отображения цвета фона в различных браузерах необходимо добавлять префиксы, а для Internet explorer - использовать фильтры.

  • Opera до версии 12.10 поддерживает -o-linear-gradient().
  • Firefox до версии 16 поддерживает -moz-linear-gradient().
  • Chrome до версии 26, Safari до версии 6.1 и Android до версии 4.4 поддерживают -webkit-linear-gradient().
Все свойства с префиксами не используют ключевое слово to при задании направления градиента.

Стоп-цвета

Как минимум, градиентный фон переходит от одного цвета к другому, однако, мы можем добавить несколько цветов к градиенту и заставить браузер сделать по всем ним переход. Для этого мы добавим стоп-цвета в данную функцию градиента — с запятыми, которые отделяют один цвет от другого.

div{
  background: #648880;
  background:linear-gradient(to right, #f6f1d3, #648880, #293f50);
}

По умолчанию браузер будет позиционировать каждый стоп-цвет на равном расстоянии от следующего и переходить между ними соответственно. Если требуется больше контроля над тем, как желательно расположить цвета, то их места в градиенте могут быть определены для каждого цвета. Место должно быть объявлено как значение размера и идёт после значения цвета. Если не указано иное, то первый цвет будет позиционироваться на 0%, а последний цвет на 100%.

div{
  background: #648880;
  background:linear-gradient(to right, #f6f1d3, #648880 85%, #293f50);
}
div{
  background: #648880;
  background:linear-gradient(to right, black, #fff 10%, black);
}
div{
  background: #648880;
  background:linear-gradient(to right, black, #fff 10%, #fff 90%, black);
}

Функция repeating-linear-gradient()

Стилевая функция repeating-linear-gradient() создаёт бесконечно повторяющийся линейный градиент, образуя тем самым фоновый узор. По своему действию похожа на функцию linear-gradient() и имеет те же параметры. При этом цвета последовательно сменяют друг друга, после последнего цвета в списке цветов градиента сразу же идёт первый.

В функции linear-gradient() можно использовать несколько цветов, которые будут переходить из одного в другой на протяжении элемента. В функции repeating-linear-gradient() есть возможность задать интервалы для того или иного цвета общая сумма которых может быть меньше размера элемента, при этом дальнейшее чередование цветов зациклено.

div{
  background: #648880;
  background: linear-gradient(to right, #ffd, #688 40%, #345 50%);
}
div{
  background: #648880;
  background: linear-gradient(to right, #ffd, #688 40%, #345 50%, transparent 50%);
}
div{
  background: #648880;
  background: repeating-linear-gradient(to right, #ffd, #688 40%, #345 50%);
}
  • Chrome до версии 26, Safari до версии 6.1 и Android до версии 4.4 поддерживают -webkit-repeating-linear-gradient().
  • Opera до версии 12.10 поддерживает -o-repeating-linear-gradient().
  • Firefox до версии 16 поддерживает -moz-repeating-linear-gradient().
Все свойства с префиксами не используют ключевое слово to при задании направления градиента.
