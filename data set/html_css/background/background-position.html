background-position

Cвойство background-position задаёт начальное положение фонового изображения, установленного с помощью свойства background-image.

Cвойство background-position имеет два значения, положение по горизонтали и вертикали - они являются сокращенной записью для свойств background-position-x и background-position-y.

Допустимо указывать несколько значений для каждого фона, перечисляя значения через запятую. В качестве параметров для свойства background-position используются слова left, center, right, top, center, bottom или значения в процентах, пикселях или других единицах. Значения свойства применяются ко всем элементам, не наследуются, aнимируются. Значение по умолчанию: 0% 0%.

.elem {
  /* background:url(img/1.ipg) no-repeat #eee; */
  background-position: [background-position-x] [background-position-y], ...;
}
Отношение между используемыми ключевыми словами и процентной записью следующее.
Значение свойстваПоложение изображения
top = top center = center top = 50% 0%сверху в центре
right = right center = center right = 100% 50%справа по середине
bottom = bottom center = center bottom = 50% 100%снизу в центре
left = left center = center left = 0% 50%слева по середине
center = center center = 50% 50%по центру элемента
top left = left top = 0% 0%в левом верхнем углу
top right = right top = 100% 0%в правом верхнем углу
bottom left = left bottom = 0% 100%в левом нижнем углу
bottom right = right bottom = 100% 100%в правом нижнем углу

background-position-x - задаёт положение фонового изображения внутри элемента по горизонтали. Если для элемента установлено несколько фоновых изображений, то можно изменять положение каждого изображения, перечисляя значения через запятую.

background-position-y - задаёт положение фонового изображения внутри элемента по вертикали. Если для элемента установлено несколько фоновых изображений, то можно изменять положение каждого изображения, перечисляя значения через запятую.

.elem {
  background-position-y: [top | center | bottom | <проценты> | <размер>];
  background-position-x: [left | center | right | <проценты> | <размер>];
}
