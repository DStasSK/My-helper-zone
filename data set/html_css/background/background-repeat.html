background-repeat

Свойство background-repeat устанавливает характер повторения фонового изображения выводимого с помощью свойства background-image. С помощью данного свойства можно установить направление, в котором будет повторяться рисунок. Повторение фонового изображения возможно только по горизонтали или по вертикали, а так же в обе стороны. Допустимо указывать несколько значений для каждого фона, перечисляя значения через запятую.

.elem {
  background-repeat: repeat | repeat-x | repeat-y |space | round | no-repeat;
}
  • repeat – фоновый рисунок будет повторяться по горизонтали и вертикали.
  • repeat-x – Фоновый рисунок повторяется только по горизонтали.
  • repeat-y – Фоновый рисунок повторяется только по вертикали.
  • space – изображение повторяется столько раз, чтобы полностью заполнить область; если это не удаётся, между картинками добавляется пустое пространство.
  • round – изображение повторяется так, чтобы в области поместилось целое число рисунков; если это не удаётся сделать, то фоновые рисунки масштабируются.
  • no-repeat – запрет на повторение фонового изображения. Положение фонового изображения в элементе определяется свойством background-position (по умолчанию в левом верхнем углу)
  • inherit – наследует значение.
Для каждого фона допустимо указывать два значения, первое ключевое слово задаёт повторение по горизонтали, второе по вертикали: например значение repeat-x аналогично repeat no-repeat
<div class="bg">...</div>

.bg {
  background-image: url(img/bg_line.jpg);
  background-color: #fff;
  background-repeat: repeat-y; /* аналогично: no-repeat repeat */
}
