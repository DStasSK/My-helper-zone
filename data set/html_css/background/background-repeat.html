background-repeat

Свойство background-repeat устанавливает характер повторения фонового изображения выводимого с помощью свойства background-image. С помощью данного свойства можно установить направление, в котором будет повторяться рисунок. Повторение фонового изображения возможно только по горизонтали или по вертикали, а так же в обе стороны.

background-repeat: repeat;
background-repeat: repeat-x;
background-repeat: repeat-y;
background-repeat: space;
background-repeat: inherit;
background-repeat: no-repeat;
repeat – фоновый рисунок будет повторяться по горизонтали и вертикали.
repeat-x – значение задающее повторение только по горизонтали.
repeat-y – значение задающее повторение только по вертикали.
space – параметр задаёт повторение пока не заполнится свободная область.
inherit – наследует значение.
no-repeat – запрет на повторение фонового изображения. 
<div style="
  background-image: url(img/bg_line.jpg);
  background-repeat: repeat-y;">
