CSS свойство row-gap

Данное свойство определяет промежуток между строками в grid или flexbox макете.

div {
   display: flex | grid;
   row-gap: length | normal | initial | inherit
}
  • length - указанная величина зазора между строками в макете.
  • normal - эквивалентно значению по умолчанию - 1em.
  • initial - устанавливает значение по умолчанию - 1em.
  • inherit - наследует значение от родительского элемента.
