CSS свойство transition

  • CSS свойство transition-property
  • CSS свойство transition-duration
  • CSS свойство transition-timing-function
  • CSS свойство transition-delay
  • CSS свойство transition-behavior

CSS свойство transition позволяет одновременно задать значения для 4-х свойств одновременно: transition-property, transition-duration, transition-timing-function и transition-delay. Устанавливает эффект перехода между двумя состояниями элемента, они могут быть определены с помощью псевдокласса :hover или :active, а также динамически через JavaScript.

Свойство можно применять ко всем элементам и псевдоэлементам. Значение по умолчанию all 0s ease 0s

CSS свойство transition определяет изменение элемента, который можно назвать переходом. Можно установить один переход для всех свойств сразу, а можно устанавлявать изменения различных CSS свойств с различными параметрами с помощью нескольких переходов. Переходы отделяются между собой запятыми, в то время как сами параметры внутри перехода разделяются через пробел.

.elem_poli {
  transition: переход , переход , ...
}

.elem_mono {
  transition: none | transition-property
                     transition-duration
                     transition-timing-function
                     transition-delay;
}
Значение свойств внутри перехода указывают в следующем порядке:
  • transition-property - название свойства, по умолчанию all
  • transition-duration - длительность анимации, по умолчанию 0s
  • transition-timing-function - вид анимации, по умолчанию ease
  • transition-delay - задержка перед анимацией, по умолчанию 0s
Примеры:
.box {
  background-color: #2db34a;
  border-radius: 0px;
  transition: background-color .5s linear,
              border-radius .5s ease-in 1s;
}

.box:hover {
  background-color: #ff7b29;
  border-radius: 50%;
}
div {
/* Применить к 1 свойству */
  /* имя свойства | длительность */
  transition: margin-left 4s;

  /* имя свойства | длительность | задержка */
  transition: margin-left 4s 1s;

  /* имя свойства | длительность | временная функция | задержка */
  transition: margin-left 4s ease-in-out 1s;


  /* Применить к 2 свойствам */
  transition:
    margin-left 4s,
    color 1s;


  /* Применить ко всем изменённым свойствам */
  transition: all 0.5s ease-out;


  /* Глобальные значения */
  transition: inherit;
  transition: initial;
  transition: unset;
}

CSS свойство transition-property

CSS свойство transition-property указывает название свойства, значение которого будет отслеживаться для создания эффекта перехода. По умолчанию, изменяться будут все свойства в различных состояниях. Если требуется добавить несколько свойств в переход, они могут быть разделены запятой внутри значения transition-property. Кроме того, может быть использовано ключевое слово all для перехода всех свойств элемента.

.elem {
  transition-property: none | all | свойство, свойство, ...
}

CSS свойство transition-duration

CSS свойство transition-duration задаёт время в секундах или миллисекундах, сколько должна длиться анимация перехода до её завершения. По умолчанию значение равно 0s, это означает, что никакой анимации нет, переход происходит мгновенно.

.elem {
  transition-duration: время , время, ...
}

В качестве значения этого свойства можно использовать основные значения времени, в том числе секунды (s) и миллисекунды (ms). Эти значения времени можно также задавать и дробными, например: .2s. Можно указать несколько значений, перечисляя их через запятую.

Каждое значение применяется к свойствам, заданным через transition-property, при этом важен порядок значений при определении отдельных свойств и длительностей. К примеру, первое свойство, заданное в transition-property, будет совпадать с первым значением в свойстве transition-duration и так далее. Если на несколько свойств в переходе приходится только одно значение длительности, то оно будет задавать длительность для всех свойств перехода.

CSS свойство transition-timing-function

Свойство transition-timing-function используется для задания скорости, с которой будет двигаться переход. Длительность перехода из свойства transition-duration может быть линейной или нелинейной в пределах времени изменения состояния.

Например: значение linear определяет переход, движущийся в постоянной скоростью из одного состояния к другому. Значение ease-in определяет переход, который начинается медленно и ускоряется на протяжении перехода, в то время как значение ease-out определяет переход, который начинает быстро и замедляется в течение всего перехода. Значение ease-in-out определяет переход, который начинается медленно, ускоряется в середине, а затем снова замедляется перед окончанием.

.elem {
  transition-timing-function: ease | ease-in | ease-out |
                              ease-in-out | linear | step-start |
                              step-end | steps | cubic-bezier
}
easeАнимация начинается медленно, затем ускоряется и к концу движения опять замедляется.
Аналогично 
cubic-bezier(0.25,0.1,0.25,1)
ease-inАнимация медленно начинается, к концу ускоряется.
Аналогично 
cubic-bezier(0.42,0,1,1)
ease-outАнимация начинается быстро, к концу замедляется.
Аналогично 
cubic-bezier(0,0,0.58,1)
ease-in-outАнимация начинается и заканчивается медленно.
Аналогично 
cubic-bezier(0.42,0,0.58,1)
linearОдинаковая скорость от начала и до конца.
step-startКак таковой анимации нет. Стилевые свойства сразу же принимают конечное значение.
step-endКак таковой анимации нет. Стилевые свойства находятся в начальном значении заданное время, затем сразу же принимают конечное значение.
stepsСтупенчатая функция, имеющая заданное число шагов. 
.elem {
  animation-timing-function: steps(число, start || end);
}
  • число - целое число больше нуля;
  • start - задаёт полунепрерывную снизу функцию;
  • end - задаёт полунепрерывную сверху функцию.
.elem {
  animation-timing-function: steps(5, end);
}
cubic-bezierЗадаёт функцию движения в виде кривой Безье 
cubic-bezier(n,n,n,n);

CSS свойство transition-delay

Свойство transition-delay устанавливает время ожидания перед запуском эффекта перехода. Значение по умолчанию 0s - запускает анимацию сразу же. Отрицательное значение также включает анимацию без задержек, но может привести к изменению вида начала анимации. Допустимо указывать несколько значений, перечисляя их через запятую. Каждое значение будет применяться к свойству, заданному в параметрах transition-property.

.elem {
  transition-delay: время, время, ...
}

CSS свойство transition-behavior

CSS свойство transition-behavior определяет, будут ли запускаться переходы для свойств, поведение анимации которых дискретно.

.elem {
transition-behavior: allow-discrete | normal;
}
  • allow-discrete - в элементе будут запущены переходы для дискретных анимированных свойств.
  • normal - переходы к элементу для отдельных анимированных свойств запускаться не будут.

Свойство transition-behavior актуально только в том случае, если оно используется в сочетании с другими свойствами перехода, в частности, свойством transition-property и transition-duration, поскольку переход не происходит, если свойства не анимируются в течение ненулевого периода времени.

Для кроссбраузерной поддержки (для отображения анимации в браузерах, которые не поддерживают свойство transition-behavior), свойствро transition нужно указать два раза:

.card {
  transition: all 0.25s;
  transition: all 0.25s allow-discrete;
}

Свойства дискретной анимации обычно переключаются между двумя значениями на 50% при переходе от одного значения к другому. Однако существует исключение, которое возникает при переходе от display: none к content-visibility: hidden. В этом случае браузер будет переключаться между двумя значениями, так что измененный контент будет отображаться в течение всей продолжительности анимации.
