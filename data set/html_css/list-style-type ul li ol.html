list-style

Универсальное свойство list-style позволяет одновременно задать стиль маркера, его положение, а также изображение, которое будет использоваться в качестве маркера списка. В качестве значений принимает любые комбинации трёх значений определяющих стиль маркеров, разделенных пробелом. Комбинации значений должны следовать в указанном порядке: вначале идёт тип маркера, затем положение и картинка. Ни одно значение не является обязательным, поэтому неиспользуемые можно опустить.

li {
  list-style: list-style-type || list-style-position || list-style-image;
}

ul {
  list-style: circle inside
}

ol {
  list-style: lower-roman;
}

list-style-type

Свойство list-style-type изменяет вид маркера для каждого элемента списка. Это свойство используется только в случае, когда значение list-style-image установлено как none. Маркеры различаются для маркированного списка и нумерованного: по умолчанию выводятся с маркерами, которые задаются параметрами disc для тега ul и decimal для тега ol. Значение свойства применяется к элементам li, ol и ul, а также ко всем элементам, у которых указано display:list-item.

li {
  list-style-type: circle | disc | square | armenian | decimal | decimal-leading-zero
                   georgian | lower-alpha | lower-greek | lower-latin | lower-roman
                   upper-alpha | upper-latin | upper-roman | none
}
Маркированный список:
  • circle - маркер в виде кружка.
  • disc - маркер в виде точки.
  • square - маркер в виде квадрата.
  • none - отменяет маркеры для списка.
Нумерованный список:
  1. armenian - Традиционная армянская нумерация.
  2. decimal - Арабские числа (1, 2, 3, 4,...).
  3. decimal-leading-zero - Арабские числа с нулем впереди для цифр меньше десяти (01, 02...).
  4. georgian - Традиционная грузинская нумерация.
  5. lower-alpha - Строчные латинские буквы (a, b, c, d,...).
  6. lower-greek - Строчные греческие буквы (α, β, γ, δ,...).
  7. lower-latin - Это значение аналогично lower-alpha.
  8. lower-roman - Римские числа в нижнем регистре (i, ii, iii, iv, v,...).
  9. upper-alpha - Заглавные латинские буквы (A, B, C, D,...).
  10. upper-latin - Это значение аналогично upper-alpha.
  11. upper-roman - Римские числа в верхнем регистре (I, II, III, IV, V,...).
  12. none - Отменяет маркеры для списка.
  1. trad-chinese-informal - цифры на китайском языке.
  2. 2
  3. 3
  4. 4
Маркеры в виде произвольго символа или слова, можно использовать псевдографику.
  • list-style-type: 'word'
  • list-style-type: '%'
  • list-style-type: '\1F44D'

list-style-position

Свойство list-style-position определяет, как будет размещаться маркер относительно текста - маркер обтекается текстом или вынесен за границу элемента списка.

ul {
  list-style-position: inside | outside;
}
  • inside - маркер является частью текстового блока и отображается в элементе списка. Pазмещает маркер списка в первой строке элемента li и позволяет содержимому обтекать маркер при необходимости.
  • outside - текст выравнивается по левому краю, а маркеры размещаются за пределами текстового блока. Значение по умолчанию.

list-style-image

Свойство list-style-image устанавливает адрес изображения, которое служит в качестве маркера списка. Это свойство наследуется, поэтому для отдельных элементов списка для восстановления маркера используется значение none.

ul {
  list-style-image: none | url( );
}
  • none - Значение по умолчанию. Отменяет изображение в качестве маркера для родительского элемента.
  • url - Относительный или абсолютный путь к графическому файлу. Значение можно указывать в одинарных, двойных кавычках или без них.
Картинка будет выводиться исходного размера! Для обхода этого "ограничения" убирают сами маркеры а картинку помещают через ::before или background элеметна li.

Маркеры в виде картинки

Чаще всего это делается путём размещения фонового изображения для элемента <li>. Процесс включает в себя удаление любого значения свойства list-style-type по умолчанию и добавление фонового изображения и полей для элемента <li>.

Итак:
- значение none свойства list-style-type удалит существующие маркеры списка.
- cвойство background задаст фоновое изображение вместе.
- cвойство padding-left обеспечит пространство слева от текста для фоновой картинки.

  • маркеры списков в виде картинки
  • картинки в виде маркеров
  • оформление списка красивыми маркерами
<ul class="list-marker">
  <li>маркеры списков в виде картинки</li>
  <li>картинки в виде маркеров</li>
  <li>оформление списка красивыми маркерами</li>
</ul>

.list-marker li {
  padding-left: 20px;
  list-style-type: none;
  background-repeat: no-repeat;
  background-image: url(img/arrow.gif);
}

Нумерованный список: список со счетчиком

В нумерованном списке номер каждого пункта выводится с точкой. При этом как-то изменить вид числа, к примеру, убрать точку или поставить скобку, напрямую нельзя. Поэтому вообще убирают исходную нумерацию и выводят свою с помощью псевдоэлемента ::before и свойства content. Чтобы нумерация работала и увеличивалась с каждым пунктом, нужно добавить свойство counter-reset с произвольным именем счётчика для его инициализации и counter-increment для приращения счётчика.

Поскольку мы сами выводим число, то можем легко его стилизовать, изменив цвет, сделав жирным, курсивным и др.

  1. простой список с номерами
  2. нумерованный список
  3. список с нумерацией
<ol>
  <li>простой список с номерами</li>
  <li>нумерованный список</li>
  <li>список с нумерацией</li>
</ol>

ol {
  list-style-type: none;          /* Убираем исходные маркеры */
  counter-reset: counter1;        /* Задаём имя счетчика */
}
li::before {
  content: counter(counter1) ")"; /* Выводим число и скобку */
  counter-increment: counter1;    /* Увеличиваем значение счётчика */
  margin-right: 10px;             /* Расстояние от цифры до текста */
}

Многоуровневый нумерованный список

Многоуровневый список используется для отображения элементов списка на разных уровнях с различными отступами.

Чтобы сделать вложенную нумерацию, нужно использовать следующие свойства:
  • counter-reset сбрасывает один или несколько счётчиков, задавая значение для сброса;
  • counter-increment задаёт значение приращения счётчика, т.е. с каким шагом будет нумероваться каждый последующий пункт;
  • content генерируемое содержимое, в данном случае отвечает за вывод номера перед каждым пунктом списка.
  1. пункт
  2. пункт
    1. пункт
    2. пункт
    3. пункт
      1. пункт
      2. пункт
      3. пункт
    4. пункт
  3. пункт
  4. пункт
<div class="multi-namber"><ol>
  <li>пункт</li> <!--1.-->
  <li>пункт
    <ol>
      <li>пункт</li> <!--2.1.-->
      <li>пункт</li> <!--2.2.-->
      <li>пункт
        <ol>
          <li>пункт</li> <!--2.3.1.-->
          <li>пункт</li> <!--2.3.2.-->
          <li>пункт</li> <!--2.3.3.-->
        </ol>
      </li> <!--2.3.-->
      <li>пункт</li> <!--2.4.-->
    </ol>
  </li> <!--2.-->
  <li>пункт</li> <!--3.-->
  <li>пункт</li> <!--4.-->
</ol></div>

.multi-namber ol {
  list-style: none;
  counter-reset: li;}
.multi-namber li:before {
  counter-increment: li;
  content: counters(li,".") ". ";}
.multi-namber ol, .multi-namber li {
  border: 1px solid #000;
  margin:0px;
  padding-left: 20px;}
