list-style-type

Изменяет вид маркера для каждого элемента списка. Это свойство используется только в случае, когда значение list-style-image установлено как none. Маркеры различаются для маркированного списка (элемент <ul>) и нумерованного (элемент <ol>).

Списки на html страницах по умолчанию выводятся с маркерами, которые задаются параметрами «disc» для тега <ul> и «decimal» для тега <ol>.

Применяется К элементам <li>, <ol> и <ul>, а также ко всем элементам, у которых указано display: list-item

                upper-alpha | upper-latin | upper-roman | none

list-style-position

Определяет, как будет размещаться маркер относительно текста - маркер обтекается текстом или вынесен за границу элемента списка

list-style-position: inside | outside
insideМаркер является частью текстового блока и отображается в элементе списка. Pазмещает маркер списка в первой строке элемента <li> и позволяет содержимому обтекать маркер при необходимости.
outsideТекст выравнивается по левому краю, а маркеры размещаются за пределами текстового блока.
В браузере Internet Explorer 6 при использовании нумерованного списка <ol> и значения inside свойства list-style-position, числа идущие с 10, начинают накладываться на текст списка.

list-style-image

Устанавливает адрес изображения, которое служит в качестве маркера списка. Это свойство наследуется, поэтому для отдельных элементов списка для восстановления маркера используется значение none.

list-style-image: none | url(<адрес>)
noneЗначение по умолчанию. Отменяет изображение в качестве маркера для родительского элемента.
urlОтносительный или абсолютный путь к графическому файлу. Значение можно указывать в одинарных, двойных кавычках или без них.
Картинка будет выводиться исходного размера! Для обхода этого "ограничения" убирают сами маркеры а картинку помещают через ::before или background элеметна <li>.
  • В разных браузерах расстояние между изображением и текстом может различаться.
  • В Internet Explorer до версии 7 включительно не отображаются маркеры, если для списка добавлено свойство float.

Универсальное свойство list-style

Свойства списка list-style-type и list-style-position, могут быть объединены в одно универсальное свойство list-style. В этом свойстве мы можем использовать одно или все значения свойств списка одновременно.

list-style: list-style-type || list-style-position || list-style-image

Любые комбинации трёх значений определяющих стиль маркеров, они разделяются между собой пробелом. Комбинации значений должны следовать в указанном порядке: вначале идёт тип маркера, затем положение и картинка. Ни одно значение не является обязательным, поэтому неиспользуемые можно опустить.

ul {list-style: circle inside}
ol {list-style: lower-roman}

Значения тегов по умолчанию

ul {padding-left: 40px; margin-top: 1em; margin-bottom: 1em;}
ol {padding-left: 40px; margin-top: 1em; margin-bottom: 1em;}
dl {margin-top: 1em; margin-bottom: 1em;}

Атрибуты тега <ol>

reversed

задает отображение списка в обратном порядке (например, 9, 8, 7…).

Атрибут reversed является логическим атрибутом, он может быть истиной, либо ложью. Ложь является значением по умолчанию, значение становится истинным, когда атрибут reversed появляется у элемента <ol>.

<ol reversed>
start

задает начальное значение, от которого пойдет отсчет нумерации.

<ol start="10">
первому пункту присвоит порядковый номер «10».
type

задает вид маркера для использования в списке (в виде букв или цифр).
принимаемые значения:

1 — значение по умолчанию, десятичная нумерация
A — нумерация списка заглавными буквами (A, B, C, D).
a — нумерация списка строчными буквами (a, b, c, d).
I — нумерация римскими заглавными цифрами (I, II, III, IV).
i — нумерация римскими строчными цифрами (i, ii, iii, iv).
<ol type="I">
value

атрибут value может применяться к отдельным элементам <li> в нумерованном списке, чтобы изменить его значение в списке. Номер любого пункта списка, который появляется ниже пункта списка с атрибутом value, будет пересчитан соответственно.

В качестве примера, если у второго пункта списка атрибут value задан как 9, номер этого пункта списка будет выводиться, как если бы он был девятым. Все последующие пункты списка будут пронумерованы от 9.

<ol>
  <li>Пункт №1</li>
  <li value="9">Пункт №2</li>
  <li>Пункт №3</li>
</ol>
  1. Пункт №1
  2. Пункт №2
  3. Пункт №3

Нумерованный список: список со счетчиком

В нумерованном списке номер каждого пункта выводится с точкой. При этом как-то изменить вид числа, к примеру, убрать точку или поставить скобку, напрямую нельзя. Поэтому вообще убирают исходную нумерацию и выводят свою с помощью псевдоэлемента ::before и свойства content. Чтобы нумерация работала и увеличивалась с каждым пунктом, нужно добавить свойство counter-reset с произвольным именем счётчика для его инициализации и counter-increment для приращения счётчика.

Поскольку мы сами выводим число, то можем легко его стилизовать, изменив цвет, сделав жирным, курсивным и др.

  1. простой список с номерами
  2. нумерованный список
  3. список с нумерацией
HTML 
<ol>
  <li>простой список с номерами</li>
  <li>нумерованный список</li>
  <li>список с нумерацией</li>
</ol>

CSS 
ol {
  list-style-type: none;          /* Убираем исходные маркеры */
  counter-reset: counter1;        /* Задаём имя счетчика */
}
li::before {
  content: counter(counter1) ")"; /* Выводим число и скобку */
  counter-increment: counter1;    /* Увеличиваем значение счётчика */
  margin-right: 10px;             /* Расстояние от цифры до текста */
}

Многоуровневый нумерованный список

Многоуровневый список используется для отображения элементов списка на разных уровнях с различными отступами.

Чтобы сделать вложенную нумерацию, нужно использовать следующие свойства:
  • counter-reset сбрасывает один или несколько счётчиков, задавая значение для сброса;
  • counter-increment задаёт значение приращения счётчика, т.е. с каким шагом будет нумероваться каждый последующий пункт;
  • content генерируемое содержимое, в данном случае отвечает за вывод номера перед каждым пунктом списка.
  1. пункт
  2. пункт
    1. пункт
    2. пункт
    3. пункт
      1. пункт
      2. пункт
      3. пункт
    4. пункт
  3. пункт
  4. пункт
HTML 
<div class="multi-namber"><ol>
  <li>пункт</li> <!--1.-->
  <li>пункт
    <ol>
      <li>пункт</li> <!--2.1.-->
      <li>пункт</li> <!--2.2.-->
      <li>пункт
        <ol>
          <li>пункт</li> <!--2.3.1.-->
          <li>пункт</li> <!--2.3.2.-->
          <li>пункт</li> <!--2.3.3.-->
        </ol>
      </li> <!--2.3.-->
      <li>пункт</li> <!--2.4.-->
    </ol>
  </li> <!--2.-->
  <li>пункт</li> <!--3.-->
  <li>пункт</li> <!--4.-->
</ol></div>

CSS 
.multi-namber ol {
  list-style: none;
  counter-reset: li;}
.multi-namber li:before {
  counter-increment: li;
  content: counters(li,".") ". ";}
.multi-namber ol, .multi-namber li {
  border: 1px solid #000;
  margin:0px;
  padding-left: 20px;}
