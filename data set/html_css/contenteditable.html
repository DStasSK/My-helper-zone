contenteditable

Свойство contenteditable сообщает, что элемент доступен для редактирования пользователем — можно удалять текст и вводить новый. Также работают стандартные команды вроде отмены, вставки текста из буфера и др. По умолчанию наследует значение родителя.

Внесенные корректировки текста не сохраняются в коде страницы и после её обновления они исчезнут.
contenteditable: true | false
<p contenteditable="true">текст</p>
true

Включает режим редактирования. Вместо true допустимо указывать пустое значение contenteditable="".

false Запрещает редактирование элемента - действует по умолчанию.

Применяется к тегам:

<a>, <abbr>, <address>, <area>, <b>, <basefont>, <bdo>, <blockquote>, <body>, <button>, <caption>, <cite>, <code>, <col>, <colgroup>, <dd>, <del>, <dfn>, <dir>, <div>, <dl>, <dt>, <em>, <fieldset>, <form>, <h1>, <h2>, <h3>, <h4>, <h5>, <h6>, <i>, <iframe>, <input>, <ins>, <kbd>, <label>, <legend>, <li>, <menu>, <ol>, <option>, <p>, <pre>, <q>, <samp>, <select>, <span>, <strong>, <sub>, <sup>, <table>, <tbody>, <td>, <textarea>, <tfoot>, <th>, <thead>, <tr>, <ul>, <var>
