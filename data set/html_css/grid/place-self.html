CSS свойство place-self

Свойство place-self — производит выравнивание элемента макета сетки внутри ячейки по оси столбца и по оси строки grid-контенера. Является сокращенным свойством для свойств align-self и justify-self.

.item {
   place-self: align-self | justify-self;
}

/*   align-self: auto | stretch | center | start | end | baseline | initial | inherit; */
/* justify-self: auto | stretch | center | start | end | baseline | initial | inherit; */
Допускается указывать одно значение, при этом значение используется для обоих свойств.
Eсли для элемента установлено свойство margin со значением auto, то свойство place-self будет игнорировано.
