CSS свойство grid-row-gap

Свойство grid-row-gap — определяет размер промежутка между строками в grid макете. Значение по умолчанию - 0, не наследуется, анимируется.

Это свойство было переименовано в CSS3 row-gap
div {
   display: grid;
   grid-row-gap: length;
}
length - любое допустимое линейное значение CSS, например, px или %. Значение по умолчанию 0.
