CSS свойство grid-gap

Свойство grid-gap — это короткая запись значений свойств grid-row-gap и grid-column-gap, определяющее размер промежутка между строками и столбцами в grid макете. Значение по умолчанию - 0 0, не наследуется, анимируется.

Это свойство было переименовано в CSS3 gap
div {
   display: grid;
   grid-gap: grid-row-gap grid-column-gap;
}
  • grid-row-gap - размер промежутка между строками в макете.
  • grid-column-gap - размер промежутка между столбцами в макете.
div {
   display: grid;
   grid-gap: 20px  40px;
}

div {
   display: grid;
   grid-gap: 40px;  /* grid-row-gap == grid-column-gap == 40px  */
}
