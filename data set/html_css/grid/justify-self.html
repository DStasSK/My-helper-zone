justify-self

CSS свойство justify-self производит выравнивание элемента макета сетки внутри ячейки по оси строки grid-контенера. Свойство justify-self переопределяет значение выравнивания, заданного для grid-контейнера свойством justify-items конкретному элементу.

.el {justify-self: auto | stretch | center | start | end | baseline | initial | inherit }
Eсли для элемента установлено свойство margin (внешние отступы) со значением auto, то свойство justify-self будет игнорировано.
Для того, чтобы произвести выравнивание элемента макета сетки внутри ячейки по оси столбца grid-контенера вы можете воспользоваться свойством align-self
