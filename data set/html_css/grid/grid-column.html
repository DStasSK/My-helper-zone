CSS свойство grid-column

Свойство grid-column — определяет с какого столбца будет начинаться элемент, сколько столбцов будет занимать элемент, или на каком столбце завершится элемент. Является сокращенным свойством для свойств grid-column-start и grid-column-end.

.item {
   grid-column: grid-column-start / grid-column-end;
}
Допускается указывать одно значение, при этом будет задаваться тольно grid-column-start.
.item {
   grid-column: auto / auto;  /* значения по умолчанию */
   grid-column: 1 / 5;        /* задает блок от первой до пятой вертикальной линии */
   grid-column: 1 / span 2;   /* охватывает два столбца */
   grid-column: 2;            /* начинается со второй вертикальной линии */
}
