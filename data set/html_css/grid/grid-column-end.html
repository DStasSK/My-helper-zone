CSS свойство grid-column-end

Свойство grid-column-end — определяет, на какой вертикальной линии будет закачинаться элемент или какое количество столбцов он будет охватывать.

Для того, чтобы определить с какой вертикальной линии начинается элемент можно воспользоваться свойством grid-column-start.

.item {
   grid-column-end: auto | line | line-name | span line | initial | inherit;
}
Значения:
autoЗначение по умолчанию - 1. Элемент будет размещен в соответствии с потоком.
lineЦелое число, которое соответствует конечной грани элемента в макете сетки (отсчет ведется слева направо от левого края элемента). Если задано отрицательное целое число, то отсчет ведется в обратном порядке, начиная с конечного края явной сетки макета.
Значение 0 недопустимо.
line-nameСтроковое значение ссылающееся на именованный столбец в макете сетки. Элемент располагается до начальной грани указанного элемента.
span lineКлючевое слово span с целым числом, определяющее количество столбцов, которое элемент будет охватывать. Если целое число опущено, то по умолчанию используется значение 1.
Отрицательные значения или значение 0 недопустимы.
initialУстанавливает свойство в значение по умолчанию.
inheritЗначение наследуется от родительского элемента.
