CSS свойство grid-area

Свойство grid-area — определяет размер и расположение элемента сетки в макете сетки, а также может использоваться для присвоения имени элементу сетки.

Является сокращенным свойством для следующих свойств:
  • grid-row-start
  • grid-column-start
  • grid-row-end
  • grid-column-end
Значение всех свойств по умолчанию - auto.
.item {
 grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end | item_name;
}
grid-row-start Определяет сколько строк будет занимать элемент, или на какой строке начинается элемент в макете сетки.
grid-column-start Определяет сколько строк будет занимать элемент, или на какой строке заканчивается элемент в макете сетки.
grid-row-end Определяет с какого столбца будет расположен элемент в макете сетки, или какое количество столбцов будет охватывать элемент.
grid-column-end Определяет сколько столбцов будет занимать элемент, или на какой строке столбца завершится элемент.
item_name Задает имя элемента сетки. Нельзя использовать с grid-... свойствами.
Возможные значения grid-... свойств:
  • auto - ключевое слово, указывающее, что свойство не влияет на размещение элемента в макете сетки (автоматическое размещение, автоматический диапазон или диапазон по умолчанию равный 1). Является значением по умолчанию.
  • целое число, которое соответствует начальной грани элемента в макете сетки (отсчет граней в макете ведется сверху вниз от верхнего края элемента). Если задано отрицательное целое число, то отсчет ведется в обратном порядке, начиная с конечного края явной сетки макета. Значение 0 недопустимо.
  • line-name - строковое значение ссылающееся на именованную строку в макете сетки, заданный свойством grid-template. Элемент располагается от начальной грани указанного элемента.
  • span n - ключевое слово с целым числом n, которое определяет какое количество строк сетки элемент будет охватывать. По умолчанию используется значение 1. Отрицательные значения или значение 0 недопустимы.

.item-b { grid-area: 2 / 2 / 4 / 4; }
.item-c { grid-area: 4 / span 4 / 5; }
.item-d { grid-area: 2 / 1 / span 2 / 2; }
