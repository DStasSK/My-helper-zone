CSS свойство grid-template-columns

Свойство grid-template-columns — определяет количество, наименование и размер столбцов (дорожек) в макете сетки. Значения представляют собой список, разделенный пробелами, где каждое значение определяет ширину столбца.

Для того, чтобы задать количество, наименование и размер строк в макете сетки вы можете воспользоваться свойством grid-template-rows.

.item {
   grid-template-columns: none | length | flex | min-content |
                          max-content | auto | minmax() |
                          repeat() | initial | inherit;
}
noneЗначение по умолчанию. Размер не задан. Столбцы создаются при необходимости, размер столбцов будет определяться с помощью свойства grid-auto-columns.
lengthЗадает размер столбцов, используя допустимое значение длины, или значение, указанное в процентах. Значение должно быть положительным.
flexУстанавливает соотношение распределения свободного пространства между столбцами. 
grid-template-columns: 3fr 1fr;

Когда доступное пространство бесконечно (ширина или высота контейнера сетки макета неопределенны), размер столбцов сетки гибкого размера определяется их содержимым с сохранением соответствующих пропорций.

Значения fr не являются значениями length, по этой причине они не могут быть представлены или объединены с другими типами единиц в выражениях типа calc().
max-contentКлючевое слово, которое задает размер каждого столбца в зависимости от самого большого элемента в столбце.
min-contentКлючевое слово, которое задает размер каждого столбца в зависимости от самого маленького элемента в столбце.
autoРазмер столбцов определяется размером контейнера и размером содержимого элементов в столбце.
minmax(min,max)Функциональная нотация, определяющая диапазон размеров больше или равный min и меньше или равный max. Если max < min, то max игнорируется, а функция обрабатывается как min; подробнее
repeat()Представляет из себя повторяющийся фрагмент списка столбцов (дорожек), позволяя записать в более компактной форме большое количество столбцов с повторяющимся шаблоном; подробнее.
initialУстанавливает свойство в значение по умолчанию.
inheritЗначение наследуется от родительского элемента.
.item {
  grid-template-columns: 50px 50px;
  grid-template-columns: 50%;
  grid-template-columns: 25% 25% 25% 25%;

  grid-template-columns: [line1] 40px [line2] 30px;       /* с указанием имени столбца */
  grid-template-columns: [line1] 40px [line2 line3] 30px; /* сначала имя - потом размер */

  grid-template-columns: 3fr 1fr;

  grid-template-columns: auto;
  grid-template-columns: none;
  grid-template-columns: min-content;
  grid-template-columns: max-content;

  grid-template-columns: minmax(100px, 1fr);
  grid-template-columns: repeat(4, 100px);    /* 4 столбца по 100рх */
}
