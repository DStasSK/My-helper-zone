justify-items

CSS свойство justify-items производит выравнивание всех элементов макета сетки по оси строки grid-контенера. Не анимируется, не наследуется.

Для того, чтобы произвести выравнивание всех элементов макета сетки по оси столбца grid-контейнера, вы можете воспользоваться свойством align-items
.el {justify-items: stretch | center | start | end | baseline | initial | inherit }
