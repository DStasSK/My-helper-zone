CSS свойство grid-row

Свойство grid-row — определяет с какой строки в макете сетки будет начинаться элемент, сколько строк будет занимать элемент, или на какой строке завершится элемент в макете сетки. Является сокращенным свойством для свойств grid-row-start и grid-row-end.

.item {
   grid-row: grid-row-start / grid-row-end;
}
Допускается указывать одно значение, при этом будет задаваться тольно grid-row-start.
.item {
   grid-row: auto / auto;  /* значения по умолчанию */
   grid-row: 1 / 5;        /* задает блок от первой до пятой горизонтальной линии */
   grid-row: 1 / span 2;   /* охватывает две строки */
   grid-row: 2;            /* начинается со второй горизонтальной линии */
}
