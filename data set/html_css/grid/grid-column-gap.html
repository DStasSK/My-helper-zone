CSS свойство grid-column-gap

Свойство grid-column-gap — определяет размер промежутка между столбцами в grid макете. Значение по умолчанию - 0, не наследуется, анимируется.

Это свойство было переименовано в CSS3 column-gap
div {
   display: grid;
   grid-column-gap: length;
}
length - любое допустимое линейное значение css, например, px или %. Значение по умолчанию 0.
