Функция repeat()

Функция CSS repeat() представляет собой повторяющийся фрагмент списка дорожек, что позволяет записать в более компактной форме большое количество столбцов или строк, имеющих повторяющийся шаблон. Принимает два аргумента разделенных запятой. Используется в CSS Grids в свойствах grid-template-columns и grid-template-rows.

repeat( count , list )
  • count - указывает количество повторов списка дорожек от 1 и более; или ключевыми словами auto-fill или auto-fit. Эти ключевые слова повторяют набор дорожек столько раз, сколько необходимо для заполнения контейнера сетки.
    • auto-fill - количество повторений определяется как наибольшее возможное положительное целое число, которое не приведет к переполнению контейнера сетки с учетом gap. Заполняет строку любым количеством столбцов, создавая неявные столбцы всякий раз, когда новый столбец может поместиться. Добавленные столбцы могут быть пустыми, но они все равно будут занимать определенное место в строке. Минимальное значение повторений равно 1.
    • auto-fit - ведет себя так же, как auto-fill за исключением того, что после размещения элементов сетки все пустые повторяющиеся дорожки объединяются. Пустая дорожка — это дорожка, в которую не помещены или не охватывают ее элементы сетки потока. Это может привести к объединению всех дорожек, если они все пустые.
  • list - указывает набор дорожек, которые будут повторяться. Cостоит из одного или нескольких значений, где каждое значение представляет размер этой дорожки. Значение задаётся:
    • в процентах или длине css
    • flex-параметрах (fr)
    • с помощью функции minmax()
    • с помощью сключевых слов - min-content, max-content или auto
    • одного или нескольких имен строк до или после каждой дорожки разделенных пробелом и заключенных в квадратные скобки
    • с помощью функции fit-content(), передающая значение длины в процентах
.container {
  display: grid;
  grid-template-columns:
    repeat(4, 1fr);
    repeat(4, 250px);
    repeat(4, [col-start] 250px [col-end]);
    repeat(4, [col-start] min-content [col-end]);
    repeat(4, [col-start] auto [col-end]);
    repeat(4, [col-start] minmax(100px, 1fr) [col-end]);
    repeat(4, [col-start] fit-content(200px) [col-end]);
    repeat(4, 10px [col-start] 30% [col-middle] auto [col-end]);
    repeat(auto-fill, 250px);
    repeat(auto-fit, 250px);
    repeat(auto-fill, 10px [col-start] 30% 400px [col-end]);
}
