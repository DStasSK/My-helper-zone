Функция minmax()

Функция CSS minmax() определяет диапазон размеров, который больше или равен min и меньше или равен max. Используется в CSS Grids. Функция принимает два параметра, min и max соответственно. Отрицательные значения не допустимы. Если max < min, то max игнорируется, а функция обрабатывается как min.

minmax( min , max )

.container {
  display: grid;
  grid-template-columns: minmax(max-content, 300px) minmax(200px, 1fr) 150px;
}
Используется в CSS Grids в свойствах:
  • grid-template-columns
  • grid-template-rows
  • grid-auto-columns
  • grid-auto-rows
