CSS свойство grid-template-rows

Свойство grid-template-rows — определяет количество, наименование и размер строк в макете сетки. Значения представляют собой список, разделенный пробелами, где каждое значение определяет высоту соответствующей строки.

Для того, чтобы задать количество, наименование и размер столбцов (дорожек) в макете можно воспользоваться свойством grid-template-columns.

.item {
   grid-template-rows: none | length | flex | min-content |
                       max-content | auto | minmax() |
                       repeat() | initial | inherit;
}
Значения:
noneЗначение по умолчанию. Размер не задан. При необходимости создаются строки, размер строк будет определяться с помощью свойства grid-auto-rows.
lengthЗадает размер строк, используя допустимое значение длины, или значение, указанное в процентах. Значение должно быть положительным.
flexУстанавливает соотношение распределения свободного пространства между строками. 
grid-template-rows: 3fr 1fr;
Значения fr не являются значениями length, по этой причине они не могут быть представлены или объединены с другими типами единиц в выражениях типа calc().
max-contentКлючевое слово, которое задает размер каждой строки в зависимости от самого большого элемента в столбце.
min-contentКлючевое слово, которое задает размер каждой строки в зависимости от самого маленького элемента в столбце.
autoРазмер строк определяется размером контейнера и размером содержимого элементов в строке.
minmax(min,max)Функциональная нотация, определяющая диапазон размеров больше или равный min и меньше или равный max. Если max < min, то max игнорируется, а функция обрабатывается как min.
repeat()Представляет из себя повторяющийся фрагмент списка строк, позволяя записать в более компактной форме большое количество строк с повторяющимся шаблоном; подробнее.
initialУстанавливает свойство в значение по умолчанию.
inheritЗначение наследуется от родительского элемента.
.item{
   grid-template-rows: 50px 50px;
   grid-template-rows: 50%;
   grid-template-rows: 25% 25% 25% 25%;

   grid-template-rows: [line1] 40px [line2] 30px;       /* с указанием имени строки */
   grid-template-rows: [line1] 40px [line2 line3] 30px; /* сначала имя - потом размер */

   grid-template-rows: 3fr 1fr;

   grid-template-rows: auto;
   grid-template-rows: none;
   grid-template-rows: min-content;
   grid-template-rows: max-content;

   grid-template-rows: minmax(100px, 1fr);
   grid-template-rows: repeat(4, 100px);    /* 4 строки по 100рх высотой */
}

grid-template-rows: repeat()

Значение repeat свойства grid-temlate-rows представляет из себя повторяющийся фрагмент списка строк, позволяя записать в более компактной форме большое количество строк с повторяющимся шаблоном.

repeat( [positive-integer | auto-fill | auto-fit] , track-list )
Возможные значения:
  • positive-integer - положительное целое число, которое определяет количество повторений.
  • auto-fill - если контейнер сетки имеет определенный или максимальный размер на соответствующей оси, то число повторений является наибольшим возможным положительным целым числом, которое не вызывает переполнения сетки. Если любое количество повторений будет вызывать переполнение контейнера, то количество повторений будет равно 1.
    В противном случае, если контейнер сетки имеет определенный минимальный размер на соответствующей оси, число повторений является наименьшим возможным положительным целым числом, удовлетворяющим этому минимальному требованию. В противном случае указанный список дорожек повторяется только один раз.
    Другими словами auto-fill заполняетмакет любым количеством строк. Таким образом, он создает неявные строки всякий раз, когда новая строка может поместиться, потому что он пытается заполнить макет как можно большим количеством строк. Добавленные строки могут быть пустыми, но они все равно будут занимать определенное место в макете.
  • auto-fit - ведет себя так же, как auto-fill, за исключением того, что после размещения элементов сетки пустые повторяющиеся дорожки объединяются (это может привести к тому, что все дорожки будут объеденены, если все они пусты). Другими словами auto-fit помещает доступные в данный момент строки в пространство, расширяя их так, чтобы они занимали любое доступное пространство. Браузер делает это после заполнения этого дополнительного пространства дополнительными строками (как при автоматическом заполнении), а затем сворачивает пустые.
