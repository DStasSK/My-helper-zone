input (от англ. input — ввод) является одним из разносторонних элементов формы и позволяет создавать разные части интерфейса и обеспечивать взаимодействие с пользователем. Главным образом input предназначен для создания текстовых полей, различных кнопок, переключателей и флажков.

Основной атрибут input, определяющий вид элемента - type. Он позволяет задавать следующие элементы формы: текстовое поле (text), поле с паролем (password), переключатель (radio), флажок (checkbox), скрытое поле (hidden), кнопка (button), кнопка для отправки формы (submit), кнопка для очистки формы (reset), поле для отправки файла (file), кнопка с изображением (image) и др.

Для каждого элемента существует свой список атрибутов, которые определяют его вид и характеристики. Кроме того, в HTML5 добавлено ещё более десятка новых элементов. Закрывающий тег - не требуется.

alt

Устанавливает альтернативный текст для поля с изображением. Такой текст позволяет получить текстовую информацию о рисунке при отключенной в браузере загрузке изображений, а также предназначен для поисковых систем.

Атрибут alt может добавляться только к элементу у которого значение атрибута type установлено image. 
<input type="image" alt="текст">
Значения - любая подходящая текстовая строка.
autocomplete Включает или отключает автозаполнение.
подробнее...
autofocus

Автоматически устанавливает фокус в поле формы. В таком поле можно сразу набирать текст без явного щелчка по нему курсором мыши (например - для поисковых систем). По умолчанию атрибут autofocus не установлен.

<input autofocus>
подробнее...
border

Браузеры обрабатывают изображения, добавленные через элемент input, подобно рисункам, созданным с помощью элемента img. В том числе вокруг изображения можно добавить рамку, цвет которой совпадает с цветом текста.

<input type="image" border="число">

dirname

При наличии атрибута dirname на сервер отправляется пара имя/значение, где имя задаётся атрибутом dirname, а значение подставляется браузером. Для текста, идущего слева направо, значение будет ltr, а для текста справа налево (арабского, к примеру) значение будет rtl.

В качестве значения используется произвольная текстовая строка, которая служит именем параметра. Чтобы показать, что этот параметр указывает направление текста, обычно в конце строки добавляют .dir.

Атрибут dirname может добавляться к полю формы, у которого атрибут type имеет одно из значений: hidden, text, search, tel, url, email, password, submit, reset, button.

<form action="handler.php">
  <label>Комментарий: 
    <input name="comment" dirname="comment.dir" required>
  </label>
  <p><button type="submit">Отправить</button></p>
</form>

disabled Блокирует доступ и изменение текстового поля. Оно в таком случае отображается серым и недоступным для активации пользователем. Кроме того, такое поле не может получить фокус путем нажатия на клавишу Tab, мышью или другим способом. Тем не менее, такое состояние поля можно менять с помощью скриптов. Заблокированный в поле текст не передаётся на сервер. 
<input disabled>
подробнее...
form

Связывает поле с формой по её идентификатору. Такая связь необходима в случае, когда поле располагается за пределами form, например, при создании её программно или по соображениям дизайна. В качестве значения выступает идентификатор формы (значение атрибута id элемента form). Если атрибут не задан, то элемент input будет связан с ближайшей формой-родителем, если такая есть.

<form id="data" action="handler.php"></form>
<input placeholder="Ваше имя" name="user" form="data">
<input type="submit" value="Отправить форму" form="data">
Элемент input может быть связан только с одной формой.
formaction

Определяет адрес обработчика формы — это программа, которая получает данные формы и производит с ними желаемые действия. Атрибут formaction по своему действию аналогичен атрибуту action элемента form. Если указаны оба атрибута, то атрибут action имеет преимущество.

<form>
  <input placeholder="Ваше имя" name="user"><br>
  <input type="submit" formaction="handler.php" value="Отправить">
</form>

Указывает на список вариантов, созданный с помощью элемента datalist, которые можно выбирать при наборе текста. Изначально этот список скрыт и становится доступным при получении полем фокуса.

Этот атрибут игнорируется, когда атрибут type принимает значения hidden, checkbox, radio, file, или элемент input используется в качестве кнопки. 
<input list="идентификатор">

  <option value="Текст1">

maxlength

Устанавливает максимальное число символов, которое может быть введено пользователем в текстовом поле. Когда это количество достигается при наборе, дальнейший ввод становится невозможным.

<input type="text" maxlength="число">
<input type="password" maxlength="число">

<input type="number" min="число">
<input type="range"  min="число">
<input type="date"   min="дата">
Значения:
  • дата в формате ГГГГ-ММ-ДД (например: 2012-12-22) для type="date".
  • целое положительное или отрицательное число (для type="number", type="range").
подробнее...
minlength

Задаёт минимальное число символов, которое может быть введено пользователем в текстовом поле. Если число символов меньше указанного значения, то браузер выдаёт сообщение об ошибке и не отправляет форму.

<input type="text" minlength="число">
<input type="password" minlength="число">
Введите пароль
  
<form action="handler.php">

  <br><input type="password" size="30" minlength="6" required>

multiple

Позволяет указывать одновременно несколько файлов в поле для загрузки файлов, а также несколько адресов электронной почты. При использовании двух и более почтовых адресов они должны перечисляться через запятую.

<input type="file" multiple>
<input type="email" multiple>
name

Определяет уникальное имя элемента формы. Как правило, это имя используется при отправке данных на сервер или для доступа к введённым данным поля через скрипты. В качестве имени используется набор символов, включая числа и буквы (чувствителен к регистру).

Введите текст 
<form>
  <script>
    function dataField(f) {alert("Вы ввели: " + f.comment.value);}
  </script>
  Введите текст
  <input type="text" name="comment">
  <input type="button" value="ОК" onClick="dataField(this.form)">
</form>
Атрибут name также определяет в одну группу ряд переключателей или флажков. 
<input type="radio" name="">
<input type="checkbox" name="">

readonly

Делает поле формы доступным только для чтения. Такое поле не может изменяться пользователем, в том числе вводиться новый текст или модифицироваться существующий. Тем не менее, состояние и содержимое поля можно менять с помощью скриптов и данные отправляются на сервер.

Этот атрибут игнорируется, когда атрибут type принимает значения hidden, range, color, checkbox, radio, file, или элемент input используется в качестве кнопки.

<input type="text" readonly>
<input type="password" readonly>
required

Этот атрибут указывает, что пользователь должен заполнить значение перед отправкой формы. Если обязательное поле пустое, браузер выведет сообщение об ошибке, а форма отправлена не будет. Вид и содержание сообщения зависит от браузера и меняться пользователем не может.

Его нельзя использовать, если атрибут type имеет значениеhidden, image или элемент input используется в качестве кнопки.

Псевдоклассы CSS :optional и :required будут применены к полю соответствующим образом. 
<input required>
А можно как-то повлиять на цвет подсветки рамки, ту что включает сам браузер (в хроме желтый, в firefox красный) когда поле не заполнено? 
input:invalid {box-shadow: none;}
size

Ширина текстового поля, которое определяется числом символов моноширинного шрифта. Иными словами, ширина задаётся количеством близстоящих букв одинаковой ширины по горизонтали. Если размер шрифта изменяется с помощью стилей, ширина также соответственно меняется. Значение по умолчанию - 20 символов.

Значение атрибута также может указывается в пикселях, если только значение атрибута type не равно text или password, в этом случае это целое число символов.

Aтрибут size может добавляться только к полю формы, у которого type имеет одно из значений: text, password, search, url, tel, email.

<input type="text" size="число">
<input type="password" size="число">
src

Адрес графического файла, который будет отображаться на веб-странице в поле с изображением. В качестве значения принимается полный или относительный путь к файлу.

<input type="image" src="адрес">
Это не поле ввода, а что-то типа графической кнопки для отправки формы на сервер, соответственно, подходит в ограниченных случаях.
аналогично: 
<input type="submit" value="Отправить">
step

Устанавливает шаг изменения числа для ползунков и полей ввода чисел. Значение - любое целое или дробное число; по умолчанию 1. Работает с атрибутами min и max для ограничения приращений, с которыми может быть установлено числовое значение или значение даты-времени.

Атрибут может добавляться только к полю формы, у которого type имеет одно из значений: date, month, week, time, datetime-local, number, range.

Введите произвольное значение от 0 до 1: 
<form action="">
  <input type="range" min="0" max="10" step="1">
  <br>Введите произвольное значение от 0 до 1:
  <input type="number" value="0" min="0" max="1" step="0.01">
</form>

<input type="image" width="значение">

Прячем текст-подсказку при фокусе поля ввода

  .ta {resize: none;}
</style>

  <input type="text" placeholder="Логин">
  <input type="password" placeholder="Пароль">
</form>

<textarea class="ta" rows="4" cols="43" placeholder="resize: none;"></textarea>
