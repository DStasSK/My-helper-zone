CSS свойство hyphens

Свойство hyphens cообщает браузеру, как расставлять переносы слов в блоке текста. Словарь переносов хранится в браузере и подключается при наличии атрибута lang с кодом языка. Так, для русского языка следует добавить lang="ru" к элементу <html> или непосредственно к абзацу текста. Правила переносов определяются браузером и могут различаться.

Перенос слов работает в основном в новых версиях браузеров и не для вех языков. Подробнее можно узнать тут.
html {
  hyphens: none | manual | auto
}
  • none – слова не переносятся, даже при наличии в тексте мягких переносов &shy;. Строки будут переноситься только по пробелам.
  • manual – значение по умолчанию. Переносы слов в тексте происходят в тех местах, где добавлен &shy; или <wbr>, если их нет, то переносы не делаются.
    При использовании <wbr> знак переноса не добавляется.
  • auto – браузер автоматически добавляет переносы слов на основе встроенного словаря переносов и в указанных точках переноса.
  • Internet Explorer и Edge поддерживают свойство -ms-hyphens.
  • Safari поддерживает свойство -webkit-hyphens.
  • Firefox до версии 43 поддерживает свойство -moz-hyphens.

CSS свойство hyphenate-character

Свойство hyphenate-character устанавливает символ или строку, которая добавляется в тексте при переносе слов на другую строку и показывает место переноса. По умолчанию для этой цели используется символ дефиса.

html {
  hyphenate-character: auto| строка
}
  • auto - устанавливает символ переноса на основе текущего языка.
  • строка - cимвол или строка, отображаемая в месте переноса.
Некоторые браузеры поддерживают только свойство -webkit-hyphenate-character.

CSS свойство hyphenate-limit-chars

Свойство hyphenate-limit-chars задаёт минимальную длину слова, при которой разрешён перенос на другую строку, а также минимальное количество букв до и после переноса.

html {
  hyphenate-limit-chars: auto| 1 число | 2 числа | 3 числа
}
  • auto - значение по умолчанию (аналогично 5 2 2).
  • 1 число - задает минимальную длину слова для переноса на другую строку. К примеру, если указать число 6, то слова, в которых содержится шесть и менее букв, переноситься не будут.
  • 2 числа - первое значение задает минимальную длину слова для переноса на другую строку. Второе значение - минимальное число символов перед и после дефиса. К примеру, значение 6 3 означает, что минимальная длина слова должна составлять 6 символов, три буквы минимум должны остаться на одной строке и три буквы на следующей строке.
  • 3 числа - первое значение задает минимальную длину слова для переноса на другую строку. Второе значение - минимальное число символов перед дефисом. Третье значение - минимальное число символов после дефиса.
