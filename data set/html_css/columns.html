Columns - колонки

Columns - универсальное свойство, которое позволяет одновременно задать ширину и количество колонок многоколоночного текста.

Браузеры, которые поддерживают данные свойства: Firefox 2+, Chrome 4+, Safari 3.1+ и Opera 11.1.

Свойство columns позволяет разработчикам разбить текст на странице в несколько колонок. К примеру мы хотим разбить текст в блоке <div> с классом content в две колонки, для этого в CSS необходимо написать:

.content {
  -webkit-column-count: 2;
  -moz-column-count: 2;
  columns: 2;
  -webkit-column-gap: 40px;
  -moz-column-gap: 40px;
  column-gap: 40px;
  -moz-column-rule: 1px dotted #ccc;
  -webkit-column-rule: 1px dotted #ccc;
  column-rule: 1px dotted #ccc;
  -moz-column-width: auto;
  -webkit-column-width: auto;
  column-width: auto;}

Атрибуты

column-count Определяет количество колонок в многоколоночном тексте. Применяется к блочным элементам (кроме таблиц), ячейкам и элементам, у которых display установлен как inline-block. При значении auto, число колонок вычисляется автоматически на основе других свойств (column-width, column-gap). 
column-count: число | auto;
column-fill Определяет, как содержимое должно распределяться внутри колонок. 
column-fill: auto | balance | balance-all;
auto - Колонки заполняются последовательно.
balance - Содержимое равномерно распределяется между колонок. При печати это правило относится только к последней странице.
balance-all - Содержимое равномерно распределяется между колонок. При печати это правило относится ко всем страницам.
column-fill практически не поддерживается брайзерами.
Safari до версии 9 поддерживает свойство -webkit-column-fill.
Firefox до версии 52 поддерживает свойство -moz-column-fill.
column-gap Задаёт расстояние между колонками в многоколоночном тексте. 
column-gap: размер | normal

normal - Межколоночный интервал устанавливается браузером.

Firefox до версии 52 поддерживает свойство -moz-column-gap.
Safari до версии 10, Chrome до версии 50, Opera до версии 37 и Аndroid до версии 80 поддерживают свойство -webkit-column-gap.
column-rule В многоколоночном тексте отрисовывает линию между колонок и определяет её параметры. 
column-rule: column-rule-width || column-rule-style ||
             column-rule-color
column-rule-style - тип линии, аналогичен значениям border.
Chrome до версии 50 b Safari до версии 9 поддерживают свойство -webkit-column-rule.
Firefox до версии 52 поддерживает свойство -moz-column-rule.
column-span Определяет, как должен отображаться элемент в многоколоночном тексте - занимать ширину всех колонок или только одну из них. Это свойство обычно применяют для заголовков текста или изображений, которые не должны разбиваться на колонки. 
column-span: none | all
none - элемент наряду с обычным содержимым занимает ширину одной колонки.
all - элемент занимает все колонки
Chrome до версии 50, Safari до версии 9 и Аndroid до версии 80 поддерживают свойство -webkit-column-span.
column-width Задаёт оптимальную ширину колонки в многоколоночном тексте. Реальная ширина колонки может быть больше введённого значения; сокращение указанной ширины приводит к отмене нескольких колонок, текст при этом выстраивается в одну колонку. 
column-width: размер | auto
auto - Ширина колонок вычисляется автоматически на основе других свойств (column-count, column-gap).
Firefox поддерживает свойство -moz-column-width.
Safari, Chrome и Аndroid поддерживают свойство -webkit-column-width.
