Columns - колонки

column-count column-fill column-gap column-rule column-rule-color column-rule-style column-rule-width column-span column-width

Columns - универсальное свойство, которое позволяет одновременно задать ширину и количество колонок многоколоночного текста. Является сокращенной формой записи свойств column-width и column-count. Порядок значения не имеет. Значение по умолчанию auto, не наследуется, применяется к блочным элементам (кроме таблиц), ячейкам и элементам, у которых свойство display установленo как inline-block; aнимируется.

.elem {
  columns: column-width || column-count
}
Свойство columns позволяет разработчикам разбить текст на странице в несколько колонок. К примеру мы хотим разбить текст в блоке <div> с классом content в две колонки, для этого в CSS необходимо написать:
.content {
  columns: 2;
  column-gap: 40px;
  column-rule: 1px dotted #ccc;
}
.el {
  columns: 200px auto; /* Ширина 200px, кол-во задаётся браузером. */
  columns: 3 12em;     /* Три колонки минимальной ширины 12em каждая. */
  columns: 2;          /* Две колонки, ширина определяется браузером */
}

Свойство column-count

Свойство column-count oпределяет количество колонок в многоколоночном тексте. Применяется к блочным элементам (кроме таблиц), ячейкам и элементам, у которых display установлен как inline-block. В качестве значений используют целые положительные числа. При значении auto, число колонок вычисляется автоматически на основе других свойств (column-width, column-gap).

.elem {
  column-count: число | auto;
}

Свойство column-fill

Свойство column-fill применяется к родительскому элементу, разбитому на столбцы и указывает как содержимое располагается внутри столбца.

.elem {
  column-fill: auto | balance | balance-all;
}
  • auto - Колонки заполняются последовательно.
  • balance - Содержимое равномерно распределяется между колонок. При печати это правило относится только к последней странице.
  • balance-all - Содержимое равномерно распределяется между колонок. При печати это правило относится ко всем страницам.
Значение balance-all имеет слабую поддержку браузерами.

Свойство column-gap

Свойство column-gap задаёт расстояние между колонками в многоколоночном тексте. Значение по умолчанию normal, не наследуется, анимируется.

.elem {
  column-gap: размер | normal;
}
  • normal - межколоночный интервал устанавливается браузером (1em).
  • размер - размер межколоночного интервала в стандартных величинах.
Поддержка браузерами значений в % и функции calc() неполная.

Свойство column-rule

Свойство column-rule устанавливает ширину, стиль и цвет линии, находящейся между колонками в мультиколоночной вёрстке. Эта краткая форма записи для свойств: column-rule-width, column-rule-style, и column-rule-color. Значение по умолчанию medium none, анимируется, не наследуется.

.elem {
  column-rule: column-rule-width || column-rule-style || column-rule-color;
}
  • column-rule-width - ширина линии в стандартных величинах. Значение по умолчанию medium (4px). 
    .elem {
      column-rule-width: размер | thin (2px) | medium (4px) | thick (6px)
    }
  • column-rule-style - тип линии, аналогичен значениям border-style. Значение по умолчанию none (или hidden)
  • column-rule-color - цвет линии в стандартных величинах, по умолчанию совпадает с цветом текста.

Свойство column-span

Свойство column-span oпределяет, как должен отображаться элемент в многоколоночном тексте - занимать ширину всех колонок или только одну из них. Это свойство обычно применяют для заголовков текста или изображений, которые не должны разбиваться на колонки. Значение по умолчанию none, не анимируется, не наследуется.

.elem {
  column-span: none | all;
}
  • none - элемент наряду с обычным содержимым занимает ширину одной колонки.
  • all - элемент занимает все колонки

Свойство column-width

Свойство column-width задаёт оптимальную ширину колонки в многоколоночном тексте. Реальная ширина колонки может быть больше введённого значения; сокращение указанной ширины приводит к отмене нескольких колонок, текст при этом выстраивается в одну колонку. Значение по умолчанию auto, не наследуется, не анимируется.

.elem {
  column-width: размер | auto;
}
  • auto - ширина колонок вычисляется автоматически на основе других свойств (column-count, column-gap).
  • размер - ширины колонки в стандартных величинах.
