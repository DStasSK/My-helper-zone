Mаркированный/нумерованный список

Для того чтобы на странице интернет ресурса можно было отобразить маркированный список, в HTML предусмотрены теги <ul> и <li>. Между тегами <ul> помещаются элементы списка начинающиеся с тега <li>.

Списки на html страницах по умолчанию выводятся с маркерами, которые задаются параметрами «disc» для тега <ul> и «decimal» для тега <ol>.

Для того чтобы на web-странице отображался нумерованный список в HTML предусмотрены теги <оl> и <li>. Между тегами <оl> помещаются сегменты списка которые начинаются с тега <li>.

Cписок определений <dl> - состоит из пар термин <dt> - <dd> определение.

<ul>

Элемент <ul> (от англ. unordered list — неупорядоченный список) устанавливает маркированный список. Каждый пункт списка должен начинаться с элемента <li>.

<ul>
  <li>пункт списка</li>
  <li>пункт списка</li>
  <li>пункт списка</li>
</ul>
Атрибуты:
type Устанавливает вид маркера списка. 
<ul type="disc | circle | square">...</ul>
Данный атрибут устарел, взамен используйте стили.
  • Чебурашка
  • Крокодил Гена
  • Шапокляк
<ul style="list-style: square">
  <li>Чебурашка</li>
  <li>Крокодил Гена</li>
  <li>Шапокляк</li>
</ul>

<ol>

Элемент <ol> (от англ. ordered list — упорядоченный список) устанавливает нумерованный список. Каждый элемент списка должен начинаться с <li>. Если к <ol> применяется таблица стилей, то элементы <li> наследуют эти свойства.

<ol>
  <li>пункт списка</li>
  <li>пункт списка</li>
  <li>пункт списка</li>
</ol>

Атрибуты тега <ol>

reversed

задает отображение списка в обратном порядке (например, 9, 8, 7…).

Атрибут reversed является логическим атрибутом, он может быть истиной, либо ложью. Ложь является значением по умолчанию, значение становится истинным, когда атрибут reversed появляется у элемента <ol>.

<ol reversed>
start

задает начальное значение, от которого пойдет отсчет нумерации.

<ol start="10">
первому пункту присвоит порядковый номер «10».
type

задает вид маркера для использования в списке (в виде букв или цифр).
принимаемые значения:

1 — значение по умолчанию, десятичная нумерация
A — нумерация списка заглавными буквами (A, B, C, D).
a — нумерация списка строчными буквами (a, b, c, d).
I — нумерация римскими заглавными цифрами (I, II, III, IV).
i — нумерация римскими строчными цифрами (i, ii, iii, iv).
<ol type="I">

<li>

Элемент <li> (от англ. list item — пункт списка) определяет отдельный пункт списка. Внешний элемент <ul> или <ol> устанавливает тип списка — маркированный или нумерованный.

<ul>
  <li>элемент маркированного списка</li>
</ul> 

<ol>
  <li>элемент нумерованного списка</li>
</ol>
Атрибуты:
type Устанавливает вид маркера нумерованного или маркированного списка. 
<li type="disc | circle | square">...</li>
<li type="A | a | I | i | 1">...</li>
Данный атрибут устарел, взамен используйте стили - list-style.
value

атрибут value устанавливает номер, с которого будет начинаться список. value применяется только для нумерованных списков, когда элемент <li> находится внутри контейнера <ol>. При этом не имеет значения, какой тип списка установлен с помощью type, атрибут value одинаково работает и с римскими и с арабскими числами.

В качестве примера, если у второго пункта списка атрибут value задан как 9, номер этого пункта списка будет выводиться, как если бы он был девятым. Все последующие пункты списка будут пронумерованы от 9.

<ol>
  <li>Пункт №1</li>
  <li value="9">Пункт №2</li>
  <li>Пункт №3</li>
</ol>
  1. Пункт №1
  2. Пункт №2
  3. Пункт №3

как увеличить размер самих маркеров?

Есть несколько способов.
  • Добавить <span> внутрь <li> и для них задать разный размер шрифта. 
    <ul style="font-size: 20px">
      <li><span style="font-size: 14px">пункт</span></li>
      <li><span style="font-size: 14px">пункт</span></li>
    </ul>
  • Установить маркеры через content у ::before, предварительно установив свойству list-style значение none.
  • Маркеры вставить картинкой с помощью background или list-style-image.

Как в теге <ol> перенести нумерацию на право и убрать точку.
По умолчанию "1. Глава", а надо что бы было "Глава 1"

  1. Section
  2. Section
  3. Section
<style>
  ol.pr1 {counter-reset:section; list-style:none; padding-left:0;}
  .pr1 li::after {content:' ' counter(section);
                  counter-increment:section;}
</style>

<ol class="pr1">
  <li>Section</li>
  <li>Section</li>
  <li>Section</li>
</ol>
