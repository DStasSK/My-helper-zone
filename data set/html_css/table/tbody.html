Тэг tbody

Элемент tbody (от англ. table body — тело таблицы) предназначен для хранения одной или нескольких строк таблицы. Это позволяет создавать структурные блоки, к которым можно применять единое оформление через стили, а также управлять их видом через скрипты.

Атрибуты тега <tbody>

Данные атрибуты устарели, вместо них необходимо использовать стили.
align char charoff bgcolor valign
align Задаёт выравнивание по горизонтали для всех ячеек в пределах tbody. Если требуется применить разное выравнивание для каждой ячейки, воспользуйтесь стилями или используйте атрибут align для элемента td или th. 
<tbody align="left | center | right | justify">...</tbody>
Для выравнивания содержимого ячеек внутри элемента tbody используйте стилевое свойство text-align, применяя его к селектору tbody.
char Выравнивание содержимого ячеек относительно заданного символа. Работает только в том случае, если значение атрибута align установлено как char. 
<table>
   <tbody align="char" char="символ">...</tbody>
</table>
Данный атрибут не поддерживается.
charoff Сдвигает содержимое ячейки относительно заданного символа. Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячейки влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" - сместить на три символа влево. 
<table>
   <tbody align="char" char="символ"  charoff="число">...</tbody>
</table>
Данный атрибут не поддерживается.
bgcolor Цвет фона ячеек, которые расположены внутри контейнера tbody. 
<tbody bgcolor="цвет">...</tbody>
Чтобы задать цвет фона внутри tbody используйте стилевое свойство background-color, применяя его к селектору tbody.
valign Задает вертикальное выравнивание содержимого ячеек в контейнере tbody. По умолчанию контент ячейки располагается по вертикали в центре. 
<tbody valign="baseline | bottom | middle | top">...</tbody>
Для изменения выравнивания по высоте внутри tbody воспользуйтесь свойством vertical-align, добавляя его к селектору tbody.
