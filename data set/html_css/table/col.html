Тэг col

Элемент col (от англ. column — колонка) задаёт ширину и другие характеристики одной или нескольких колонок таблицы. При наличии этого элемента браузер начинает показывать содержимое таблицы, не дожидаясь её полной загрузки. col можно использовать совместно с элементом colgroup, который задаёт группу колонок, обладающих общими характеристиками.

Для col допустимо использовать следующие стилевые свойства: border, background, width, visibility. Остальные свойства не оказывают никакого эффекта на элемент.

Атрибуты тега <col>

Данные атрибуты устарели, вместо них необходимо использовать стили.
align char charoff span valign width
align Задаёт выравнивание содержимого колонок по горизонтали. 
<col align="left | center | right | justify">

<col align="left">
<col span="3" align="center">
Данный атрибут не поддерживается.
char Атрибут char заставляет выравниваться содержимое колонки по заданному символу. Эта возможность может пригодиться для выравнивания колонки чисел по точке. Работает только в том случае, если значение атрибута align установлено как char. 
<col align="char" char=",">
Данный атрибут не поддерживается.
charoff Сдвигает содержимое ячейки относительно заданного символа. Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячейки влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" — сместить на три символа влево. 
<col align="char" char="," charoff="-2">
Данный атрибут не поддерживается.
span Определяет число колонок, к которым будут применяться заданные характеристики. Если этот атрибут отсутствует, то элемент col работает для одной колонки. Допускается применять атрибут span к нескольким колонкам и таким образом формировать группы колонок с одинаковыми характеристиками. 
<table>
   <col width="150">
   <col span="5" width="50">
   <col span="4" width="75">
   <tr>...</tr>
</table>
valign Задает вертикальное выравнивание содержимого колонки. По умолчанию контент ячеек в колонке располагается по центру вертикали. 
<col valign="baseline | bottom | middle | top">
Данный атрибут не поддерживается.
Для изменения выравнивания в ячейках воспользуйтесь свойством vertical-align, добавляя его к селектору td.
width Ширина колонок. В качестве значения обычно применяется любое целое значение в пикселях или процентах от доступного пространства. Альтернативный вариант основан на использовании символа *, который обозначает всю доступную ширину. Допустимо использовать кратные значения, например 3*. Например, если для таблицы установлена ширина 600 пикселей, а для колонок - 1*, 2*, то в пикселях это будет соответственно 200 и 400. 
<col width="значение">
Для изменения ширины колонок можно использавоть псевдокласс :nth-child и свойство width, добавляя их к селектору td.
