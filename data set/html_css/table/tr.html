Тэг tr

Элемент tr (от англ. table row - строка таблицы) служит контейнером для создания строки таблицы. Каждая ячейка в пределах такой строки устанавливается с помощью элемента th или td.

<table>
  <tr>
    <td>...</td><td>...</td>
  </tr>
</table>

Атрибуты тега <tr>

Данные атрибуты устарели, вместо них необходимо использовать стили.
align bgcolor bordercolor char charoff valign
align Задаёт выравнивание содержимого ячеек строки по горизонтали. Выравнивание осуществляется для всех ячеек в пределах одной строки. Если требуется применить разное выравнивание для каждой ячейки, воспользуйтесь стилями или используйте атрибут align для элемента <td> или <th>. 
<tr align="left | center | right | justify">...</tr>
Для выравнивания содержимого ячеек внутри <tr> используйте стилевое свойство text-align, применяя его к селектору tr.
bgcolor Устанавливает цвет фона строки таблицы. 
<tr bgcolor="Цвет">...</tr>
bordercolor Цвет рамки. 
<tr bordercolor="Цвет">...</tr>
Данный атрибут не поддерживается браузерами.
char Выравнивает содержимое ячеек внутри <tr> по заданному символу. Эта возможность может пригодиться для выравнивания колонки чисел по точке. Работает только в том случае, если значение атрибута align установлено как char. 
<tr align="char" char="символ">...</tr>
Данный атрибут не поддерживается браузерами.
charoff Смещает содержимое ячейки относительно заданного символа. Если атрибут align установлен как char, а для атрибута char в качестве значения задан символ, то с помощью атрибута charoff можно смещать содержимое ячейки влево или вправо относительно заданного символа. Так, charoff="3" означает, что браузер должен сместить содержимое ячейки на три символа вправо, а charoff="-3" — сместить на три символа влево. 
<tr align="char" char="," charoff="-2">...</tr>
Данный атрибут не поддерживается браузерами.
valign Устанавливает вертикальное выравнивание содержимого ячеек в строке. По умолчанию контент ячейки располагается по ее вертикали в центре. 
<th valign="baseline | bottom | middle | top">...</th>
Для изменения выравнивания в ячейках воспользуйтесь свойством vertical-align, добавляя его к селектору tr
