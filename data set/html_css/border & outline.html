border

CSS cвойство border позволяет задавать толщину, цвет и тип линии по периметру вокруг элемента. Параметры данного свойства могут записываться в одной строке, разделяясь через пробел и в любом порядке.

border: border-width || border-style || border-color
border-width - определяет толщину границы
border-color - устанавливает цвет границы в любом CSS формате
border-style - определяет вид границы
dotted dashed solid double groove ridge inset outset

Для установки границы только на определённых сторонах элемента, воспользуйтесь свойствами border-top, border-bottom, border-left, border-right - позволяет одновременно установить толщину, стиль и цвет границы элемента. Значения могут идти в любом порядке, разделяясь пробелом.

border-top: [border-top-width || border-top-style || border-top-color]

outline

CSS cвойство outline задаёт внешнюю границу для всех четырех сторон элемента одновременно. С помощью параметров этого свойства можно установить цвет, стиль, а так же толщину линий расположенных по периметру элемента.

outline: outline-color || outline-style || outline-width
В отличие от линии, задаваемой через border, у свойства outline есть следующие особенности:
  • не влияет на размеры и положение самого элемента;
  • не занимает место, не влияет на окружающие элементы и отображается поверх них;
  • свойство border-radius не действует;
  • применяется сразу ко всем четырём сторонам - нельзя задать параметры линии на отдельных сторонах элемента;

outline-offset

Устанавливает расстояние между рамкой, созданной с помощью свойства outline, и краем элемента или его границей добавленной через border. Отрицательное значение отображает рамку внутри элемента, положительное - вокруг элемента.

div{
  outline:2px dashed #000;
  outline-offset:-10px;
  padding:20px;
}

border-style

Устанавливает стиль границы вокруг элемента. Допустимо задавать индивидуальные стили для разных сторон элемента. Значение по умолчанию none, не наследуется, применяется ко всем элементам, не анимируется.

Для управления видом границы предоставляется несколько значений свойства border-style. Вид зависит от используемого браузера и заданной толщины границы.

border-style: none | hidden | dotted | dashed | solid |
              double | groove | ridge | inset | outset
none не отображает границу и её толщина (border-width) задаётся нулевой.
hidden имеет тот же эффект, что и none за исключением применения border-style к ячейкам таблицы, у которой значение свойства border-collapse установлено как collapse. В этом случае вокруг ячейки граница не будет отображаться вообще.

Разрешается использовать одно, два, три или четыре значения, разделяя их между собой пробелом.

Если указано одно значение - стиль границы будет задан для всех сторон элемента. Если указано два значения - первое значение устанавливает стиль верхней и нижней границ, второе - левой и правой. Если указано три значения - первое значение задаёт стиль верхней границы, второе - одновременно левой и правой границ, а третье - нижней границы. Если указано четыре значения - поочередно устанавливается стиль верхней, правой, нижней и левой границ.
