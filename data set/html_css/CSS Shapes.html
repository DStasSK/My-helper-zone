CSS Shapes позволяет создавать интересные и уникальные макеты, определяя геометрические фигуры, изображения и градиенты, вокруг которых может обтекать текстовое содержимое.

CSS Shapes позволяет определить геометрические фигуры, вокруг которых обтекает текст. Такими фигурами могут быть: круг, эллипс, простые или сложные многоугольники, а также изображения и градиенты. На практике CSS Shapes позволяет отображать круговой текст вокруг круглого аватара; выводить текст поверх части полноразмерного фонового изображения; отображать текст, обтекающий буквицы в статье.

IE не поддерживает CSS свойства Shapes.

Свойство shape-margin

CSS свойство shape-margin добавляет внешние отступы к фигуре, созданной с помощью shape-outside. Значение может быть задано в px, em, % или в любой другой стандартной единице измерения CSS.

.shape {
  float: left;
  width: 150px;
  height: 150px;
  clip-path: polygon(0 0, 150px 150px, 0 150px);
  shape-outside: polygon(0 0, 150px 150px, 0 150px);
  shape-margin: 20px;
}

Свойство shape-image-threshold

CSS свойство shape-image-threshold определяет порог альфа-канала фигуры, другими словами, какой процент изображения будет считаться прозрачным и использует полученную фигуру для обтекания её текстом. Принимает одно значение в диапазоне от 0,0 (полностью прозрачный) до 1,0 (полностью непрозрачный).

Любые пиксели, значение альфа-компоненты которых больше порогового значения, считаются частью фигуры для целей определения ее границ. Например, значение 0,5 означает, что фигура будет включать все пиксели, которые непрозрачны более чем на 50%, а остальная "прозрачная" часть фигуры не будет учитываться и это пространство будет доступно для обтекания текстом.

В качестве фигуры также можно использовать градиенты. Они работают как изображения. Текст будет обтекать прозрачную часть.

<div class="gradient"></div>
<div>text...</div


.gradient {
  float: left;
  height:230px; width:220px;
  background: linear-gradient(135deg, #000, transparent);
  shape-outside: linear-gradient(135deg, #fff, transparent);
  shape-image-threshold: 0.6;
}

Свойство shape-outside

CSS свойство shape-outside определяет фигуры, вокруг которых может обтекать текст, позволяет обтекать тексту сложные объекты, а не прямоугольные блоки. Чтобы добавить свойство shape-outside к элементу, у него должна быть установлена высота и ширина а также он должен быть обтекаемым, т.е. значение у свойства float должно быть left|right.

.elem {
  shape-outside: none | [basic-box] figure | [image-url | gradient];
}
  • none - контент обтекает элемент как обычно, значение по умолчанию.
  • basic-box - площадь плавающего элемента вычисляется в соответствии с формой краев плавающего элемента. Форма включает любую кривизну, созданную свойством border-radius. Принимает одно из четырех значений:
    • margin-box - обтекаемая форма по границам внешних отступов (значение по умолчанию).
    • border-box - обтекаемая форма по внешним границам элемента (без margin).
    • padding-box - обтекаемая форма по внутренним границам элемента (включает padding).
    • content-box - обтекаемая форма по внешним границам контента (не включает padding).
  • figure - форма плавающего объекта вычисляется на основе одной из черытех фигур, созданных функциями inset(), circle(), ellipse() или polygon().
  • image-url - форма плавающего объекта извлекается и вычисляется на основе альфа-канала указанного изображения, как определено свойством shape-image-threshold. Еесли изображение недействительно, эффект будет таким же, как если бы было указано ключевое слово none.
  • gradient - форма плавающего объекта извлекается и вычисляется на основе альфа-канала указанного градиента, как определено свойством shape-image-threshold.

Базовые фигуры свойства shape-outside

На данный момент есть четыре значения функций у свойства shape-outside:
circle() ellipse() inset() polygon()
circle()

Функция circle() в дополнение к радиусу принимает позиционирование через at. Положение по умолчанию — центр круга, поэтому circle() будет явно записан как circle(50% at 50% 50%) или circle(100px at 100px 100px); значениями являются горизонтальное и вертикальное положение, соответственно.

Представьте ситуацию, что есть круглый аватар автора и чтобы текст с описанием обтекал его. Для придания тексту круглой формы недостаточно использовать border-radius: 50% на элементе аватара; текст по прежнему будет считать аватар прямоугольным элементом.

  float:left;
  height:200px; width:200px;

circle() = circle(50%)

ellipse()
Похожей на circle() является функция ellipse(), которая создаёт эллипс. Для демонстрации добавим элемент с классом ellipse. На этот раз установили разную высоту и ширину, чтобы сделать вертикально вытянутый овал.
text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text text		 
<div class="ellipse"></div>
<span>text...</span>


.ellipse {
  float: left;
  height:200px; width:100px;
  shape-outside: ellipse(40px 80px);
  clip-path: ellipse(40px 80px);
  shape-margin: 10px;
}

Разница между ellipse() и circle() заключается в том, что эллипс имеет два радиуса - rx и ry или радиус по оси X и радиус по оси Y. Поэтому приведённый выше пример можно записать в виде ellipse(75px 150px).

Параметры позиционирования одинаковы для кругов и эллипсов. Радиусы, помимо того, что они являются единицей измерения, также включают farthest-side и closest-side. closest-side указывает на длину от центра до ближайшей стороны связанного блока и наоборот, farthest-side указывает на длину от центра до самой дальней стороны связанного блока. Эти два значения не действуют, если не установлено положение, отличное от значения по умолчанию.

Вот демонстрация разницы перестановки closest-side и farthest-side для ellipse() со смещением на 25% по осям X и Y.

  clip-path: ellipse(farthest-side closest-side at 25% 25%);
}

inset()
Используя функцию inset() можно вставлять прямоугольники с внутренними отступами - подобно свойствам padding или margin. Функция inset() принимает значение top right bottom left по часовой стрелке, при использовании только одного значения - установит все четыре стороны одинаковыми.
Функция inset() также может принимать параметр border-radius со значением радиуса и текст будет учитывать скруглённые уголки. В примере с 40px со всех сторон и скруглением 50px.

polygon()

Интересное использование функции polygon() заключается в том, что текстовое содержимое может располагаться между двумя или более фигурами. Поскольку фигура polygon() очень гибкая и динамичная, это одна из самых больших возможностей для создания действительно уникальных макетов в журнальном стиле.

В данном примере текст находится между двумя многоугольниками. Многоугольники можно использовать для вырезания фигур вокруг изображений или других элементов, например - вокруг большой буквы.

CSS Shapes и изображения

Интересной особенностью CSS Shapes является то, что вам не всегда нужно явно определять фигуру с помощью функции. Можено использовать адрес полупрозрачного изображения для определения фигуры и текст будет автоматически обтекать её по прозрачным участкам изображения.

Важно отметить, что используемое изображение должно быть CORS-совместимым (т.е. с того же сервера что и остальные файлы сайта), в противном случае вы получите ошибку подобную приведённой ниже.

Access to image at 'file:///users/tania/star.png' from origin 'null' has been blocked by CORS policy: The response is invalid.

Если изображения размещены на другом домене, например, на CDN, вы должны убедиться, что они отправляют правильные заголовки, чтобы их можно было использовать для фигур. Из-за этого требования к изображениям при просмотре страницы локально, не используя локальный веб-сервер, фигура не будет работать.

В отличие от других примеров, здесь будет использован тег img вместо div. На этот раз CSS простой — просто поместим url() в свойство shape-outside, как при использовании свойства background-image.

text text text text text text text text text text text text text text text text text text text text text text text text text text 
<img src="img/star.png" class="star">
<span>text...</span>

.star {
  float: left;
  height: 200px;
  width: 200px;
  shape-outside: url('img/star.png');
  shape-image-threshold: 0.5;
  shape-margin: 5px;
}
