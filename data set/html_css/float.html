CSS свойство float

Свойство float oпределяет, по какой стороне будет выравниваться элемент, при этом остальные элементы будут обтекать его с других сторон. Когда значение свойства float равно none, элемент выводится на странице как обычно, при этом допускается, что одна строка обтекающего текста может быть на той же линии, что и сам элемент. Применяется ко всем элементам (за исключением абсолютно позиционированных).

div {
  float: left;
  float: right;
  float: none;
}
  • left – устанавливает обтекание по левому краю
  • right – устанавливает обтекание по правому краю
  • none – отменяет обтекание элемента (значение по умолчанию)
К обтекаемым элементам автоматически применяется display:block и они в основном будут вести себя как блоки. При этом ширина элемента равна его содержимому плюс значения свойств padding, border и margin. В этом отношении элемент ведёт себя как строчно-блочный.
  • можно установить определённую высоту и ширину;
  • если высота не установлена, высотой элемента является line-height;
  • если применяется width:100%, то будет выглядеть как блочный элемент.

CSS свойство clear

Свойство clear устанавливает, с какой стороны элемента будет запрещено его обтекание остальными элементами. В случае если установлено обтекание, какого либо элемента с помощью свойства float, то параметр clear отменяет его действие для выбранных сторон.

div {
  clear: both;
  clear: left;
  clear: right;
  clear: none;
}
  • both – отмена обтекания элемента с правого и левого края одновременно
  • left – отмена обтекания по левому краю элемента
  • right – отмена обтекания по правой стороне элемента
  • none – отмена действия свойства clear

Очистка float.

Действие float может продолжаться и за пределами родителя, если высота обтекаемого элемента превышает высоту родителя. Чтобы отменить дейсвие float можно воспользоваться одним из способов:

  • использование overflow со значением auto, scroll или hidden на родителе элементов с float.
  • добавлять к элементу, идущему после элемента с float свойство clear:both.
  • добавить к родителю псевдоэлемент ::after со свойством clear:both.

Обычно вводят универсальный класс, к примеру, clearfix и вставляют пустой <div class="cleаrfix"> с этим классом. Но частое включение пустого элемента <div> со свойством clear захламляет код, особенно при активном использовании свойства float. Логично перенести всё в стили, избавившись от лишних элементов. Для этого используется псевдоэлементом :after, который в комбинации со свойством content добавляет пустой текст после содержимого элемента. К такому тексту можно применить стилевые свойства, в частности clear.

.clearfix::after {
  content: "";     /* Генерируем пустой элемент */
  clear: both;     /* Отменяем обтекание*/
  display: block;
}

Вместо значения block у display иногда вставляют значение table, но смысл это не меняет, так же как и набор других возможных свойств. Их задача - создать невидимый блок, который будет отменять действие float у вышестоящих элементов.

Удобство приведённого метода заключается в использовании класса clearfix, который при необходимости добавляется к любому элементу.
