position

  • position
    absolute fixed relative static sticky
    top right bottom left inset inset...

Свойство position устанавливает способ позиционирования элемента относительно окна браузера или других объектов на веб-странице, т.е. указывает, как элемент позиционируется в документе.

div {
  position: absolute | fixed | relative | static | sticky;
}
absolute

Абсолютное позиционирование. Элемент удаляется из обычного потока документа, и для элемента в макете страницы не выделяется дополнительное пространство. Он располагается относительно своего ближайшего спозиционированного предка, если такой есть; в противном случае он помещается относительно края окна браузера.

Положение элемента задаётся значениями left, top, right и bottom, также на положение влияет значение свойства position родительского элемента.

Eсли у родителя значение position установлено как static или родителя нет, то отсчёт координат ведётся от края окна браузера.

Если у родителя значение position задано как relative, то отсчёт координат ведётся от края родительского элемента.

fixed

Фиксированное позиционирование.

По своему действию это значение близко к absolute, но в отличие от него привязывается к области просмотра браузера и не меняет своего положения при прокрутке веб-страницы. Положение определяется значениями left, top, right и bottom.

При выходе элемента за пределы видимой области справа или снизу от неё, не возникает полос прокрутки и элемент просто «обрезается».

При печати fixed-элемент помещается в одно и то же место на каждой странице.
relative

Относительное позиционирование.

Положение элемента устанавливается относительно его исходного места. Добавление свойств left, top, right и bottom изменяет позицию элемента и сдвигает его в ту или иную сторону от первоначального расположения.

Смещение не влияет на положение любых других элементов.

static

Статичное позиционирование.

Значение по умолчанию. Элементы отображаются как обычно. Использование свойств left, top, right и bottom не приводит к каким-либо результатам.

sticky

Это сочетание относительного и фиксированного позиционирования.

Элемент рассматривается как позиционированный относительно, пока он не пересекает определённый порог, после чего рассматривается как фиксированный. Обычно применяется для фиксации заголовка на одном месте, пока содержимое, к которому относится заголовок, прокручивается на странице.

C помощью свойства top мы указываем точку, по достижении которой при прокрутке элемент превращается в фиксированный.

Следует понимать, что при вертикальной прокрутке для элемента имеет смысл указывать только top и bottom. При горизонтальной прокрутке будут работать только свойства left и right. Также имеет значение, чтобы высота или ширина родительского контейнера была больше размера элемента. В противном случае не будет полосы прокрутки и элемент останется на своём месте.

Прилипание элемента можно делать внутри блока с overflow, у которого значение задано как scroll или auto.

Смещение не влияет на положение любых других элементов.
Работает только, если родительский блок имеет назначенную высоту height | min-height
Браузер Safari поддерживает значение -webkit-sticky.
EI не поддерживает данное значение свойства position.
Убедитесь, что элементы, расположенные с absolute или fixed значением, не затеняют другой контент при увеличении масштаба отображения страницы, увеличивая размер текста.

top

Cвойство top частично определяет вертикальную позицию позиционируемого элемента. Для позиционированного элемента определяет расстояние от верхнего края родительского элемента, не включая отступ, поле и ширину рамки, до верхнего края дочернего элемента. Оно не влияет на непозиционируемые элементы (т.е. top не применится, если задано position: static). Значение по умолчанию - auto.

.elem {
  position: relative;
  top: размер | проценты | auto;
}

В качестве значений принимаются любые единицы длины, принятые в CSS — например, пиксели (px), дюймы (in), пункты (pt) и др. Значение свойства top может быть и отрицательным, в этом случае возможны наложения разных элементов друг на друга.

  • Когда свойству position задано значение absolute или fixed — значение свойства top устанавливается как расстояние между верхней гранью элемента и верхней гранью родительского контейнера.
  • Когда свойству position задано значение relative — значение свойство top устанавливается как расстояние, на которое смещается верхний край элемента от нормальной позиции.
  • Когда свойству position задано значение sticky — свойство top работает так, как при position: relative во время нахождения элемента внутри области просмотра, и как position: fixed вне области просмотра.

right

Cвойство right частично определяет горизонтальное положение позиционируемого элемента. Оно не влияет на непозиционируемые элементы (т.е. right не применится, если задано position: static). Значение по умолчанию - auto.

.elem {
  position: relative;
  right: размер | проценты | auto;
}

В качестве значений принимаются любые единицы длины, принятые в CSS — например, пиксели (px), дюймы (in), пункты (pt) и др. Значение свойства может быть и отрицательным, в этом случае возможны наложения разных элементов друг на друга.

Параметр свойства right может быть проигнорирован, если объект не может менять свою ширину и задано свойство left, при этом ориентация элементов задана слева направо. Если ориентация задана справа налево, то будет игнорироваться значение свойства left.

Для позиционированного элемента определяет расстояние от нижнего края родительского элемента, не включая отступ, поле и ширину рамки, до нижнего края дочернего элемента. Оно не влияет на непозиционируемые элементы (т.е. bottom не применится, если задано position: static). Значение по умолчанию - auto.

left

Cвойство left частично определяет горизонтальное положение позиционируемого элемента. Оно не влияет на непозиционируемые элементы (т.е. left не применится, если задано position: static). Значение по умолчанию - auto.

inset

Свойство inset задает значение для свойств top, right, bottom и left, с помощью которого значения отступов будут автоматически подстроены в зависимости от направления текста на разных языках. Это свойство одновременно задает значение для отступов со всех сторон.

Свойство применяется к позиционированным элементам у которых свойство position имеет значение отличное от static.

inset: размер | проценты | auto
В качестве значений принимаются px, em, rem и %. Значение по умолчанию 0.

Можно указать от одного до четырёх значений через пробел. Значения применяются к четырём сторонам соответственно, по часовой стрелке, начиная с верхнего края. Если значений меньше 4-х, то порядок определяется аналогично margin или padding.

Если указывать left и right в процентах, то они считаются от ширины элемента, если top и bottom, то от высоты.

div {
   position: absolute;
   top: 0;
   right: 0;
   bottom: 0;
   left: 0;
}

/* или */

div {
   position: absolute;
   inset: 0;
}

Свойство inset определяет физичекое смещение блока и поэтому оно не является логическим свойством. В то же время его составляющие являются логическими свойствами, где позиционирование опирается не на стороны окна браузера, а на направление письма (определяется свойсвом writing-mode) и будет меняться вместе с ним.

  • inset-block-start

  • inset-inline
  • inset-inline-start
  • inset-inline-end

Данные свойства применяется к позиционированным элементам у которых свойство position имеет значение отличное от static.

CSS свойство inset-block задает расстояние между элементом и родительским элементом в направлении блока. Первое значение определяет расстояние от родителя до начала текущего элемента, второе значение - до конца текущего элемента. Для страниц на английском языке направление блока - сверху вниз.

el{ inset-block: inset-block-start inset-block-end; }
el{ inset-block: 10px 20%; }
el{ inset-block: 10px; }       /* inset-block-start == inset-block-end */
el{
   inset-block-start: 10px;
   inset-block-end: 10px;
}

CSS свойство inset-inline задает расстояние между элементом и родительским элементом во встроенном направлении. Первое значение определяет расстояние от родителя до начала текущего элемента, второе значение - до конца текущего элемента. Для страниц на английском языке направление направление строки - слева направо.

el{ inset-inline: inset-inline-start inset-inline-end; }
el{ inset-inline: 10px 20%; }
el{ inset-inline: 10px; }       /* inset-inline-start == inset-inline-end */
el{
   inset-inline-start: 10px;
   inset-inline-end: 10px;
}
