text-indent

Свойство text-indent используется для установки величины отступа первой строки текста. Воздействия на все последующие строки, при установке отступа, не оказывается. В качестве значений принимаются стандартные положительные единицы CSS.

.elem {
  text-indent: 20px;     /* отступ 30 пикселей */
  text-indent: 1em;      /* тступ 1em (1em = размер шрифта) */
  text-indent: inherit;  /* наследует параметры родителя */
}
Отступ первой строки 36px; font-size:18px; Lorem ipsum dolor sit amet consectetur adipisicing elit. Ratione debitis vitae nesciunt esse odit alias eligendi magni tempora.
