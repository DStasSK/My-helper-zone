text-decoration

Содержание web-сайта, в тех или иных случаях, требует оформления его части в виде подчёркнутого, перечёркнутого, а также текста с линией сверху или мигающий вариант отображения, который в прочем в настоящее время мало используется в виду не соответствия с текущими требованиями. Одновременно можно задействовать более одного стиля, перечисляя выбранные значения через пробел.

a {text-decoration: none;}
отменяет все заданные ранее эффекты, включая и подчеркивания у ссылок которое установлено по умолчанию. 
a {text-decoration: underline;}
подчёркивание текста сплошной линией. 
a {text-decoration: line-through;}
вывод перечеркнутого текста. 
a {text-decoration: overline;}
отображает текст с линией поверх него. 
a {text-decoration: blink;}
этот параметр задаёт пульсирующий текст (который не пульсирует, т.к. устарело). 
a {text-decoration: underline line-through overline;}

Т.к. это свойство наследуется, то если очень нужно избавиться от этого свойства, то нужно дочерние элементы сделать абсолютно спозиционированными, либо добавить им inline-block, либо сделать их плавающими(float).

div {
  text-decoration: none;
  /* position: absolute; */
  /* display: inline-block; */
  /* float: left; */
}

text-decoration-color

Устанавливает цвет линии, которая добавляется через свойство text-decoration.

text-decoration-line

Определяет, как должна добавляться декоративная линия к тексту — подчёркивание, перечёркивание, над текстом. Одновременно можно добавлять несколько линий, перечисляя значения через пробел.

text-decoration-skip-ink

Задаёт, как выводить линию под или над текстом, когда линия пересекает выносные элементы букв, вроде «у», «р», «ц» и др. Линия может быть сплошной или прерываться, чтобы пропустить «хвостики» букв.

auto - линия прерывается на выносных элементах символов.

none - линия отображается сплошной.

a {
  text-decoration:underline;
  text-decoration-skip-ink:none;
}

text-decoration-style

Устанавливает стиль декоративной линии, которая добавляется к тексту через свойство text-decoration или text-decoration-line.

solid - Одинарная линия.

double - Двойная линия.

dotted - Точечная линия.

dashed - Пунктирная линия.

wavy - Волнистая линия.

a {
  text-decoration:underline;
  text-decoration-style:wavy;
}
