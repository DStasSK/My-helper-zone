text-decoration

CSS свойство text-decoration добавляет декорирование текста (подчеркивание, перечеркивание, линия над текстом). Ссвойство text-decoration является сокращенным свойством для:

  • text-decoration-line
  • text-decoration-style
  • text-decoration-color
.el {
   text-decoration: text-decoration-line text-decoration-style text-decoration-color |
                    none | initial | inherit;
}
text-decoration-lineОпределяет расположение декоративной линии (над текстом, под текстом, проходящей через текст или отсутствие линии).
text-decoration-styleУстанавливает стиль декоративной линии.
text-decoration-colorУстанавливает цвет декоративной линии.
noneОтменяет все заданные ранее эффекты, включая и подчеркивания у ссылок которое установлено по умолчанию.
initialУстанавливает свойство в значение по умолчанию.
inheritЗначение наследуется от родительского элемента.

text-decoration-line

Определяет, как должна добавляться декоративная линия к тексту — подчёркивание, перечёркивание, над текстом. Одновременно можно добавлять несколько линий, перечисляя значения через пробел.

.el{
   text-decoration-line: none | underline | overline | line-through | initial | inherit
}
a {text-decoration: none;}
отменяет все заданные ранее эффекты, включая и подчеркивания у ссылок которое установлено по умолчанию. 
a {text-decoration: underline;}
подчёркивание текста сплошной линией. 
a {text-decoration: overline;}
отображает текст с линией поверх него. 
a {text-decoration: line-through;}
вывод перечеркнутого текста. 
a {text-decoration: blink;}
этот параметр задаёт пульсирующий текст (который не пульсирует, т.к. устарело). 
a {text-decoration: underline line-through overline;}
допускается использовать несколько значений одновременно.
Т.к. это свойство наследуется/раньше наследовалось, то если очень нужно избавиться от этого свойства, то нужно дочерние элементы сделать абсолютно спозиционированными, либо добавить им display: inline-block, либо сделать их плавающими при помощи свойства float.
div {
  text-decoration: none;
  /* position: absolute; */
  /* display: inline-block; */
  /* float: left; */
}

text-decoration-style

Устанавливает стиль декоративной линии, которая добавляется к тексту через свойство text-decoration или text-decoration-line.
solidОдинарная линия.
doubleДвойная линия.
dottedТочечная линия.
dashedПунктирная линия.
wavyВолнистая линия.
a {
  text-decoration: underline;
  text-decoration-style: wavy;
}

text-decoration-color

Устанавливает цвет декоративной линии. 
.el {
   text-decoration-color: color | initial | inherit;
}
colorУстанавливает цвет декоративной линии (HEX, RGB, RGBA, HSL, HSLA, "Предопределённые цвета"). Значением по умолчанию является текущий цвет элемента.
initialУстанавливает свойство в значение по умолчанию.
inheritЗначение наследуется от родительского элемента.
.el {
   text-decoration: underline;
   text-decoration-color: red;
}

text-decoration-skip-ink

Данное свойство, при выводе линии под или над текстом, определят поведении линии при пересечении выносных элементов букв, вроде «у», «р», «ц» и др. Линия может быть сплошной или прерываться, чтобы пропустить «хвостики» букв.

auto-  линия прерывается на выносных элементах символов.
none-  линия отображается сплошной.
a {
  text-decoration:underline;
  text-decoration-skip-ink:none;
}
