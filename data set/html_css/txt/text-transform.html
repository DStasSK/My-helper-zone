text-transform

Свойство text-transform управляет преобразованием текста элемента в заглавные или прописные символы. Когда значение свойства отлично от none, регистр исходного текста будет изменён.

.elem {
  text-transform: capitalize | lowercase | uppercase | none;
}
  • uppercase - вывод всех символов в верхнем регистре
  • lowercase - ВЫВОД СТРОЧНЫХ СИМВОЛОВ (НИЖНИЙ РЕГИСТР)
  • capitalize - отображение всех слов в предложении с заглавной буквы
  • none - регистр символов остаётся неизменным
