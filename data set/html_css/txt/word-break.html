word-break

Свойство word-break указывает, как делать перенос строк внутри слов, которые не помещаются по ширине в заданную область. Значение по умолчанию normal, наследуется, применяется ко всем элементам.

Chrome до версии 44, Opera до версии 31, Safari до версии 9, Android и Opera Mobile не поддерживают значение keep-all.

Значения:
normal Используются правила переноса строк по умолчанию. Как правило, в этом случае строки не переносятся или переносятся в тех местах, где явно задан перенос (например, с помощью <br>, <wbr>, &shy;).
break-all Перенос строк добавляется автоматически, чтобы слово поместилось в заданную ширину блока. Значение не работает для текста на китайском, корейском или японском языке.
keep-all Не разрешает перенос строк в словах на китайском, корейском или японском языке. Для остальных языков действует как normal.
word-break:normal;
 
word-break:break-all;

word-wrap

Свойство word-wrap указывает, переносить или нет длинные слова, которые не помещаются по ширине в заданную область. Валидатор заменяет word-wrap на overflow-wrap.

Значения:
normal Строки не переносятся или переносятся в тех местах, где явно задан перенос (например, с помощью <br>, <wbr>, &shy;).
break-word Перенос строк добавляется автоматически, чтобы слово поместилось в заданную ширину блока.
