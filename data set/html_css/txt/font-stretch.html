CSS свойство - font-stretch

Свойство font-stretch устанавливает узкое, нормальное или широкое начертание шрифта, что позволяет уплотнять или расширять текст, при этом растягиваются сами символы. Значения свойства применяются ко всем элементам, наследуется, анимируется. Значение по умолчанию - normal.

Это не имеет никакого эффекта, если выбранный шрифт не содержит сжатых или расширенных вариантов символов.
p{
  font-stretch: ultra-condensed;  /* ультра-сжатый */
  font-stretch: extra-condensed;  /* экста-сжатый */
  font-stretch: condensed;        /* сжатый */
  font-stretch: semi-condensed;   /* полу-уплотненный */
  font-stretch: normal;           /* значение по умолчанию */
  font-stretch: semi-expanded;    /* немного растянутый */
  font-stretch: expanded;         /* растянутый */
  font-stretch: extra-expanded;   /* экста-растянутыйй */
  font-stretch: ultra-expanded;   /* ультра-растянутый */
}

Браузеры применяют свойство font-stretch не ко всем шрифтам, поэтому уплотнение или расширение текста может не работать с некоторыми популярными и распространёнными гарнитурами шрифтов. Также некоторые значения могут не давать видимого результата.
