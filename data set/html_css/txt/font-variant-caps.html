CSS свойство - font-variant-caps

Свойство font-variant-caps применяется для управления начертанием заглавных букв в тексте. Обычно используется в заголовках для создания капители, когда все буквы приобретают вид заглавных. Значения свойства применяются ко всем элементам, наследуется, не анимируется. Значение по умолчанию - normal.

  • normal - обычный текст без изменений.
  • small-caps - все буквы отображаются в верхнем регистре, но размер прописных букв соответствует размеру строчных.
  • all-small-caps - все буквы отображаются в верхнем регистре одинакового размера.
  • petite-caps - все буквы отображаются прописными, но размер прописных букв немного меньше обычного.
  • all-petite-caps - все буквы прописные уменьшенного размера.
  • unicase - все буквы отображаются в верхнем регистре, но размер прописных букв соответствует строчным.
  • titling-caps - первая буква каждого слова прописная, остальные строчные.
.elem {
  font-variant-caps:normal;          /* Text текст ТексТ */
  font-variant-caps:small-caps;      /* Text текст ТексТ */
  font-variant-caps:all-small-caps;  /* Text текст ТексТ */
  font-variant-caps:petite-caps;     /* Text текст ТексТ */
  font-variant-caps:all-petite-caps; /* Text текст */
  font-variant-caps:unicase;         /* Text текст ТексТ */
  font-variant-caps:titling-caps;    /* Text текст ТексТ */
}
