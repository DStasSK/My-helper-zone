CSS свойство user-select

CSS свойство user-select управляет поведением выделения текста и других элементов на странице, в частности, позволяет запретить выделение текста. Обычно применяется для интерактивных элементов, на которые можно щёлкать, например, вкладки, и для которых выделение текста нежелательно. Значение по умолчанию auto, наследуется, применяется ко всем элементам.

div {
  -webkit-user-select: auto | none | text | all | contain;
     -moz-user-select: auto | none | text | all | contain;
      -ms-user-select: auto | none | text | all | contain;
          user-select: auto | none | text | all | contain;
}
auto
  • Для редактируемых элементов значение принимается contain.
  • Для псевдо-элементов ::before и ::after вычисляемое значение - none.
  • Если у родителя значение user-select установлено как all, то для элемента оно тоже будет all.
  • Если у родителя значение user-select установлено как none, то для элемента оно тоже будет none.
  • Во всех остальных случаях принимается значение text.
none Пользователю запрещено выделять элемент.
text Пользователь может выделить текст в элементе.
all Позволяет выделить текст одним кликом внутри элемента, включая дочерние элементы, т.е. если клик или контекстный клик произошёл во вложенном элементе, будет выбрано все содержимое коренного предка с этим значением свойства.
containПозволяет выделять текст, но только внутри границ элемента.

Значение contain поддерживается только в IE.

elementАналогичен contain. Поддерживается только в Internet Explorer.
