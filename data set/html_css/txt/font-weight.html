CSS свойство - font-weight

Свойство font-weight устанавливает насыщенность шрифта. Значение указывается в виде чисел от 100 до 900 с шагом 100 или с помощью заданных ключевых слов. Значения свойства применяются ко всем элементам, наследуется, анимируется. Значение по умолчанию - normal.

Насыщенность шрифта можно задавать с помощью ключевых слов: bold - жирное начертание (700), normal - нормальное начертание (400), bolder - изменяют жирность относительно насыщенности родителя в большую сторону, lighter - изменяют жирность относительно насыщенности родителя в меньшую сторону.

.elem {
  font-weight:bold;
}
Также допустимо использовать условные единицы от 100 до 900 с шагом 100:
  • 100 - тонкое начертание;
  • 200 - сверхсветлое;
  • 300 - светлое;
  • 400 - нормальное (аналогично normal);
  • 500 - среднее;
  • 600 - полужирное;
  • 700 - жирное (аналогично bold);
  • 800 - сверхжирное;
  • 900 - тяжёлое.

Необходимо также учитывать, что не все шрифты поддерживают этот набор от 100 до 900. Если указанное значение не поддерживается, то браузер приведёт шрифт к ближайшей насыщенности. К примеру, если вы указали 900 и оно не может быть показано, то браузер в действительности применит значение 700 как ближайшее, которое работает корректно.
