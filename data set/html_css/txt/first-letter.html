:first-letter

Псевдоэлемент :first-letter устанавливает характер отображения первого символа участка текста. Первая буква может оформляться в соответствии с установленными атрибутами, связанными со свойствами шрифта.

К этому псевдоэлементу могут применяться только стилевые свойства, связанные со свойствами шрифта, полями, отступами, границами, цветом и фоном. Hе работает на элементах со значением display:inline;. Ставим display:block или display:inline-block;

первая буква будет выделена
HTML 
<p>первая буква будет выделена</p>
CSS 
p {font-size: 1em;}
p:first-letter {font-size: 2em; color: #f00;}
