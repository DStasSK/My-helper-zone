:first-letter

Псевдоэлемент :first-letter устанавливает характер отображения первого символа участка текста. Первая буква может оформляться в соответствии с установленными атрибутами, связанными со свойствами шрифта. К этому псевдоэлементу могут применяться только стилевые свойства, связанные со свойствами шрифта, полями, отступами, границами, цветом и фоном.

Hе работает на элементах со значением display:inline;.
Ставим display:block или display:inline-block;
К псевдоэлементу:first-letter применяется ограниченное количество свойств:
  • float
  • width (только к элементу c float)
  • margin
  • padding
  • background
  • border
  • vertical-align (только к элементу без float)
  • line-height
  • text-decoration
  • text-transform
  • letter-spacing
  • word-spacing
  • font properties
  • color

HTML 
<div class="fl">первая буква будет выделена</div>
<p class="fl">первая буква будет выделена</p>
<span class="fl">первая буква не будет выделена</span>
CSS 
fl:first-letter {font-size: 2em; color: red;}
