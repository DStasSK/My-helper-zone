text-aling

text-aling - определяет горизонтальное выравнивание текста в пределах элемента. Применяется к блочным контейнерам, не анимируется, значение по умолчанию left.

div {
  text-align:left;     /* выравнивание текста по левому краю */
  text-align:right;    /* выравнивание текста по правому краю */
  text-align:center;   /* расположение текста по центру */
  text-align:justify;  /* выравнивание по левому и правому краю одновременно*/
  text-align:start;
  text-align:end;
}

start - aналогично значению left, если текст идёт слева направо и right, когда текст идёт справа налево.

text-aling-last

Задаёт выравнивание последней строки блока текста.

div {
  -moz-text-align-last:left;
  text-align-last:left;
}
autoСовпадает с выравниванием, заданным свойством text-align, за исключением значения justify. Для него выравнивание будет как start.
startСтрока выравнивается по начальному краю блока, который может меняться в зависимости от направления текста (слева направо или справа налево).
endСтрока выравнивается по конечному краю блока, он определяется направлением текста.
leftСтрока выравнивается по левому краю.
rightСтрока выравнивается по правому краю.
centerСтрока выравнивается по центру.
justifyСтрока выравнивается по ширине. Если в последней строке только одно слово, то оно будет выравнено по левому краю.
