CSS свойство - font-size

Свойство font-size определяет размер шрифта элемента. В качестве значений свойства разрешается использовать любые допустимые единицы CSS, определяющие размер: px, em, ex, pt, проценты (За 100% берётся размер шрифта родительского элемента) и другие. Значения свойства применяются ко всем элементам, наследуется, анимируется. Значение по умолчанию - medium.

Отрицательные значения не допускаются.
.elem {
  font-size: 150%;
  font-size: 30px;
  font-size: 11pt;
  font-size: 0.5em;
}

Также допускается задавать размер шрифта при помощи ключевых слов (xx-small, x-small, small, medium, large, x-large, xx-large), но итоговый размер шрифта зависит от настроек браузера и операционной системы.
