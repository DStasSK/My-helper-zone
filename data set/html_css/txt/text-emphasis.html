text-emphasis

Сокращённое свойство, позволяющее установить значения свойств text-emphasis-style и text-emphasis-color одновременно. Устанавливает маркеры над каждым символом текста. Такие маркеры обычно используются в азиатских языках чтобы показать, как произносится слово. Для русского языка аналогом является знак ударения, который ставится над гласной буквой.

Edge, Chrome, Opera, Android поддерживают свойство -webkit-text-emphasis, IE и Hrome - не поддерживают.
word
<div>w<span>o</span>rd</div>
 
span {
  text-emphasis:"@" red;
  -webkit-text-emphasis:"@" red;
}

text-emphasis-style

Устанавливает вид маркеров над символом текста.

.elem {
  text-emphasis-style: none | значение
}
В некоторых браузерах поддерживается альтернативное свойство -webkit-text-emphasis-style
noneМаркеры не выводятся.слово
filledМаркеры будут залиты однотонным цветом.слово
openМаркеры будут пустыми внутри.слово
dotТочка.слово
circleКружок.слово
double-circleДвойной кружок.слово
triangleТреугольник.слово
sesameАпостроф.слово
"строка"Произвольный символ, пишется в кавычках.слово
Значения filled и open можно комбинировать с другими значениями, к примеру, triangle. В этом случае мы получим сплошной или контурный треугольник.

text-emphasis-position

Устанавливает положение маркеров, отображаемых с помощью свойства text-emphasis-style. Значение по умолчанию over right.
  • over - маркеры выводятся над текстом при его горизонтальном написании.
  • under - маркеры выводятся под текстом при его горизонтальном написании.
  • right - маркеры выводятся справа от текста при его вертикальном написании.
  • left - маркеры выводятся слева от текста при его вертикальном написании.
Edge, Chrome, Opera, Android поддерживают свойство -webkit-text-emphasis-position.
span {
  text-emphasis-style: triangle;
  -webkit-text-emphasis-style: triangle;
  text-emphasis-position: over;
  -webkit-text-emphasis-position: over;
}
Браузеры не поддерживают значения left и right.

text-emphasis-color

Устанавливает цвет маркеров, отображаемых с помощью свойства text-emphasis-style.
word
<div>w<span>o</span>rd</div>
 
span {
  text-emphasis-style: triangle;
  -webkit-text-emphasis-style: triangle;
  text-emphasis-color: #008000;
  -webkit-text-emphasis-color: #008000;
}
Edge, Chrome, Opera, Android поддерживают свойство -webkit-text-emphasis-position.
