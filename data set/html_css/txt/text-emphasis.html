text-emphasis

Устанавливает маркеры над каждым символом текста. Такие маркеры обычно используются в азиатских языках чтобы показать, как произносится слово. Для русского языка аналогом является знак ударения, который ставится над гласной буквой.

Edge, Chrome, Opera, Android поддерживают свойство -webkit-text-emphasis, IE и Hrome - не поддерживают.

text-emphasis-style

Устанавливает вид маркеров над каждым символом текста.

text-emphasis-style: none | значение

span {
  text-emphasis-style: double-circle;
  -webkit-text-emphasis-style: double-circle;
}
noneМаркеры не выводятся.слово
filledМаркеры будут залиты однотонным цветом.слово
openМаркеры будут пустыми внутри.слово
dotТочка.слово
circleКружок.слово
double-circleДвойной кружок.слово
triangleТреугольник.слово
sesameАпостроф.слово
"строка"Произвольный символ, пишется в кавычках.слово

Значения filled и open можно комбинировать с другими значениями, к примеру, triangle. В этом случае мы получим сплошной или контурный треугольник.

text-emphasis-position

Устанавливает положение маркеров, отображаемых с помощью свойства text-emphasis-style.

Значения:
overМаркеры выводятся над текстом при его горизонтальном написании.
underМаркеры выводятся под текстом при его горизонтальном написании.
rightМаркеры выводятся справа от текста при его вертикальном написании.
leftМаркеры выводятся слева от текста при его вертикальном написании.
span {
  text-emphasis-style: triangle;
  -webkit-text-emphasis-style: triangle;
  text-emphasis-position: over;
  -webkit-text-emphasis-position: over;
}

Edge, Chrome, Opera, Android поддерживают свойство -webkit-text-emphasis-position.

Браузеры не поддерживают значения left и right.

text-emphasis-color

Устанавливает цвет маркеров, отображаемых с помощью свойства text-emphasis-style.

word
span {
  text-emphasis-style: triangle;
  -webkit-text-emphasis-style: triangle;
  text-emphasis-color: #008000;
  -webkit-text-emphasis-color: #008000;
}

Edge, Chrome, Opera, Android поддерживают свойство -webkit-text-emphasis-position.

text-emphasis

Сокращённое свойство, позволяющее установить значения свойств text-emphasis-style и text-emphasis-color одновременно.

word
span {
  text-emphasis:"@" red;
  -webkit-text-emphasis:"@" red;
}

Если одно из двух значений отсутствует, то принимается значение по умолчанию.
