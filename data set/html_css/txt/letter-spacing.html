letter-spacing

Свойство letter-spacing определяет межбуквенное расстояние в тексте. Значение по умолчанию normal, наследуется, анимируется. Как правило, браузеры выводят символы с интервалом, установленным по умолчанию, характерным для вида и типа, а так же других индивидуальных параметров шрифта.

.elem {
  letter-spacing: 2px;
  letter-spacing: 1em;
  letter-spacing: inherit;
  letter-spacing: normal;
}
px – расстояние в пикселях
em – расстояние в em (1em - размер шрифта в px)
inherit – наследует родительское значение
normal – устанавливает стандартный интервал шрифта

Свойства letter-spacing и text-align плохо сочетаются вместе, потому что пустой промежуток добавляется справа к каждой букве. Чтобы компенсировать это, когда указанные свойства используются совместно, добавляют padding-left со значением, соответствующим letter-spacing (в данном случае 0.5em). В данном примере у первой строчки расстояние слева и справа отличается, это заметно при уменьшении ширины страницы.

CSS
.pr1 {text-align:center; letter-spacing:.5em}
.pr2 {padding-left:.5em}
