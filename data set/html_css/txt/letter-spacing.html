letter-spacing

С помощью CSS можно задать интервал между символами в пределах выбранного элемента. Как правило, браузеры выводят символы с интервалом, установленным по умолчанию, характерным для вида и типа, а так же других индивидуальных параметров шрифта. Значение по умолчанию normal, наследуется, анимируется

letter-spacing: 2px;
letter-spacing: 1em;
letter-spacing: inherit;
letter-spacing: normal;
px – расстояние в пикселях
em – расстояние в em (1em; = размеру шрифта в px;)
inherit – наследует родительское значение
normal – устанавливает стандартный интервал шрифта

Свойства letter-spacing и text-align плохо сочетаются вместе, потому что пустой промежуток добавляется справа к каждой букве. Чтобы компенсировать это, когда указанные свойства используются совместно, добавляют padding-left со значением, соответствующим letter-spacing (в данном случае 0.5em).

CSS
.pr1 {
   background-color:#9d223c;
   margin:3px 0; padding:.2em 0;
   color:#fff; font-weight:300; font-size:1.4em;
   text-transform:uppercase; text-align:center;
   letter-spacing:.5em;}
.pr2 {padding-left:.5em;}

HTML
<div class=".pr2">пример letter-spacing</div>
<div class=".pr2 .pr2">пример letter-spacing</div>
