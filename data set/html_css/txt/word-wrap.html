word-wrap

Свойство word-wrap указывает, переносить или нет длинные слова, которые не помещаются по ширине в заданную область. Значение по умолчанию normal, наследуется, применяется ко всем элементам.

.elem {
  word-wrap: normal | break-word
}
normal Строки не переносятся или переносятся в тех местах, где явно задан перенос (например, с помощью br, wbr, &shy;).
break-word Перенос строк добавляется автоматически, чтобы слово поместилось в заданную ширину блока.
