Сопоставление логических и физических свойств CSS.

В зависимости от языка, для которого разрабатывается сайт, поток сайта - его блочная ось (определяемая свойством writing-mode) в большинстве случаев будет идти сверху вниз. Но для некоторых языков блочная ось может идти справа налево (японский) или даже слева направо (монгольский). В обоих случаях у нас будет горизонтальная прокрутка, а не вертикальная, при этом конструкция сайта может быть нарушена.

Для сохранения структуры сайта при использовании на разных языках были введены некоторые дублирующие свойства, аналогичные имеющимся.

На сайтах, написанных под английский (или русский) языки, физические и логические свойства будут работать одинаково. Для сайтов на других языках значения сокращений, как в примере с margin, должны работать в соответствии со значением свойства direction или writing-mode.
