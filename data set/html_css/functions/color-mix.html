Функция color-mix()

Функция color-mix() принимает два значения color и возвращает результат их смешивания по указанному алгоритму с заданным процентным соотношением.

div {
  property: color-mix(method, color [percent], color [percent]);
}
  • method - указывает, какой метод следует использовать для смешивания цветов. Он состоит из ключевого слова in, за которым следует сам метод смешивания.
  • color - значение цвета в css формате; transparent допускается как значение.
  • percent - значение от 0% до 100%, указыват % при смешивании.
    • если данные параметры пропущены, то они считаются как 50%.
    • если указан только у одного цвета, то второй будет (100% - percent).
    • если указаны оба значения как 0% - то функция игнорируется.
    • если сумма % < 100%, то к полученному цвету применяется альфа-канал (100% - % - %).
    • если сумма % > 100%, то браузер пропорционально "нормализует" сумму до 100%.
Мметоды для смешивания.
  • srgb - цветовое пространство RGBA.
  • srgb-linear - цветовое пространство RGB.
  • display-p3 - цветовое пространство определенное Apple, объединяет цветовую гамму DCI-P3, точку белого D65 и кривую гаммы sRGB
  • a98-rgb - цветовое пространство Adobe® 1998 RGB, для представления всех цветов CMYK как RGB.
  • prophoto-rgb - цветовое пространство от Kodak, может представлять все цвета, которые могут встречаться в природе, и около 90% цветов CIElab.
  • rec2020 - это стандарт вещательной отрасли для телевизоров сверхвысокой четкости 4k и 8k. Способно отображать практически все видимые цвета реального мира, выходящие за рамки возможностей большинства дисплеев.
  • lab - цветовое пространство CIELAB (или CIELab), также называемое L*a*b* (или Lab* для краткости), представляет весь диапазон цветов, которые может видеть человек. Оно выражает цвет в виде трех значений для четырех уникальных цветов человеческого зрения: красного, зеленого, синего и желтого.
  • oklab - цветовое пространство CIELab с белой точкой — D65.
  • xyz - цветовые пространства определенными количественными связями между распределениями длин волн в электромагнитном видимом спектре и воспринимаемыми цветами в человеческом зрении. Хотя комбинации красного, зеленого и синего хорошо подходят для представления цветов на экране, sRGB не соответствует напрямую тому, как люди воспринимают цвет.
  • xyz-d50 - цветовое пространство xyz с белой точкой — D50.
  • xyz-d65 - цветовое пространство xyz с белой точкой — D65.
  • hsl - выражает цвет в формате HSL.
  • hwb - выражает цвет в формате HWB.
  • lch - выражает заданный цвет в формате LCH.
  • oklch - выражает заданный цвет в цветовом пространстве Oklab.
div {
  background-color: color-mix(in oklab, #a71e14 75%, white);
  background-color: color-mix(in srgb, var(--base) 25%, transparent);
  color: color-mix(in lab, plum 60%, #f00 50%);
  color: color-mix(in hsl, hsl(200 50 80), coral 80%);
}
