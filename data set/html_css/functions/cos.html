Функция cos()

Функция CSS cos() — это тригонометрическая функция, которая возвращает косинус указанного в ней угла в виде числа от -1 до 1.

cos(angle)

cos(45deg)
cos(0.125turn)
cos(3.14159 / 4)
angle - любое CSS значение угла; при указании безразмерных чисел, например при расчете, они интерпретируются как число радиан, представляющее угол. Если угол задан как infinity, -infinity или NaN, результат будет NaN.
