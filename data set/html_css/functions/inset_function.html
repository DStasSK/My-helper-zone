Функция inset()

Функция inset() — определяет прямоугольник на указанных расстояниях отступа от каждой стороны своего родителя. Положение прямоугольника и его размеры определяются с помощью одного, двух, трёх или четырёх параметров функции разделенных пробелами - аналогично свойству padding. Эта функция используется в CSS свойствах clip-path и shape-outside.

inset( length [length length length] [round radius] )
  • length - от одного до четырех параметров в виде длинн CSS формата, включая проценты, определяют положение прямоугольника аналогично свойству padding (верхнее, правое, нижнее и левое смещения).
  • round - ключевое слово для определения радиусов.
  • radius - значение из css свойства border-radius.
Если параметры функции заданы в процентах, то они считаются от ширины или высоты родителя, при этом сумма процентов по вертикали и горизонтали не должна превышать 100%. Если сумма превышает это значение, то размеры будут уменьшены пропорционально, чтобы получилось 100%.
