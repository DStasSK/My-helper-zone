Функция xywh()

Функция xywh() — позволяет создать прямоугольник на основе указанных расстояний от левого и верхнего краёв области а также его ширины и высоты. Положение и размеры прямоугольника определяются с помощью четырёх параметров функции. Также дополнительным параметром является ключевое слово round с параметром border-radius для добавления скругленных углов к прямоугольнику.

Функцию xywh() можно использовать в свойствах CSS, таких как offset-path, для создания прямоугольного пути, по которому перемещается элемент, и в clip-path для определения формы области отсечения.

xywh( left top width height [round radius] )

offset-path: xywh(0 1% 2px 3% round 0 1px 2% 3px);
clip-path: xywh(1px 2% 3px 4em round 0 1% 2px 3em);
  • left - расстояние от родителя до левой границы.
  • top - расстояние от родителя до верхней границы.
  • width - ширина прямоугольника.
  • height - высота прямоугольника.
  • round - ключевое слово для добавления скругленных углов.
  • radius - до четырех параметров border-radius.
Браузеры Edge, Chrome и Opera поддерживают функцию xywh() только для свойства offset-path.
