Функция polygon()

Функция polygon() принимает в качестве значений массив точек-коодинат по осям x и y, разделенных запятыми, для создания сложной фигуры с ломанным контуром. Сами координаты X и Y разделяются пробелом. Наименьшим количеством наборов точек является три, что создаёт треугольник. Эта функция используется в CSS свойствах clip-path и shape-outside.

polygon( [fill-rule,] [point], [point]... )
  • point - значение координат точки в которую или из которой будет идти линия контура фигуры. Представляет собой два значения: первое - координата по оси Х, второе - по оси Y.
  • fill-rule - необязательное значение, которое определяет правило заполнения. Значение представляется в виде одного из ключевых слов:
    • nonzero - значение по умолчанию. Заполняет все внутреенее пространство.
    • evenodd - определяет внутреннюю область на холсте, проводя линию из этой области через всю фигуру в любом направлении, подсчитывая сегменты контура, пересекаемых линией. Если при этом получится нечетное число, то точка находится внутри, если четное — то снаружи. Направление рисования пересекаемых линий фигуры - не имеет значения.

.triangle {
  width: 200px;
  height: 150px;
  background-color: magenta;
  clip-path: polygon(0% 0%, 50% 50%, 0% 100%);
}
