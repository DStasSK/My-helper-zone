Функция acos()

Функция CSS acos() — это тригонометрическая функция, которая возвращает арккосинус числа от -1 до 1 - это число радиан, представляющих угол от 0 до 180 градусов. Если нужны градусы а не радианы, то возвращаемое функцией число нужно умножить на 180 и разделить на pi (3.14159...).

acos(number)
number - любое число или расчет который даст результат в диапазоне от -1 до 1. Если number < -1 или number > 1, результат — NaN. Если число равно 1, то результат — 0.
