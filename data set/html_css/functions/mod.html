Функция mod()

Функция mod() — бинарная операция «взятия остатка» - возвращает число, которое остаётся после того, как одно целое число делится на другое с учетом знаков. Возвращаемый результат всегда принимает знак делителя. В качестве значений допустимо указывать целые числа, проценты и любые допустимые единицы CSS, при этом делимое и делитель должны быть одного типа.

mod(a, b) => {a - ( Math.floor(a / b) * b )}   /* Math.floor - округление вниз */
rem(a, b) => {a - ( Math.trunc(a / b) * b )}   /* Math.trunc - без дробной части */
mod(9, 4) = 1
rem(9, 4) = 1
mod(-9, 4) = 3
rem(-9, 4) = -1
mod(9, -4) = -3
rem(9, -4) = 1
mod(-9, -4) = -1
rem(-9, -4) = -1

div {
  margin: mod(15%, 2%);       /* 1% */
  margin: mod(18px, 4px);     /* 2px */
  margin: mod(19rem, 5rem);   /* 4rem */
  margin: mod(29vmin, 6vmin); /* 5vmin */
}
Если делимое число Infinity или делитель 0, то результатом будет NaN.
