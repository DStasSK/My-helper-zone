Функция pow()

Функция CSS pow() — принимает два значения, разделенныные запятой, и возвращает рузультат возведения первого числа в степень, значением которой является второе число.

pow(base, number) = basenumber

div {
  width: calc(10px * pow(5, 2));   /* 10px * 25 = 250px */
  width: calc(10px * pow(5, 3));   /* 10px * 125 = 1250px */
  width: calc(10px * pow(2, 10));  /* 10px * 1024 = 10240px */
}
  • base - число-основание для возведение в степень.
  • number - число-показатель степени.

Функция CSS pow() работает только с числами, для использования с единицами CSS, такими как px, em, rem и др. можно воспользоваться функцией calc():

.elem {
  font-size: calc(1rem * pow(4, 0.5));   /* 1rem × 2 = 2rem */
}
