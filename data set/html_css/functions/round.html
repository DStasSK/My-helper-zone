Функция round()

Функция round() — возвращает округленное число на основе выбранной стратегии округления. В качестве аргументов указываются: необязательная стратегия округления, округляемое значение (или математическое выражение) и интервал округления (или математическое выражение). Значение округляемое значение округляется в соответствии со стратегией округления до ближайшего целого числа, кратного интервалу округления.

round(strategy, valueToRound, roundingInterval)
  • strategy - стратегия округления.
    • nearest - стратегия округления по умолчанию - математическое округление до ближайшего целого кратного roundingInterval. Эквивалентно JS Math.round( ).
    • up - округление вверх до ближайшего целого кратного roundingInterval (если значение отрицательное, оно станет «более положительным»). Это эквивалентно методу JS Math.ceil( ).
    • down - округление вниз до ближайшего целого кратного roundingInterval (если значение отрицательное, оно станет «более отрицательным»). Это эквивалентно методу JS Math.floor( ).
    • to-zero - округление до ближайшего целого кратного roundingInterval ближе к нулю (положительное число уменьшится, а отрицательное станет «менее отрицательным»). Это эквивалентно методу JS Math.trunc( ).
  • valueToRound - округляемое значение или рассчет.
  • roundingInterval - интервал округления - указвается число или математическое выражение.
dvi {
  width: round(12vw, 50px);
  width: round(up, 101px, var(--interval));
  width: round(down, var(--height), var(--interval));
  width: round(to-zero, -105px, 10px);
}
