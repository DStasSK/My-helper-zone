Функция hypot()

Функция CSS hypot() — возвращает значение квадратного корня из суммы квадратов указанных чисел. Функция hypot() от одного и более аргументов, разделенных запятой, с одинаковым типом данных (числа, px, em и пр. смешивать нельзя). Функция вернёт значение в тех же единицах, что и её параметры.

hypot(number, number, ...)

div {
  width: hypot(2em);           /* 2em */
  width: hypot(3em, 4em);      /* 5em */
  width: hypot(30px, 40px);    /* 50px */
  width: hypot(48px, 64px);    /* 80px */
  width: hypot(3px, 4px, 5px); /* 7.0710678118654755px */
}
