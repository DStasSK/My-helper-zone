Функция sqrt()

Функция CSS sqrt() — возвращает квадратный корень заданного числа. В качестве значения применяется число больше или равное нулю (x ≥ 0). Если указанное число отрицательное (например: sqrt(-4)), то функция вернёт NaN.

Функция sqrt() работает только с числами без единиц CSS. Для использования с единицами CSS, такими как px, em, rem и др. можно воспользоваться функцией calc().
sqrt(number)

div {
  width: calc(10px * sqrt(9));   /*  30px */
  width: calc(10px * sqrt(25));  /*  50px */
  width: calc(10px * sqrt(100)); /* 100px */
}
