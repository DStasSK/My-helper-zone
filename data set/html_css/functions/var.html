Функция var()

Функция CSS var() используют для вставки пользовательского значения свойства (иногда называемого «переменной CSS») вместо значения другого свойства. Функцию var() нельзя использовать в именах свойств, селекторов или чем-либо еще, кроме значений свойств.

var( propertyName [, value] )
  • propertyName - имя пользовательской переменной.
  • value - резервное значение. Синтаксис резервного значения допускает запятые, то есть все, что находится между первой запятой и концом функции, считается резервным значением.
:root {
  --bg-color: pink;
}

body {
  background-color: var(--bg-color, #fafafa);
}
Функция var() может вернуть недопустимое значение, если:
  • Пользовательское свойство не определено и не указано резервное значение.
  • Пользовательское свойство определено, но его значение является недопустимым значением для свойства, в котором оно используется.

Пользовательские переменные.

Пользовательские переменные и их значения описываются внутри произвольного селектора и должны начинаться с двух дефисов. Через двоеточие переменной присваивается любое допустимое для CSS значение. При этом область видимости переменной ограничена указанным селектором и применение переменной в другом селекторе не даст результата, но все потомки данного селектора будут иметь доступ к переменным родителя.

Для задания имени переменной допускается использовать буквы, цифры, знаки подчёркивания и дефис. Переменные CSS чувствительны к регистру.

Для создания глобальных переменных их следует описать внутри селектора :root.
:root {
  --bg-color: pink;
  --bg_gradient: linear-gradient(to top, #a0f, #fff);
}
Пример использования: переменные определены непосредственно у img элеметнов HTML, а расчет размеров изображений ведется для одного класса. При этом полученные изображения с одним классом имеют разные размеры.
<div class="example_img">
  <div><img src="img/winnie.png" class="img" style="--size: 3"></div>
  <div><img src="img/winnie.png" class="img" style="--size: 2"></div>
  <div><img src="img/winnie.png" class="img" style="--size: 1"></div>
</div>

.example_img {
  display: flex;
  gap: 10px;
}
.example_img div { width: calc( (100% - 20px) / 3 ); }
.example_img .img {
  max-width: 100%;
  width: calc( 1 / var(--size, 1) * 100% );
}

Так же переменные можно использовать если есть смного объектов с однотипной анимацией: для каждого класса объекта задаются свои переменные, которые будут работать как значения свойств в одном @keyframes, которого будет достаточно для всех объектов.

При использовании переменной CSS внутри правила @keyframes переменная не может быть анимирована, т.е. анимация будет работать некорректно, если внутри @keyframes будут задаваться переменные.
Переменные нужно задавать вне @keyframes и использовать из как значения свойств.
