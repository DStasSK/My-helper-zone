Функция exp()

Функция CSS exp() — это экспоненциальная функция, которая принимает число в качестве аргумента и возвращает математическую константу e, возведенную в степень указанного числа.

exp(number) = enumber
number - число в степень которого возводится константа e.
  • eсли указать -Infinity результатом будет 0.
  • если указать Infinity результатом будет Infinity.
  • если указать 0 результатом будет 1.
  • если указать 1 результатом будет число e (примерно 2.71828).
Математическая константа e является основанием натуральных логарифмов и приблизительно равна 2,718281828459045.
