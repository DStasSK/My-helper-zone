Функция rect()

Функция rect() — позволяет создать прямоугольник на основе указанных расстояний от левого и верхнего краёв области. Положение и размеры прямоугольника определяются с помощью четырёх параметров функции, которые, соответственно, задают верхнюю, правую, нижнюю и левую стороны прямоугольника. Также дополнительным параметром является ключевое слово round с параметром border-radius для добавления скругленных углов к прямоугольнику.

Функцию rect() можно использовать в свойствах CSS, таких как offset-path, для создания прямоугольного пути, по которому перемещается элемент, и в clip-path для определения формы области отсечения.

rect( top|auto right|auto bottom|auto left|auto round radius )

offset-path: rect(50px auto 200px 50px round 20%);
offset-path: rect(50px auto 200px auto);
  • top - расстояние от родителя до верхней границы.
  • right - расстояние от родителя до правой границы.
  • bottom - расстояние от родителя до нижней границы.
  • left - расстояние от родителя до левой границы.
  • auto - ключевое слово, при котором соответствующее расстояние равно 0.
  • round - ключевое слово для добавления скругленных углов.
  • radius - до четырех параметров border-radius.
Браузеры Edge, Chrome и Opera поддерживают функцию rect() только для свойства offset-path.
