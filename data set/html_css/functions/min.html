Функция min()

Функция CSS min() — позволяет установить наименьшее (самое отрицательное) значение из списка (от одного и более) значений/выражений, разделенных запятыми. В качестве аргументов принимает параметры CSS одинаковой размерности.

div {
  width: min( value, value, [value,] ... );
}

Выражение может быть значениями, объединяющими операторы сложения ( + ), вычитания ( - ), умножения ( * ) и деления ( / ), используя стандартные правила приоритета операторов. Обязательно поставьте пробел с каждой стороны операндов + и - .

.block {
  width: min(100%, 1024px);
}

/* или */

.block {
   width: 100%;
   max-width: 1024px;
}
