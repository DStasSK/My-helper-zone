Функция image()

Функция image() — определяет изображение аналогично функции url(), но с дополнительной функциональностью, включая указание направленности изображения, отображение только части этого изображения, определенной фрагментом мультимедиа, и указание сплошного цвета в качестве резервного варианта на случай, если ни одно из указанных изображений не может быть отображено.

image( [ltr | rtl]  [url() | src]  [, color] )


div {
  background-image: image(rgb(0 0 0 / 25%)), url("bg.png");
  list-style-image: image(ltr "rightarrow.png");
  cursor: image("sprite.png#xywh=32,64,16,16");
}
  • ltr | rtl - направленность изображения: ltr — слева направо, rtl — справа налево. Если направленность опущена, изображение не будет перевернуто при изменении направления текста.
  • url() | src - ноль или более источников изображений, с необязательными размерами фрагментов изображений.
  • color - запасный вариант цвета в формате CSS.

При указании адрса изображения через src есть возможность указать фрагмент изображения через начальную точку по осям x и y, а также ширину и высоту — это отобразит только часть исходного изображения. Часть изображения, определенная в параметре, становится отдельным изображением. Синтаксис выглядит следующим образом:

div {
  background-image: image("my-image.webp#xywh=0,20,40,60");
/* Фоновое изображение элемента будет частью изображения myImage.webp
   которая начинается с координат 0px, 20px (верхний левый угол)
   и имеет ширину 40px и высоту 60px. */
}

Синтаксис фрагмента медиа #xywh=#,#,#,# принимает четыре числовых значения, разделенных запятыми. Первые два представляют координаты X и Y для начальной точки создаваемого блока. Третье значение — ширина блока, а последнее значение — высота. По умолчанию это значения в пикселях, но координаты можно задать и в процентах.

div {
  background-image: image("my-image.webp#xywh=160,120,320,240");
  background-image: image("my-image.webp#xywh=pixel:160,120,320,240");
  background-image: image("my-image.webp#xywh=percent:25,25,50,50");
}
/* xywh=160,120,320,240        размеры 320x240, начало в x=160 and y=120
   xywh=pixel:160,120,320,240  размеры 320x240, начало в x=160 and y=120
   xywh=percent:25,25,50,50    размеры 50%x50%, начало в x=25% and y=25% */
Если браузер не понимает нотацию фрагментов медиа, он игнорирует фрагмент, отображая всё изображение или бдет отображен запасный цвет.
Если изображение содержит информацию, критическую для понимания общего назначения страницы, лучше описать ее семантически в документе. Это будет полездно для экранных ридеров.
