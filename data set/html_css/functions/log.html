Функция log()

Функция CSS log() — возвращает логарифм указанного числа по указанному основанию. Логарифм представляет собой показатель степени, в которую необходимо возвести основание логарифма (второй параметр), чтобы получить число из первого параметра. Функция log() в качестве значений принимает два числа. Если указан один параметр, то в качестве основания используется константа e (2,7182818...) - получается натуральный логарифм.

pow(value, base)

div {
  width: calc(100px * log(7.389));  /* 200px */
  width: calc(100px * log(8, 2));   /* 300px */
  width: calc(100px * log(625, 5)); /* 400px */
}
  • value - число для расчета логарифма.
  • base - число-основание для расчета логарифма.
