В CSS можно создавать свои счетчики, которые будут считать количество отображений определенного элемента. Значение такого счётчика может выводиться с помощью свойства content и псевдоэлементов ::after и ::before.

В основном такие счетчики используются при создании списков (в том числе многоуровневых), в которых нумерация и вид задаются через стили. В нумерованном списке номер каждого пункта выводится с точкой. При этом как-то изменить вид числа, к примеру, убрать точку или поставить скобку, напрямую нельзя. Поэтому вообще убирают исходную нумерацию и выводят свою с помощью псевдоэлемента ::before и свойства content. Чтобы нумерация работала и увеличивалась с каждым пунктом, нужно добавить свойство counter-reset с произвольным именем счётчика для его инициализации и counter-increment для приращения счётчика.

Название/идентификатор счетчика чувствительным к регистру. Идентификатор счетчика не может начинаться с двух дефисов и не может быть none, unset, initial или inherit.

CSS свойство counter-reset.

CSS свойство counter-reset устанавливает переменную, в которой будет храниться счётчик отображений определенного элемента, а также начальное значение счётчика. В качестве значений принимает список из одного или нескольких имен счетчиков, разделенных пробелами, или ключевое слово none.

ol {
  counter-reset: name integer | reversed(name) integer | none;
  counter-reset: inherit | initial | revert | revert-layer | unset;
}
  • name - имя/идентификатор счетчика.
  • integer - начальное значение счетчика - целое число. По умолчанию 0.
  • reversed(name) - функциональная нотация для обращенного счетчика. При создании обратных счетчиков без значения счетчик начнет со значения, равного количеству элементов в наборе, отсчитывая в обратном порядке.
    Поддерживается только в FF.
  • none - отключает счетчик; значение счетчика удаляется.
Установка counter-increment:none для селектора с большей специфичностью переопределяет создание именованного набора счетчиков для селекторов с меньшей специфичностью.

CSS свойство counter-set.

CSS свойство counter-set устанавливает значение счетчика в указанное значение. Если счетчики не существуют, свойство counter-set создает новый счетчик для каждого именованного счетчика в списке пар счетчика и значения, разделенных пробелом. Если значение свойства не указано, значение счетчика будет установлено на 0.

ol {
  counter-set: name integer | none;
}
  • name - имя/идентификатор счетчика.
  • integer - устанавливаемое значение счетчика - целое число, включая отрицательные. По умолчанию 0.
  • none - отключает счетчик; значение счетчика удаляется.

CSS свойство counter-increment.

CSS свойство counter-increment использоваться для увеличения или уменьшения значения именованных счетчиков CSS на указанные значения. Если именованный счетчик в списке разделенных пробелами счетчиков и значений не существует, он будет создан. Если в списке счетчиков не указано значение для счетчика, счетчик будет увеличен на 1.

ol {
  counter-increment: name integer | none;
}
  • name - имя/идентификатор счетчика.
  • integer - величина приращения счетчика - целое число, включая отрицательные и 0. По умолчанию 1.
  • none - отключает изменения значений всех счетчиков. Чтобы предотвратить изменение значения определенных счетчиков, установите целочисленное значение 0 для соответствующего счетчика(ов).
ol {
  counter-reset: list -1;  /* Список начинается с нуля. */
  counter-reset: list 9;   /* Список начинается с 10 */
}
li::before {
  counter-increment: list 1;    /* Выводятся 1, 2, 3,... */
  counter-increment: list 2;    /* Выводятся 2, 4, 6,... */
  content: counter(list) ". ";

  /* изменит "page" на 1, "section" на 2, "chapter" - не меняется */
  counter-increment: chapter 0 section 2 page;
}
Для элементов, у которых установлено display: none, значение счётчика не меняется.

CSS функция counter()

CSS функция counter() возвращает текущее значение именованного счетчика, если таковой имеется, в виде строки. Функция counter() обычно используется в псевдоэлементе через свойство content, но теоретически ее можно использовать везде, где поддерживается значение типа string.

li::before {
  content: counter(name, style);
  content: counter(name, symbols( symbols-type | url() );
}
  • name - имя/идентификатор счетчика.
  • style - стиль оформления счетчика, обычно это параметр свойства list-style-type для нумерованного списка. Также можно указать имя из правила @counter-style или функцию symbols(). Если стиль оформления не указан, то применяется decimal - десятичный стиль.
  • symbols-type - cyclic | numeric | alphabetic | symbolic | fixed.
  • url() - адрес изображения.
Чтобы объединить и отобразить значения нескольких счетчиков, используют функцию counters().

CSS функция counters()

CSS функция counters() возвращает строку, которая объединяет текущие значения именованных и вложенных счетчиков, если таковые имеются, разделенные предоставленной строкой. Третий, необязательный параметр позволяет определить стиль списка. Сгенерированный текст представляет собой значение всех счетчиков с указанными идентификаторами, расположенных от самого внешнего к самому внутреннему и разделенных указанной строкой.

li::before {
  content: counters(name, string, style);
  content: counters(name, string, symbols( symbols-type | url() );
}
  • name - имя/идентификаторы счетчиков.
  • string - строка-разделитель между значениями счетчиков. Любое количество текстовых символов. Нелатинские символы должны быть закодированы с использованием их управляющих последовательностей Unicode.
  • style - стиль оформления счетчика, обычно это параметр свойства list-style-type для нумерованного списка. Также можно указать имя из правила @counter-style или функцию symbols(). Если стиль оформления не указан, то применяется decimal - десятичный стиль.
  • symbols-type - cyclic | numeric | alphabetic | symbolic | fixed.
  • url() - адрес изображения.

Пример - простой нумерованный список.

В нумерованном списке номер каждого пункта выводится с точкой. При этом как-то изменить вид числа, к примеру, убрать точку или поставить скобку, напрямую нельзя. Поэтому вообще убирают исходную нумерацию и выводят свою с помощью псевдоэлемента ::before и свойства content. Чтобы нумерация работала и увеличивалась с каждым пунктом, нужно добавить свойство counter-reset с произвольным именем счётчика для его инициализации и counter-increment для приращения счётчика.

Поскольку мы сами выводим число, то можем легко его стилизовать, изменив цвет, сделав жирным, курсивным и др.
  1. простой список с номерами
  2. нумерованный список
  3. список с нумерацией
<ol>
  <li>простой список с номерами</li>
  <li>нумерованный список</li>
  <li>список с нумерацией</li>
</ol>

ol {
  list-style-type: none;           /* Убираем исходные маркеры */
  counter-reset: myCounter;        /* Задаём имя счетчика */
}
li::before {
  content: counter(myCounter) ")"; /* Выводим число и скобку */
  counter-increment: myCounter;    /* Увеличиваем значение счётчика */
  margin-right: 10px;              /* Расстояние от цифры до текста */
}

Многоуровневый нумерованный список

Чтобы сделать вложенную нумерацию, нужно использовать следующие свойства:
  • counter-reset сбрасывает один или несколько счётчиков, задавая значение для сброса;
  • counter-increment задаёт значение приращения счётчика, т.е. с каким шагом будет нумероваться каждый последующий пункт;
  • content генерируемое содержимое, в данном случае отвечает за вывод номера перед каждым пунктом списка.

  counter-increment: li;
  content: counters(li,".") ". ";
}
