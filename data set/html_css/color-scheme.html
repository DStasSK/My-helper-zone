CSS свойство color-scheme

  • CSS свойство color-scheme
  • Стилевая функция light-dark()

Свойство color-scheme позволяет указать предпочтительную цветовую схему для элемента - светлую или тёмную. Выбранная схема влияет на цвета полей формы, полос прокрутки, а также используемые значения системных цветов. Значение по умолчанию normal, наследуется, применяется ко всем элементам, не анимируется.

.elem { color-scheme: normal | light | dark | only }
  • normal - элемент не поддерживает какие-либо цветовые схемы, поэтому его следует отображать с использованием цветовой схемы браузера по умолчанию.
  • only - запрещает браузеру переопределять цветовую схему элемента.
  • light - элемент поддерживает светлую цветовую схему.
  • dark - элемент поддерживает тёмную цветовую схему.
  • light dark - цвета элемента будут меняться в зависимости от установленной темы браузера. Для изменения цвета фона формы в зависимости от темы, можно воспользоваться медиа-запросами. 
    :root {
      color-scheme: light dark;
    }
    @media (prefers-color-scheme: light) {
      form {
        background-color: #f0f0f0; /* Светло-серый цвет фона */
      }
    }
    @media (prefers-color-scheme: dark) {
      form {
        background-color: #333; /* Тёмно-серый цвет фона */
        accent-color: #ba84fa; /* Цвет акцента */
      }
    }
Для определения цветовой схемы всей веб-страницы целиком, color-scheme добавляется к псевдоклассу :root.
:root {
  color-scheme: light dark;
}
Для указания цветовой схемы отдельных элементов, color-scheme добавляется к соответствующим селекторам.
h2 {
  color-scheme: light dark;
}

Стилевая функция light-dark()

Стилевая функция light-dark() позволяет установить два цвета в качестве значения какого-либо свойства. Возвращает одно значение, в зависимости от выбранной светлой или тёмной цветовой схемы. Принимает два значения цвета; возвращает первое значение, если установлена светлая тема и второе значение, если установлена тёмная тема.

Чтобы включить поддержку функции light-dark(), свойство color-scheme должно иметь значение light dark.
:root {
  color-scheme: light dark;
}
p {
  background: light-dark(white, black);
  color: light-dark(black, white);
}
