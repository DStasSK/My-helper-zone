accent-color

CSS свойство accent-color устанавливает цвет акцента для некоторых элементов управления пользовательским интерфейсом, таких как:

<input type="checkbox"> 

<input type="radio"> 

<input type="range"> 

<progress> 
input[type=checkbox] { accent-color: red; }
input[type=radio] { accent-color: green; }
input[type=range] { accent-color: rgb(0, 0, 255); }
progress { accent-color: hsl(39, 100%, 50%); }

Акцентом в данном случае называют яркий цвет, который контрастно выделяется на цвете фона. Акцентный цвет обычно используется для привлечения внимания пользователя и выделения важных элементов на веб-странице.
