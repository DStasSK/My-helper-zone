border-image

  • border-image-source
  • border-image-slice (пример)
  • border-image-width
  • border-image-outset (пример)
  • border-image-repeat (пример)
  • Градиентная рамка (пример)

Используется для отображения рисованной рамки вокруг элемента. Толщина рамки задаётся свойством border, при этом если указано border: 0, то рамка не выводится. При других значениях border рисунок всегда имеет приоритет. Фон, если он задан через свойство background, отображается под рамкой.

Устанавливает рамку-изображение с помощью одного свойства, являющегося краткой записью свойств border-image-source, border-image-slice, border-image-width, border-image-outset и border-image-repeat.

Значения свойств умолчанию: none 100% 1 0 stretch, не наследуется, применяется ко всем элементам, за исключением тех, у кого border-collapse задан как collapse.

.box {
 border-image:none |
              border-image-source  /* Путь к графическому файлу. */
              border-image-slice   /* Размер частей рамки, px не пишется */
              border-image-width   /* Устанавливает ширину рамки */
              border-image-outset  /* Смещение рамки-изображения */
              border-image-repeat; /* Повтор рамки-изображения */
}

/* Значения свойств по умолчанию: none 100% 1 0 stretch */
Firefox до v15.0 поддерживает свойство -moz-border-image
Safari до v6.0, Chrome до v15.0 поддерживают свойство -webkit-border-image
Opera до v15.0 поддерживает свойство -o-border-image

border-image-source

Свойство задаёт путь к изображению, которое будет использоваться для оформления границ блока. Значение по умолчанию - none.

div {border-image-source: url(img/border1.png);}

border-image-slice

Свойство определяет размер частей изображения, используемых для оформления границ элемента и делит изображение на девять частей: четыре угла, четыре края между углами и центральную часть.

div {border-image-slice: значение;}
Значения
число

Одно, два, три или четыре значения, которые указывают размеры частей изображения в пикселях, задавая тем самым области деления. Сами единицы не пишутся, только число (10, а не 10px).

Одно значение устанавливает границы одинакового размера для каждой стороны элемента.

Два значения: первое определяет размер верхней и нижней границы, второе — правой и левой.

Три значения: первое определяет размер верхней границы, второе — правой и левой, а третье — нижней границы.

Четыре значения: определяет размеры верхней, правой, нижней и левой границы.

div {
  border-image-slice:40;
  или
  border-image:url(..) 40;
}
% Аналогично числу, но значения задаются в процентах. Размеры границ рассчитываются относительно размера изображения. Горизонтальные относительно ширины, вертикальные — относительно высоты.
fill Значение указывается вместе с числом или процентным значением. Если оно задано, изображение не обрезается внутренним краем рамки, а заполняет также область внутри рамки. Если задано только fill без числа то рамка не отрисовывается. 
div {border-image-slice:fill;}     /* рамки не будет */
div {border-image-slice: 40 fill;} /* рамка и заполнение будет */
initial Устанавливает это свойство в значение по умолчанию.
inherit Наследует значение этого свойства от родительского элемента.

Пример

Данное изображение будет использано в качестве рамки. Ширина и высота изображения - 120px.

Если параметр border-image-source не задан, то рамка исказится - заполнятся только углы, см. bg-1.

Если параметр border-image-source равен или больше половине от размеры картинки, используемой в качестве рамки - то рамка будет не полная (см. bg-5 и bg-6), а также параметр fill эффекта давать не будет (см. bg-3 vs bg-6).

bg-1
border-image: url(img/border1.png); (100%)
bg-2
border-image: url(img/border1.png) 20;
bg-3
border-image: url(img/border1.png) 40 fill;
bg-4
border-image: url(img/border1.png) 59;
bg-5
border-image: url(img/border1.png) 60;
bg-6
border-image: url(img/border1.png) 100 fill;

border-image-width

Свойство задаёт ширину изображения для границы элемента. Если ширина не задана, то по умолчанию она равна 1.

Через слэш пишется одно, два, три или четыре значения толщины границы на каждой стороне элемента. Является аналогом border-width и использует тот же синтаксис.

Если задать border-image-width < border-image-slice, то от исходной картинки отрежется фрагмент согласно border-image-slice и смасштабируется в размер, заданный через border-image-width.

div {border-image-width: значение;}
Значения
размер Устанавливает ширину рамки в единицах длины - px / em. Можно задавать от одной до четырех значений одновременно. Если задано одно значение, то ширина всех сторон рамки одинакова, два значения задают ширину верхней-нижней и правой-левой и т.д.
число Числовое значение, на которое умножается значение border-width
% Ширина рамки элемента вычисляется относительно размера изображения. Горизонтальные относительно ширины, вертикальные — относительно высоты.
auto Соответствует значению border-image-slice
initial Устанавливает это свойство в значение по умолчанию.
inherit Наследует значение этого свойства от родительского элемента.

border-image-outset

Свойство позволяет переместить изображение-рамку за пределы границ элемента на указанную длину. Может задаваться как с помощью одного, так и четырёх значений.

div {border-image-outset: 10px;}
Значения
размер Отступ рамки-изображения задается с помощью любого положительного числа, указанного в px / em
число Числовое значение, на которое умножается значение border-width
initial Устанавливает это свойство в значение по умолчанию.
inherit Наследует значение этого свойства от родительского элемента.

Пример

Данное изображение будет использано в качестве рамки. Ширина и высота изображения - 120px.

Oтрицательные значения - не допускаются, параметр будет игнорироваться.

bg-1
border-image:url(img/border2.png) 40;
border-image-outset:0px;
bg-2
border-image:url(img/border2.png) 40 / 20px;
border-image-outset:-20px;
bg-3
border-image:url(img/border2.png) 20 / 0.25 / 0.25em;

border-image-repeat

Свойство управляет заполнением фоновым изображением пространства между углами рамки. Можно задавать как с помощью одного значения, так и с помощью пары значений.

div {border-image-repeat: значение;}
Значения
stretch Растягивает изображение на все пространство. Значение по умолчанию.
repeat Повторяет заполняющую часть изображения, при этом видны места стыков с угловой частью.
round Повторяет рисунок и масштабирует его так, чтобы на стороне элемента оказалось целое число изображений.
space Что-то среднее между round и stretch, при этом может образовать пустое пространство (разрывы) между частями рамки.
initial Устанавливает это свойство в значение по умолчанию.
inherit Наследует значение этого свойства от родительского элемента.

Пример

stretch
border-image:url(img/border2.png) 40;
repeat
border-image:url(img/border2.png) 40 / 20px;
round
border-image:url(img/border2.png) 20 / 0.25 / 0.25em;
space
border-image:url(img/border2.png) 40;
space
border-image:url(img/border2.png) 40 / 20px;
space
border-image:url(img/border2.png) 20 / 0.25 / 0.25em;
round
border-image:url(img/border2.png) 40;
round
border-image:url(img/border2.png) 40 / 20px;
repeat
border-image:url(img/border2.png) 20 / 0.25 / 0.25em;

Градиентная рамка

Значением border-image может выступать не только изображение, но и градиентная заливка.

div {border-image: linear-gradient(...);}

Пример

bg1
border-image: radial-gradient(#fff, #fff 50%, #d3e8c7 65%, #1255c7);
bg2
border-image: radial-gradient(#fff, #fff 50%, #d3e8c7 65%, #1255c7) 40;
bg3
border-image: radial-gradient(#fff, #fff 50%, #d3e8c7 65%, #1255c7) 40 fill;
bg4
border-image: linear-gradient(to right, transparent 0%, #ADF2F7 100%) 1 / 10px;
bg5
border-image: linear-gradient(to right, transparent 0%, #ADF2F7 100%) 1;
bg6
border-image: linear-gradient(to right, transparent 0%, #ADF2F7 100%) 1 fill / 1px;
bg7
border: 10px solid transparent;

border-image: 10 repeating-linear-gradient(45deg, #A7CECC, #A7CECC 10px, transparent 10px, transparent 20px, #F8463F 20px, #F8463F 30px, transparent 30px, transparent 40px);
bg8
border: 10px solid transparent;

border-image: repeating-linear-gradient(45deg, #A7CECC, #A7CECC 10px, transparent 10px, transparent 20px, #F8463F 20px, #F8463F 30px, transparent 30px, transparent 40px) 14/10px;
bg9
border: 10px solid transparent;
border-image: repeating-linear-gradient(45deg, #A7CECC, #A7CECC 10px, transparent 10px, transparent 20px, #F8463F 20px, #F8463F 30px, transparent 30px, transparent 40px) 1 fill/20px;
