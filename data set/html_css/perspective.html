CSS свойство perspective

  • Функция perspective()
  • CSS свойство perspective-origin

CSS свойство perspective oпределяет расстояние от плоскости экрана до точки сходимости линий и тем самым задаёт, насколько выражен эффект перспективы. Значение по умолчанию none, не наследуется, анимируется, применяется к трансформируемым элементам.

div {
  -webkit-perspective: none | size;  /* для мобильных браузеров */
  perspective: none | size;
}
  • none - значение по умолчанию, отменяет эффект 3d пространства.
  • size - расстояние от плоскости монитора до точки сходимости линий. Нулевое или отрицательное значение отменяет действие перспективы. Чем меньше значение, тем более выраженной выглядит перспектива и наоборот.
Элементы части которые находятся за пользователем, т.е. z-координата этих элементов больше чем значение свойства perspective, не отрисовываются.
Перспектива для элемента может быть задана двумя разными способами:
  • использование значения в виде функции perspective() в свойстве transform для отдельных элементов.
  • использование свойства perspective для родительского элемента, внутри которого трансформируются дочерние элементы.
div {
  -webkit-transform: perspective(none | size);
  transform: perspective(none | size);

  -webkit-perspective: none | size;
  perspective: none | size;
}

Использование значения perspective в свойстве transform прекрасно работает для преобразования одного элемента относительно единичной уникальной перспективы. Если вы желаете преобразовать группу элементов с одинаковой перспективой или точкой схода, то применяйте свойство perspective для их родительского элемента.

Эффект глубины по оси z расчитывается по формуле d / (d - z), где d - значение свойства perspective.

Точка сходимости определяет расстояние или горизонт в которой сходятся все параллейные линии, при этом создается эффект объемного изображения. Точка сходимости по умолчанию располагается в центре элемента и может быть изменена с помощью свойства perspective-origin. Чем меньше значение, тем более выраженной выглядит перспектива и наоборот.

<div>
    <img src="img/winnie.png">
</div>

div {
    transform-style: preserve-3d;
    perspective: 150px;
}
img {
    transform: rotateX(20deg);
}
perspective:150px;
 
perspective:400px;
 
perspective:2000px;

perspective-origin

CSS свойство perspective-origin задаёт координаты точки, куда смотрит наблюдатель. Это свойство работает совместно со свойством perspective и определяет точку сходимости линий при перспективе. По умолчанию точка находится в центре элемента. Значение свойства perspective-origin нужно задавать к родителю элемента, как и глубину перспективы.

div {
  -webkit-perspective-origin: позиция;
  perspective-origin: позиция;
}

Свойство принимает одно или два значения. Если указано только одно значение, то оно считается координатой X, два значения определяют вначале координату X, затем Y. Проценты задаются относительно ширины элемента, отсчёт координат начинается с его левого верхнего угла. Проценты, пиксели и другие единицы CSS могут быть отрицательными. Ключевые слова (top, left, bottom, right) могут идти в произвольном порядке.

<div class="box">
  <img src="img/winnie.png">
</div>


.box {
  perspective-origin: 0 0;
  perspective: 300px;
}
img {
  transform: rotateX(20deg);
}
perspective-origin:0 0;
 
perspective-origin:50% 0;
 
perspective-origin:100% 0;
