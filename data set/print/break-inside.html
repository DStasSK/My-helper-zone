CSS свойство break-inside

Разрешает или запрещает разрыв страницы внутри элемента при печати. Устанавливает, как разрывы страницы, столбца или региона должны вести себя внутри сгенерированного блока. Если сгенерированного блока нет, свойство игнорируется. Значение по умолчанию auto, не наследуется, применяется к блочным элементам.

.elem {
  break-inside: auto | avoid | avoid-page | avoid-column | avoid-region;
}
autoВставляет разрыв страницы при необходимости.
avoidЗапрещает разрыв страницы внутри элемента.
avoid-pageПозволяет избежать разрыва страницы внутри блока.
avoid-columnПозволяет избежать разрыва столбцов внутри блока.
avoid-regionПозволяет избежать разрыва области внутри блока.
<!DOCTYPE html>

  <head>
    <meta charset="utf-8">
    <title>break-inside</title>
    <style>
      @media print {   /* медиазапросы не поддерживется в IE8 */
        .more {break-inside: avoid;}
      }
    </style>
  </head>
