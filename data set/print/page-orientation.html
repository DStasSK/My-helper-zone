CSS свойство page-orientation.

Свойство page-orientation для правила @page управляет направлением поворота печатной страницы. Этот дескриптор помогает с макетом и ориентацией печатных документов, особенно когда документы печатаются с двух сторон. Пользователь может указать, как будут повернуты страницы при печати. ​​Это особенно полезно для макетирования контента, такого как таблицы, которые могут быть шире остального контента, в другой ориентации.

@page {
  page-orientation: upright | rotate-left | rotate-right;
}
  • upright - ориентация не изменяется.
  • rotate-left - поворачивает страницу на 90° против часовой стрелки.
  • rotate-right - поворачивает страницу на 90° по часовой стрелке.
Пример:
@media print {
  .upright {
    page: upright;
  }
  .left {
    page: left;
  }
  .right {
    page: right;
  }
}
