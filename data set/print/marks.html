marks

Отображает специальные метки на странице при печати документа, предназначенные для дальнейшего обрезания страницы или для ровного соединения нескольких листов между собой. Значение по умолчанию none, не наследуется, применяется к печатной странице.

marks: [ crop || cross ] | none
cropВыводит метки для дальнейшего обрезания страницы после её печати.
crossВыводит метки, по которым происходит совмещение с другими страницами, например, при печати большой картинки на нескольких листах.
noneНикакие метки не отображаются.
Данное свойство не поддерживается браузерами - рабочий проект.
