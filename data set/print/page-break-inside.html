page-break-inside

Разрешает или запрещает разрыв страницы внутри элемента при печати. Значение по умолчанию auto, не наследуется, применяется к блочным элементам.

page-break-inside: auto | avoid
autoВставляет разрыв страницы при необходимости.
avoidЗапрещает разрыв страницы внутри элемента.
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>page-break-inside</title>
    <style>
      @media print {   /* медиазапросы не поддерживется в IE8 */
        .more {page-break-inside: avoid;}
      }
    </style>
  </head>
  <body>
    <h2>Мусорные пакеты</h2>
    <p>...</p>
    <p class="more">Читать дальше</p>
  </body>
</html>
