page-break-after

Добавляет разрыв страницы при печати документа после заданного элемента. Значение по умолчанию auto, не наследуется, применяется к блочным элементам.

page-break-after: always | auto | avoid | left | right
alwaysВсегда добавляет разрыв страницы после элемента.
autoВставляет разрыв страницы при необходимости.
avoidЗапрещает разрыв страницы после элемента.
leftПропускает одну или две страницы после элемента, чтобы следующая страница при печати была чётной.
rightПропускает одну или две страницы после элемента, чтобы следующая страница при печати была нечётной.
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>page-break-after</title>
    <style>
      @media print {   /* медиазапросы не поддерживется в IE8 */
        .more {page-break-after: always;}
      }
    </style>
  </head>
  <body>
    <h2>Мусорные пакеты</h2>
    <p>...</p>
    <p class="more">Читать дальше</p>
  </body>
</html>
