CSS свойство break-before

Добавляет разрыв страницы при печати документа перед заданным элементом. Устанавливает, как разрывы страницы, столбца или региона должны вести себя перед сгенерированным блоком. Если сгенерированный блок отсутствует, свойство игнорируется. Значение по умолчанию auto, не наследуется, применяется к блочным элементам.

.elem {
  /* Generic break values */
  break-before: auto | avoid | always | all;

  /* Page break values */
  break-before: avoid-page | page | left | right | recto | verso;

  /* Column break values */
  break-before: avoid-column | column;

  /* Region break values */
  break-before: avoid-region | region;
}
autoВставляет разрыв страницы при необходимости.
avoidЗапрещает разрыв страницы перед элементом.
alwaysВсегда добавляет разрыв страницы перед элементом. Вызывает разрыв страницы сразу перед основновным блоком. Тип этого разрыва — это тип непосредственно содержащегося контекста фрагментации. Если мы находимся внутри мультиколоночного контейнера, то это вызовет разрыв столбца, внутри постраничного носителя (но не внутри контейнера) — разрыв страницы.
Поддерживается только в FF
allВызывает разрыв страницы перед блоком. Прорывая все возможные контексты фрагментации. Таким образом, разрыв внутри мультиколоночного контейнера вызовет разрыв столбца и страницы.

Значения разрыва страницы
avoid-pageПозволяет избежать разрыва страницы перед блоком.
Не поддерживается в FF
pageПринудительно вставляет разрыв страницы перед блоком.
leftПропускает одну или две страницы перед элементом, чтобы следующая страница при печати была чётной. Эта страница размещена на левой стороне корешка книги или на задней стороне страницы при двусторонней печати.
rightПропускает одну или две страницы перед элементом, чтобы следующая страница при печати была нечётной. Эта страница размещена на правой стороне корешка книги или на передней стороне страницы при двусторонней печати.
rectoПринудительно вставляет один или два разрыва страницы перед блоком, в зависимости от того, какой из них превратит следующую страницу в страницу с обратным направлением. (Страница с обратным направлением — это правая страница в развороте с письмом слева направо или левая страница в развороте с письмом справа налево.)
Не поддерживается в FF
versoПринудительно вставляет один или два разрыва страницы перед блоком, в зависимости от того, какой из них превратит следующую страницу в оборотную страницу. (Оборотная страница — это левая страница на развороте слева направо или правая страница на развороте справа налево.)
Не поддерживается в FF

Значения разрыва столбца
avoid-columnПозволяет избежать разрыва столбца перед блоком.
Не поддерживается в FF
columnПринудительно разрывает столбец сразу перед блоком.
Не поддерживается в FF
Значения разрыва региона
avoid-regionПозволяет избежать разрыва региона перед блоком.
regionПринудительно разрывает область перед блоком.
