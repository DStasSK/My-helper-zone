CSS свойство size.

Свойство size для правила @page определяет размер и ориентацию поля, которое используется для представления страницы. В большинстве случаев этот размер соответствует целевому размеру печатной страницы, если применимо. Размер может быть определен либо с помощью ключевого слова scalable (в этом случае страница заполнит доступные размеры), либо с помощью абсолютных размеров.

@page {
  size: auto | portrait | landscape | value | value value |
        A4 | B5 | JIS-B4 | letter;

  /* размер и ориентация листа */
  size: A4 portrait;
}
  • auto - Размер страницы определяется агентом пользователя. В большинстве случаев используются размеры и ориентация целевого листа.
  • portrait - Содержимое страницы отображается в портретном режиме (т.е. самая длинная сторона окна расположена вертикально). Это ориентация по умолчанию.
  • landscape - Содержимое страницы отображается в альбомном режиме.
  • value - Любое значение длины в едицах CSS. Первое значение соответствует ширине страницы, а второе - ее высоте. Если указано только одно значение, оно используется как для ширины, так и для высоты.
  • A3 - Размеры соответствуют стандарту ISO: 297 x 420 мм.
  • A4 - Размеры соответствуют стандарту ISO: 210 x 297 мм.
  • A5 - Размеры соответствуют стандарту ISO: 148 x 210 мм.
  • B4 - Размеры соответствуют стандарту ISO: 250 x 353 мм.
  • B5 - Размеры соответствуют стандарту ISO: 176 x 250 мм.
  • JIS-B4 - Размеры соответствуют стандарту JIS: 257 мм x 364 мм.
  • JIS-B5 - Размеры соответствуют стандарту JIS: 182 мм x 257 мм.
  • letter - Соответствует размерам почтовой бумаги в Северной Америке, 8,5 х 11 дюймов.
  • legal - Соответствует размерам юридических документов в Северной Америке, 8,5 x 14 дюймов.
  • ledger - Соответствует размерам страниц бухгалтерской книги в Северной Америке, 11 x 17 дюймов.
