Правило CSS - @page.

Правило CSS @page используется для модификации некоторых свойств CSS при печати документа или для страниц, у которых тип носителя задан как print. Вы не можете изменить все CSS-свойства с @page. Вы можете изменить только margin, orphans, widows, и разрывы страницы документа. Попытки изменить любые другие свойства CSS будут игнорироваться.

@page [ { :left | :right | :first } ] {
   /* параметры свойств */
}
  • :left - позволяет задать поля для всех левых страниц.
  • :right - позволяет задать поля для всех правых страниц.
  • :first - позволяет задать поля для первой страницы.
В качестве параметров можно использовать только абсолютные единицы измерения, определяя внешние отступы - css свойство margin. Все остальные свойства внутри правила @page, такие как border, padding и пр. - будут проигнорированы.
8(9) 12 18 9.2 6 116 4.4 116 10.1 6

Пример настроки полей при печати.

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>@page</title>
    <link rel="stylesheet" href="/css/print.css" media="print">
  </head>
  <body>
    <h1>Метод ловли льва в пустыне</h1>
    <h2>Метод последовательного перебора</h2>
    <p>...</p>
  </body>
</html>
Сам файл print.css выглядит следующим образом.
body {font-family: Times, 'Times New Roman', serif;}
h1, h2, p {color: #000;}
@page :first {margin: 1cm;}               /* Отступы для первой страницы */
@page :left  {margin: 1cm 3cm 1cm 1.5cm;} /* Отступы для левых страниц */
@page :right {margin: 1cm 3cm 1cm 1.5cm;} /* Отступы для правых страниц */
