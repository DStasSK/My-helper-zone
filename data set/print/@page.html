@page

Правило @page позволяет задать значение полей при печати документа или для страниц, у которых тип носителя задан как print.

@page [ { :left | :right | :first } ] { ... }

Для установки полей на странице используется универсальное свойство margin или его производные - margin-top, margin-right, margin-bottom, margin-left, задающие соответственно отступ сверху, справа, снизу и слева. Отрицательное значение допускается, но часть страницы может оказаться «отрезанной». Свойства border и padding не применимы.

Ключевое слово :left позволяет задать поля для всех левых страниц, :right — для всех правых, а :first — для первой страницы.

По поводу поддержки браузерами. IE8 действительно поддерживает правило @page, но не поддерживает медиазапросы @media. Поддержка медиазапросов появилась только в IE9. Лучше поместить все правила в стилевой файл и подключить его через <link> со значением media="print".
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>@page</title>
    <link rel="stylesheet" href="/css/print.css" media="print">
  </head>
  <body>
    <h1>Метод ловли льва в пустыне</h1>
    <h2>Метод последовательного перебора</h2>
    <p>...</p>
  </body>
</html>

Сам файл print.css выглядит следующим образом.

body {font-family: Times, 'Times New Roman', serif;}
h1, h2, p {color: #000;}
@page :first {margin: 1cm;}               /* Отступы для первой страницы */
@page :left  {margin: 1cm 3cm 1cm 1.5cm;} /* Отступы для левых страниц */
@page :right {margin: 1cm 3cm 1cm 1.5cm;} /* Отступы для правых страниц */
