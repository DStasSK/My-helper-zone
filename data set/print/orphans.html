CSS свойство orphans

Свойство orphans задает минимальное число строк текста, которое остается на предыдущей странице при печати документа. Это свойство работает в том случае, если весь текст размещается на двух и более печатных страницах. Значение по умолчанию 2, наследуется, применяется к блочным элементам. В качестве значения используют целое положительное число.

<head>
  <style>
    @media print {     /* IE8 не поддерживает медиазапросы */
      p {orphans: 3;}
    }
  </style>

  <!-- или для поддержки IE8 отдельым файлом -->
  <link rel="stylesheet" href="/css/print.css" media="print">
</head>
В типографии, orphan это первая линия абзаца, которая отображается внизу страницы. (Абзац продолжается на следующей странице.)
8 12 25 9.2 7 4.4 14 7.1
